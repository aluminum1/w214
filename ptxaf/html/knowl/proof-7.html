Bewys

(1) \(\Rightarrow\) (2). Veronderstel die lys \(\ve{v}_1\text{,}\) \(\ve{v}_2\text{,}\) \(\ldots\text{,}\) \(\ve{v}_n\) is lineêr afhanklik. Dit beteken dat daar skalare. \(k_1\text{,}\) \(k_2\text{,}\) \(\ldots\text{,}\) \(k_n\) bestaan, waarvan nie almal nul is nie, sodat

\begin{equation} k_1 \ve{v}_1 + k_2 \ve{v}_2 + \cdots + k_n \ve{v}_n = \ve{0}\text{.}\tag{2.2.6} \end{equation}

Laat \(r \in \{1, \,2, \ldots, \, n\}\) die grootste index wees sodat \(k_r \neq 0\text{.}\) (Ons weet dat nie alle \(k_i\) is nul nie, so dit maak sin.) As \(r=1\text{,}\) dan is (2.2.6) eenvoudig die vergelyking

\begin{equation*} k_1 \ve{v}_1 = \ve{0}, \mbox{waar } k_1 \neq 0\text{.} \end{equation*}

Daarom is \(\ve{v}_1 = \ve{0}\) volgens Lemma 1.5.6 en ons is klaar. Andersins, veronderstel \(r \neq 1\text{.}\) Dan word (2.2.6) die vergelyking

\begin{equation*} k_1 \ve{v}_1 + k_2 \ve{v}_2 + \cdots + k_r \ve{v}_r = \ve{0}, \mbox{waar } k_r \neq 0\text{.} \end{equation*}

Deur met \(k_r\) te deel, kan ons nou vir \(\ve{v}_r\) oplos in terme van die voorafgaande vektore \(\ve{v}_1\text{,}\) \(\ve{v}_2\text{,}\) \(\ldots\text{,}\) \(\ve{v}_{r-1}\text{:}\)

\begin{equation*} \therefore \ve{v}_r = - \frac{k_1}{k_r} \ve{v}_1 - \frac{k_2}{k_r} \ve{v}_2 - \cdots - \frac{k_{r-1}}{k_r} \ve{v}_{r-1} \, \text{.} \end{equation*}

(2) \(\Rightarrow\) (1). As \(\ve{v}_1 = \ve{0}\text{,}\) dan is die lys vektore \(\ve{v}_1\text{,}\) \(\ve{v}_2\text{,}\) \(\ldots\text{,}\) \(\ve{v}_n\) sonder twyfel lineêr afhanklik deur Opmerking 2.2.2. Andersins, veronderstel dat vir 'n \(r \in \{2, 3, \ldots, n\}\) is \(\ve{v}_r\) 'n lineêre kombinasie van die voorafgaande vektore, dit is

\begin{equation*} \ve{v}_r = a_1 \ve{v}_1 + a_2 \ve{v}_2 + \cdots + a_{r-1} \ve{v}_{r-1}\text{.} \end{equation*}

Dan volg dit duidelik dat

\begin{equation*} a_1 \ve{v}_1 + a_2 \ve{v}_2 + \cdots + a_{r-1} \ve{v}_{r-1} - \ve{v}_r + 0 \ve{v}_{r+1} + \cdots + 0 \ve{v}_n = \ve{0} \end{equation*}

wat wys dat die vektore \(\ve{v}_1\text{,}\) \(\ve{v}_2\text{,}\) \(\ldots\text{,}\) \(\ve{v}_n\) nie lineêr onafhanklik is nie, aangesien nie al die koëffisiënte nul is nie — in besonder is die koëffisiënt van \(\ve{v}_r\) gelyk aan \(-1\text{.}\)

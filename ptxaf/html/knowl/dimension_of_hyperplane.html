Voorbeeld 2.3.13 Dimension of a hyperplane

Let \(\ve{v} \in \mathbb{R}^n\) be a fixed vector, and consider the hyperplane \(W \subset \mathbb{R}^n\) orthogonal to \(\ve{v}\) as in Voorbeeld 1.6.13. Then you will prove in Exercise Verstaanpunt 2.3.15 that \(\Dim(W) = n-1\text{.}\)

For instance, consider the specific example from Voorbeeld 1.6.13, namely the plane \(W \subset \mathbb{R}^3\) of vectors orthogonal to \(\ve{v} = (1,2,3)\text{.}\) In other words,

\begin{equation} W = \{ (w_1, w_2, w_3) \in \mathbb{R}^3 : w_1 + 2w_2 + 3w_3 = 0 \}.\tag{2.3.25} \end{equation}

There is no 'standard' basis for \(W\text{.}\) But, here is one basis (as good as any other):

\begin{equation} \ve{a} = (1, 0, -\frac{1}{3}), \quad \ve{b} = (0, 1, -\frac{1}{2}).\tag{2.3.26} \end{equation}

You will show this is indeed a basis for \(W\) in Verstaanpunt 2.3.14 below. I computed these vectors as follows. To obtain \(\ve{a}\text{,}\) I simply set \(w_1 = 1, w_2 = 0\) and then solved for \(w_3\) using Equation (2.3.25). Similarly, for \(\ve{b}\text{,}\) I simply set \(w_1 = 0, w_2 = 1\) and then solved for \(w_3\) using (2.3.25).

There is nothing special about my method above for computing a basis for \(W\text{.}\) Here is a different basis for \(W\text{,}\) which I arrived at by choosing random values of \(w_1\) and \(w_2\) and then calculating what \(w_3\) must be in order to satisfy Equation (2.3.25):

\begin{equation} \ve{u} = (1, 2, -\frac{5}{3}), \quad \ve{v} = (-4, 2, 0).\tag{2.3.27} \end{equation}

In any event, we see that \(\Dim(W) = 2\text{.}\)

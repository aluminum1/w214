Daar is ook 'n sommeringsbewerking in die versameling \(C\text{.}\) Ons kan twee polinome algebraïes bymekaartel deur hulle ooreenstemmende koëffisiënte bymekaar te tel:

\begin{gather} [c_4 x^4 + c_3 x^3 + c_2 x^2 + c_1 x^1 + c_0] + [d_4 x^4 + d_3 x^3 + d_2 x^2 + d_1 x^1 + d_0]\notag\\ := (c_4 + d_4) x^4 + (c_3 + d_3) x^3 + (c_2 + d_2) x^2 + (c_1 + d_1) x^1 + (c_0 + d_0)\tag{1.1.6} \end{gather}

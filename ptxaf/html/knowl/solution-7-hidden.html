The solution is relatively straightforward. Define the following two functions:

\begin{equation*} f_{\text{even}} (x)= \frac{1}{2}\left(f(x) + f(-x) \right), \quad f_{\text{odd}} (x)= \frac{1}{2}\left( f(x) - f(-x) \right) \end{equation*}

It is easy to see that, as the names suggest, \(f_{\text{even}} \) is an even function and \(f_{\text{odd}} (x) \) is an odd function. We simply sum \(f_{\text{even}} \) and \(f_{\text{odd}} \text{:}\)

\begin{equation*} f_{\text{even}} (x) + f_{\text{odd}} (x) = \frac{1}{2}\left(f(x) + f(-x) \right) + \frac{1}{2}\left( f(x) - f(-x) \right) = f(x). \end{equation*}

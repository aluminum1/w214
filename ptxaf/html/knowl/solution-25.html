Oplossing 2.3.2.7.1

  1. True. Suppose there is a linear relation on \(\left\{ \ve{u} + \ve{v}, \, \ve{v} + \ve{w}, \, \ve{u} + \ve{w} \right\}\text{:}\)

    \begin{equation*} a( \ve{u} + \ve{v}) + b(\ve{v} + \ve{w}) + c(\ve{u} + \ve{w}). \end{equation*}

    This induces a linear relation on \(\left\{ \ve{u}, \ve{v}, \ve{w} \right\}\text{:}\) \((a + c)\ve u + (a +b )\ve v + (b + c)w = \ve 0\) Since \(\left\{ \ve{u}, \ve{v}, \ve{w} \right\}\) is linearly independent, we must have that

    \begin{equation*} a + c = 0 \qquad (1) \\ a + b = 0 \qquad (2) \\ b + c = 0 \qquad (3) \end{equation*}

    \((1) + (2) \) combined with \((3) \) gives us that \(a = 0 \) and so \(b = c = 0 \) too. We conclude that \(\left\{ \ve{u} + \ve{v}, \, \ve{v} + \ve{w}, \, \ve{u} + \ve{w} \right\}\) is linearly independent.

  2. False. Let \(V = \mathbb{R}^3 \text{.}\) Let

    \begin{equation*} \ve u = (1,0,0) \, \ve v = (0,1,0), \, \ve w = (0,0,1) \end{equation*}

    and so

    \begin{equation*} \ve u - \ve v = (1,-1,0) \\ \ve v - \ve w = (0,1,-1) \\ \ve u - \ve w = (1,0,-1)\text{.} \end{equation*}

    By inspection, we see that

    \begin{equation*} (1,-1,0) + (0,1,-1) = (1,0,-1) \end{equation*}

    and so \(\left\{ \ve{u} - \ve{v}, \, \ve{v} - \ve{w}, \, \ve{u} - \ve{w} \right\}\) is linearly dependent.

    Alternatively, we could have noticed that for any vectors \(\ve u , \ve v , \ve w \text{:}\)

    \begin{equation*} (\ve u - \ve v) + (\ve v - \ve w) = \ve u - \ve w. \end{equation*}

    Thus \(\left\{ \ve{u} - \ve{v}, \, \ve{v} - \ve{w}, \, \ve{u} - \ve{w} \right\}\) satisfies StellingÂ 2.2.8 and so is linearly dependent.

Oplossing 2.1.7.1

We must show that every vector \(\ve{v} \in V\) can be written as a linear combination of the vectors from \(\basis{T}\text{.}\) So let \(\ve{v} \in V\text{.}\) Since \(\basis{S}\) spans \(V\text{,}\) we know we can write \(\ve{v}\) as a linear combination of the vectors from \(\basis{S}\text{:}\)

\begin{equation} \ve{v} = b_1 \ve{v}_1 + \cdots + b_r \ve{v}_r + \cdots + b_{n} \ve{v}_n\tag{2.1.33} \end{equation}

Substituting (2.1.32) into (2.1.33) gives

\begin{align} \ve{v} \amp = b_1 \ve{v}_1 + \cdots + b_r (a_1 \ve{v}_1 + \cdots + a_{r-1} \ve{v}_{r-1}) + b_{r+1} \ve{v}_{r+1} + \cdots + b_n \ve{v}_n \tag{2.1.34}\\ \amp = (b_1 + b_r a_1) \ve{v}_1 + \cdots + (b_r + b_r a_{r-1}) \ve{v}_{r-1} + b_{r+1} \ve{v}_{r+1} + \cdots + b_n \ve{v}_n \tag{2.1.35} \end{align}

Equation (2.1.35) shows that we can express \(\ve{v}\) as a linear combination of the vectors from \(\basis{T}\text{.}\) Hence \(\basis{T}\) spans V.

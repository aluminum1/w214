Voorbeeld 1.6.26 Finding the general solution to a differential equation in SageMath

Let us use SageMath to find the general solution of the following ordinary differential equation

\begin{equation} y^{\prime \prime} + 2y^\prime + y = 0.\tag{1.6.10} \end{equation}

We can do this as follows. Note that we need to be a bit more careful now, first defining our variable \(x\) and then declaring that \(y\) is a function of \(x\text{.}\)

SageMath reports that the general solution is given in terms of two unspecified constants _K1 and _K2 as (_K2*cos(2*x) + _K1*sin(2*x))*e^(-x).

If we set _K1 equal to 1 and _K2 equal to 0 in the general solution, we will get a particular solution \(y_1\) of the differential equation.

SageMath is telling us that \(y_1 = e^{-x} \sin(2x)\) is a particular solution.

Edit the code to set _K2 equal to 0 and _K1 equal to 1 in the general solution to get a different particular solution \(y_2\text{.}\) What is \(y_2\text{?}\)

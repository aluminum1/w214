Voorbeeld 1.6.21 An example of a \(2\)nd order homogenous linear differential equation

For instance,

\begin{equation} x^2 y^{''} -3 x y' + 5 y = 0, \quad x \in (0, \infty)\tag{1.6.5} \end{equation}

is a homogenous 2nd order linear differential equation on the interval \((0, \infty)\text{,}\) and

\begin{equation} y_1(x) = x^2 \sin (\log x)\tag{1.6.6} \end{equation}

is a solution to (1.6.5). Similarly,

\begin{equation} y_2(x) = x^2 \cos (\log x)\tag{1.6.7} \end{equation}

is also a solution to (1.6.5).

We can use SageMath to check that these are indeed solutions to (1.6.5). Click the Evaluate (Sage) button --- it should output 'True', indicating that \(y_1\) is indeed a solution to the differential equation.

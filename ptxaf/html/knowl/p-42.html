As ons aan die polinoom as 'n funksie dink, dan is die nul-polinoom \(\ve{0}\) die funksie wat vir alle waardes van \(x\) nul is, i.e. \(\ve{0} (x) = 0\) vir alle \(x\text{.}\) Hoe ons ookal daaraan dink, as ons die nul-polinoom by 'n ander polinoom tel, gebeur niks nie!

\begin{gather*} [0 x^4 + 0 x^3 + 0 x^2 + 0 x + 0] + [c_4 x^4 + c_3 x^3 + c_2 x^2 + c_1 x + c_0] \\ = [c_4 x^4 + c_3 x^3 + c_2 x^2 + c_1 x + c_0] \end{gather*}

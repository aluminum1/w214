Bewys

Let

\begin{equation*} \basis{C} = \bopen \ve{v}_1, \ldots, \ve{v}_n \bclose \end{equation*}

be a basis for \(V\text{,}\) so that \(\Dim(V) = n\text{.}\) We just need to show that \(W\) is finite-dimensional, i.e. that there exists a basis

\begin{equation*} \basis{B} = \bopen \ve{w}_1, \ldots, \ve{w}_k \bclose \end{equation*}

for \(W\text{.}\) For then \(\basis{B}\) will be a list of \(k\) linearly independent vectors which live in \(W\) (and hence also in \(V\)) and hence we must have \(k \leq n\) by Stelling 2.2.10, as \(\basis{C}\) spans \(V\text{.}\)

We show that \(W\) is finite-dimensional as follows.

If \(W\) is the zero vector space \(\{ \ve{0} \} \text{,}\) then \(W\) is finite-dimensional by definition.

If \(W\) is not the zero vector space, then there exists a nonzero vector \(\ve{w}_1 \in W\text{.}\) Consider the list \(\basis{B}_1 = \bopen \ve{w}_1 \bclose\text{.}\) Note that \(\basis{B}_1\) is linearly independent, by Item 3 of Stelling 2.2.8. So, if \(\basis{B}_1\) spans \(W\text{,}\) then it is a basis for \(W\text{,}\) and so \(W\) is finite-dimensional and we are done.

If \(\basis{B}_1\) does not span \(W\text{,}\) then there exists a vector \(\ve{w}_2 \in W\) which is not a scalar multiple of \(\ve{e}_1\text{.}\) Now consider the list \(\basis{B}_2 = \left\{ \ve{e}_1, \ve{e}_2 \right\}\text{.}\) Once again, \(\basis{B}_2\) is linearly independent, by Item 3 of Stelling 2.2.8. So, if \(\basis{B}_2\) spans \(W\text{,}\) then it is a basis for \(W\text{,}\) and we are done.

If \(\basis{B}_2\) does not span \(W\text{,}\) then there exists a vector \(\ve{w}_3 \in W\) which is not a linear combination of \(\ve{w}_1\) and \(\ve{w}_2\text{.}\) Now consider the list \(\basis{B}_3 = \left\{ \ve{w}_1, \ve{w}_2, \ve{w}_3 \right\}\text{.}\) Again, \(\basis{B}_3\) is linearly independent, by Item 3 of Stelling 2.2.8. If it does not span \(W\text{,}\) then there exists a vector \(\ve{w}_4 \in W\) which is not a linear combination of \(\ve{w}_1, \ve{w}_2, \ve{w}_3\text{.}\) So consider the list \(\basis{B}_4 = \bopen \ve{w}_1, \ve{w}_2, \ve{w}_3, \ve{w}_4 \bclose \text{.}\)

This process must terminate for some \(k \leq n\text{.}\) If not, then it will produce a list \(\basis{B}_{n+1} = \bopen \ve{w}_1, \ldots, \ve{w}_{n+1} \bclose\text{.}\) This would be a linearly independent list of \(n+1\) vectors from \(V\text{.}\) But \(\Dim V = n\text{,}\) so this is impossible, by Stelling 2.2.10. Hence for some \(k \leq n\) we must have that \(\basis{B}_k\) is a basis for \(W\text{,}\) and we are done.

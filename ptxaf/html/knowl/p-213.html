Onthou dat lyn \(L\) gespesifiseer kan word deur 'n homogene lineêre vergelyking van die form:

\begin{equation} L = \{ (x,y) \in \mathbb{R}^2 : ax + by = 0 \}\tag{1.6.1} \end{equation}

vir konstantes \(a\) en \(b\text{.}\) So, as \(\ve{v} = (x, \, y)\) en \(\ve{v}' = (x', \, y')\) op \(L\) lê, dan lê hulle som \(\ve{v} + \ve{v}' = (x + x', \, y + y')\) ook op \(L\text{,}\) want hul komponente bevredig die definiërende vergelyking (1.6.1):

\begin{align*} \amp a(x + x') + b (y + y')\\ \amp = (ax + by) + (ax' + by')\\ \amp = 0 + 0 \amp \amp \mbox{(want \(ax + by = 0\) en \(ax' + by' = 0\))}\\ \amp = 0\text{.} \end{align*}

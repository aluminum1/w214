Oplossing 2.1.3.1
  1. We must set up the appropriate system of linear equations:
    \begin{align*} \amp a\ve r_1(x) + b \ve r_2(x) + c \ve r_3(x) + d \ve r_4(x) \amp = \amp\ve p(x)\\ \implies \amp a(3x^2 - 2) + b(x^2 + x) + c(x+1) + d(x-1) \amp = \amp x^2 + 1 \end{align*}
    After grouping like powers of \(x\) we obtain
    \begin{equation*} x^2(3a + b) + x(b + c + d) + (-2a + c -d ) = x^2 + 1 . \end{equation*}
    We equate coefficients on both sides of the equation to obtain the following system of linear equations:
    \begin{gather*} 3a + b + 0c + 0d = 1,\\ 0a + 1b + 1c + 1d = 0, \\ -2a + 0b + 1c + -1d = 1 . \end{gather*}
    Using your preferred method for solving a system of linear equations (such as Gauss reduction), we obtain a solution set of the form:
    \begin{align*} \amp d \text{ is free},\\ \amp a = 2 + 2d, \\ \amp b = -5 - 6d, \\ \amp c = 5 + 5d. \end{align*}
    And so \(\ve p (x)\) is indeed a linear combination of \(\ve r_1(x),\ve r_2(x),\ve r_3(x),\ve r_4(x)\text{.}\)
  2. Since \(d\) is free in the above solution set, we can write \(\ve p (x)\) as a linear combination of \(\ve r_1(x),\ve r_2(x),\ve r_3(x),\ve r_4(x)\) in an uncountably infinite number of ways (one for each real number!).

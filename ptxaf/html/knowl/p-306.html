To check this, we need to check if the equation

\begin{equation} p = a_1 p_1 + a_2 p_2 + a_3 p_3 + a_4 p_4,\tag{2.1.9} \end{equation}

which is an equation in the unknowns \(a_1, a_2, a_3, a_4\text{,}\) has any solutions. Let us write out (2.1.9) explicitly, grouping together powers of \(x\text{:}\)

\begin{align*} p \amp = a_1 p_1 + a_2 p_2 + a_3 p_3 + a_4 p_4 \\ \therefore x^2 - 1 \amp = a_1 (1+x^2) + a_2 (x-3) + a_3 (x^2 + x + 1) + a_4 (x^2 + x - 1) \\ \therefore -1 + x^2 \amp = (a_1 -3a_2 + a_3 - a_4) + (a_2 + a_3 + a_4) x + (a_1 + a_3 + a_4)x^2 \end{align*}

Now, two polynomials are equal if and only if all their coefficients are equal. So, (2.1.9) is equivalent to the following system of simultaneous linear equations:

\begin{align} a_1 - 3a_2 + a_3 - a_4 \amp = -1 \tag{2.1.10}\\ a_2 + a_3 + a_4 \amp = 0\tag{2.1.11}\\ a_1 + a_3 + a_4 \amp = 1 \tag{2.1.12} \end{align}

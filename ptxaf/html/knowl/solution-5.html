Oplossing 2.1.8.1

To check this, we need check if every vector \(\ve{v} \in V\) can be written as a linear combination of \(\ve{f}_1, \ve{f}_2\) and \(\ve{f}_3\text{.}\)

So, let \(\ve{v} = (v_1,v_2) \) be a fixed, but arbitrary, vector in \(\mathbb{R}^2\text{.}\) We need to check if the following equation has a solution for \(a_1, a_2, a_3\text{:}\)

\begin{equation} \ve{v} = a_1 \ve{f}_1 + a_2 \ve{f}_2 + a_3 \ve{f}_3\tag{2.1.15} \end{equation}

Let us write this equation out explicitly:

\begin{align} \ve{v} \amp = a_1 \ve{f}_1 + a_2 \ve{f}_2 + a_3 \ve{f}_3 \tag{2.1.16}\\ \therefore (v_1, v_2) \amp = a_1 (-1, 2) + a_2 (1, 1) + a_3 (2, -1) \tag{2.1.17}\\ \therefore (v_1, v_2) \amp = (-a_1 + a_2 + 2a_3, 2a_1 + a_2 - a_3) \tag{2.1.18} \end{align}

(2.1.18) is an equation between two vectors in \(\mathbb{R}^2\text{.}\) Two vectors in \(\mathbb{R}^2\) are equal if and only if their corresponding coefficients are equal. So, (2.1.18) is equivalent to the following system of simultaneous linear equations:

\begin{align} -a_1 + a_2 + 2a_3 \amp = v_1 \tag{2.1.19}\\ 2a_1 + a_2 - a_3 \amp = v_2\tag{2.1.20} \end{align}

In other words, the original question

Is \(\mathbb{R}^2\) spanned by \(\ve{f}_1, \ve{f}_2, \ve{f}_3 \) ?

is equivalent to the question

Can we always solve (2.1.19)–(2.1.20) for \(a_1, a_2, a_3\text{,}\) no matter what the fixed constants \(\ve{v}_1, \ve{v}_2\) are?

You already know how to solve simultaneous linear equations such as (2.1.19)–(2.1.20) by hand:

\begin{align} -a_1 + a_2 + 2a_3 \amp = v_1 \tag{2.1.21}\\ 2a_1 + a_2 - a_3 \amp = v_2\tag{2.1.22}\\ \tag{2.1.23}\\ \therefore -a_1 + a_2 + 2a_3 \amp = v_1 \tag{2.1.24}\\ 3a_2 + 3a_3 \amp = 2v_1 + v_2 \amp \amp R2 \rightarrow R2 + 2R1 \tag{2.1.25}\\ \tag{2.1.26}\\ \text{Let } a_3 \amp = t \tag{2.1.27}\\ \therefore a_2 \amp = \frac{1}{3}(2v_1 + v_2) - t \tag{2.1.28}\\ \therefore a_1 \amp = -\frac{1}{3} (-v_1 + v_2) + t \tag{2.1.29} \end{align}

In other words, no matter what \(v_1, v_2\) are, there are always infinitely many solutions (they are parameterized a free parameter \(t\)) to (2.1.19)–(2.1.20), and hence to our original equation (2.1.15). That is, we can express any \(\ve{v} \in \mathbb{R}^2\) as a linear combination of the vectors \(\ve{f}_1, \ve{f}_2, \ve{f}_3\text{...}\) and in fact there are infinitely many ways to do it, parameterized by a free parameter \(t\text{!}\)

For instance, suppose we try to write \(\ve{v} = (2,3)\) as a linear combination of \(\ve{f}_1, \ve{f}_2, \ve{f}_3\text{.}\) If we take our general solution ((2.1.27)–(2.1.29)), and set \(t=0\text{,}\) then we get

\begin{gather*} a_1 = \frac{1}{3}, \, a_2 = \frac{7}{3}, \, a_3 = 0 \\ \text{i.e. } \ve{v} = \frac{1}{3} \ve{f}_1 + \frac{7}{3} \ve{f}_2 \end{gather*}

Or we could take, say, \(t = 1\text{.}\) Then our solution will be

\begin{gather*} a_1 = \frac{4}{3}, \, a_2 = \frac{4}{3}, \, a_3 = 1 \\ \text{i.e. } \ve{v} = \frac{4}{3} \ve{f}_1 + \frac{4}{3} \ve{f}_2 + \ve{f}_3 \end{gather*}

There are infinitely many solutions. But the important point is that there is always a solution to (2.1.15), no matter what \(\ve{v}\) is. Therefore, the vectors \(\ve{f}_1, \ve{f}_2, \ve{f}_3\) indeed span \(\mathbb{R}^2\text{.}\)

Finally, let us solve this problem using SageMath. Working by hand, we arrive at the simultanous linear equations (2.1.19)–(2.1.20), and then put it into a Sage cell:

Note that I needed to tell Sage that v1 and v2 are variables, and that I am asking it to solve for a1, a2 and a3. On my computer, Sage outputs:

Here, r1 is to be interpreted as our free parameter \(t\text{.}\) So Sage is giving us the same solution as we found by hand, (2.1.27)–(2.1.29).

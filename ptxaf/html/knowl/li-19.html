D2. Nulvektor

Ons definieer die nulvektor \(\ve{0} \in B\) as

\begin{equation} \ve{0} := (0, 0, 0)\text{.}\tag{1.3.3} \end{equation}

Ons moet seker maak dat dit sin maak. Is \((0,0,0)\) regtig 'n element van \(B\text{,}\) oftewel, bevredig dit vergelyking (1.3.1)? Ja, omdat \(0 - 0 + 0 = 0\text{.}\) So ons het 'n goed-gedefinieerde nul-vektor.

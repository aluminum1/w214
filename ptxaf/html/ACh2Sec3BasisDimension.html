Afdeling 2.3 Basis en dimensie

¶
Definisie 2.3.1

'n Lys vektore \(\basis{B} = \bopen \ve{e}_1, \ve{e}_2, \ldots, \ve{e}_n \bclose \) in 'n vektorruimte \(V\) word 'n basis van \(V\) genoem as dit lineêr onafhanklik is en \(V\) span.

Stelling 2.3.2 Invariansie van dimensie

As \(\ve{e}_1, \ve{e}_2, \ldots, \ve{e}_n\) en \(\ve{f}_1, \ve{f}_2, \ldots, \ve{f}_n\) albei basisse van 'n vektorruimte \(V\) is, dan is \(m=n\text{.}\)

Bewys

Dit is 'n gevolg van Proposisie 2.2.10 (die afstampproposisie). Aangesien die \(\ve{e}\)-vectore \(V\) lineêr onfafhanklik is en die \(\ve{f}\)-vektore \(V\) span, het ons \(m \leq n\text{.}\) Aan die ander kant, omdat die \(\ve{f}\)-vektore linêr onfahanklik is en die \(\ve{e}\)-vektore \(V\) span, het ons \(n \leq m\text{.}\) Daarom is \(m=n\text{.}\)

Definisie 2.3.3

'n Vektorruimte \(V\) is eindigdimensioneel as dit 'n eindige basis het. In daardie geval is die dimensie van \(V\) die aantal elemente in 'n basis vir \(V\text{.}\) 'n Vektorruimte is oneindigdimensioneel as dit nie eindigdimensioneel is nie.

Let op dat die konsep van die ‘dimensie van’n vektorruimte' slegs goedgedefinieerd is as gevolg van Stelling 2.3.2.

Die geval van die nulvektorruimte \(Z = \{\ve{0}\}\) is nie eksplisiet in Definisie 2.3.3 gehanteer nie. Ons hanteer dit as 'n spesiale geval. Naamlik, ons defineer die dimensie van die nulvektorruimte \(Z\) as \(0\text{.}\) So, volgens die definisie is \(Z\) is eindig-dimensionaal, en sy dimensie is gelyk aan \(0\text{.}\)

Voorbeeld 2.3.4 Standaard basis vir \(\mathbb{R}^n\)

Die lys vektore

\begin{equation*} \ve{e}_1 := (1, 0, \ldots, 0), \,\,\, \ve{e}_2 := (0, 1, \ldots, 0), \,\,\, \ldots, \,\,\, \ve{e}_n := (0, 0, \ldots, 0, 1) \end{equation*}

is 'n basis vir \(\mathbb{R}^n\text{.}\) Ons het reeds in Voorbeeld 2.1.10 gesien dat die lys \(\mathbb{R}^n\) span. Ons moet seker maak dat die lys lineêr onafhanklik is. So, suppose that

\begin{equation*} a_1 \ve{e}_1 + a_2 \ve{e}_2 + \cdots + a_n \ve{e}_n = \ve{0}\text{.} \end{equation*}

Uitbreiding van die linkerkant na komponente volgens die definisie van die standaard basisvektore \(\ve{e}_i\) lewer die vergelyking

\begin{equation*} (a_1, 0, 0, \ldots, 0) + (0, a_2, 0, \ldots, 0) + \cdots + (0, 0, 0, \ldots, a_n) = (0, 0, 0, \ldots, 0)\text{.} \end{equation*}

Met ander woorde, ons het dat

\begin{equation*} (a_1, a_2, a_3, \ldots, a_n) = (0, 0, 0, \ldots, 0) \end{equation*}

wat beteken dat \(a_1 = a_2 = a_3 = \cdots = a_n = 0\text{,}\) presies wat ons moes bewys. Gevolglik is die lys vektore \(\ve{e}_1, \ve{e}_2, \ldots, \ve{e}_n\) lineêr onafhanklik en daarom is dit 'n basis vir \(\mathbb{R}^n\text{.}\) So \(\mathbb{R}^n\) het dimensie \(n\text{.}\)

Voorbeeld 2.3.5 A basis for \(\mathbb{R}^4\)

Check whether the following list of vectors

\begin{equation} \ve{v}_1 = (1, 0, 2, -3), \, \ve{v}_2 = (1, 3, -1, 2), \, \ve{v}_3 = (0, 1, 2, -1), \ve{v}_4 = (1, 2, 3, 4)\tag{2.3.1} \end{equation}

is a basis for \(\mathbb{R}^4\text{.}\)

Oplossing

First we check if the list of vectors is linearly independent. Consider the equation

\begin{align} a_1 \ve{v}_1 + a_2 \ve{v}_2 + a_3 \ve{v}_3 + a_4 \ve{v}_4 \amp = \ve{0} \tag{2.3.2}\\ \therefore a_1 (1, 0, 2,-3) + a_2 (1, 3, -1, 2) + a_3 (0, 1, 2, -1) + a_4 (1, 2, 3, 4) \amp = (0, 0, 0, 0) \tag{2.3.3}\\ \therefore (a_1 - a_2 + a_4, 3a_2 + a_3 + 2a_4, 2a_1 - a_2 + 2a_3 + 3a_4, -3a_1 + 2a_2 -a_3 + 4a_4) \amp = (0,0,0,0) \tag{2.3.4} \end{align}

So the list of vectors is linearly indepedent if and only if the following equations have only the trivial solution \(a_1 = 0, a_2 = 0, a_3 = 0, a_4 = 0\text{:}\)

\begin{align} a_1 - a_2 + a_4 \amp = 0 \label{R4-example-eqn1}\tag{2.3.5}\\ 3a_1 + a_3 + 2a_4 \amp = 0 \tag{2.3.6}\\ 2a_1 - a_2 + 2a_3 + 3a_4 \amp = 0 \tag{2.3.7}\\ -3a_1 + 2a_2 - a_3 + 4a_4 \amp = 0 \label{R4-example-eqn4}\tag{2.3.8} \end{align}

We can compute the solutions to equations (2.3.5)–(2.3.8) by hand, or using SageMath.

SageMath outputs:

[[a1 == 0, a2 == 0, a3 == 0, a4 == 0]]

So indeed, equations (2.3.5)–(2.3.8) have only the trivial solution. Therefore the list of vectors is linearly independent.

Next, we need to check that the list of vectors spans \(\mathbb{R}^4\text{.}\) (There is a shorter way of doing this, using Gevolg 2.3.19 below, but for now we prove it from first principles.) So, let \(\ve{w} = (w_1, w_2, w_3, w_4)\) be an arbitrary vector in \(\mathbb{R}^4\text{.}\) We need to show that there exists at least one way to express \(\ve{w}\) as a linear combination of the vectors \(\ve{v}_1, \ve{v}_2, \ve{v}_3, \ve{v}_4\text{.}\) In other words, we need to check if there exists at least one solution to the following equation:

\begin{align} a_1 \ve{v}_1 + a_2 \ve{v}_2 + a_3 \ve{v}_3 + a_4 \ve{v}_4 \amp = \ve{w} \label{fund_eqn_span_r4_ex}\tag{2.3.9}\\ \therefore a_1 (1, 0, 2,-3) + a_2 (1, 3, -1, 2) + a_3 (0, 1, 2, -1) + a_4 (1, 2, 3, 4) \amp = (w_1, w_2, w_3, w_4) \tag{2.3.10}\\ \therefore (a_1 - a_2 + a_4, 3a_2 + a_3 + 2a_4, 2a_1 - a_2 + 2a_3 + 3a_4, -3a_1 + 2a_2 -a_3 + 4a_4) \amp = (w_1, w_2, w_3, w_4) \tag{2.3.11} \end{align}

So the list of vectors spans \(\mathbb{R}^4\) if and only if the following equations for \(a_1, a_2, a_3, a_4\) always have a solution, no matter what the values of \(w_1, w_2, w_3, w_4\) are:

\begin{align} a_1 - a_2 + a_4 \amp = w_1 \label{R4-example-span-eqn1}\tag{2.3.12}\\ 3a_1 + a_3 + 2a_4 \amp = w_2 \tag{2.3.13}\\ 2a_1 - a_2 + 2a_3 + 3a_4 \amp = w_3 \tag{2.3.14}\\ -3a_1 + 2a_2 - a_3 + 4a_4 \amp = w_4 \label{R4-example-span-eqn4}\tag{2.3.15} \end{align}

We can compute the solutions to equations (2.3.12)–(2.3.15) by hand, or using SageMath:

Note that we ask SageMath to solve for a1, a2, a3, a4, since w1, w2, w3, w4 are regarded as constants in the equation... we are not trying to solve for them, they are fixed, but arbitrary! SageMath outputs:

[[a1 == 1/9*w1 + 7/18*w2 - 2/9*w3 - 1/18*w4, a2 == -2/3*w1 + 5/12*w2 - 1/6*w3 + 1/12*w4, a3 == -7/9*w1 - 2/9*w2 + 5/9*w3 - 1/9*w4, a4 == 2/9*w1 + 1/36*w2 + 1/18*w3 + 5/36*w4]]

In other words, there does indeed exist a solution, no matter what \((w_1, w_2, w_3, w_4)\) is. For instance, if \((w_1, w_2, w_3, w_4) = (3, 1, 2, 4) \text{,}\) then the solution is

\begin{equation*} a_1 = \frac{1}{18}, a_2 = - \frac{19}{12}, a_3 = -\frac{17}{9}, a_4 = \frac{49}{36}. \end{equation*}

In other words,

\begin{equation*} (3, 1, 2, 4) = \frac{1}{18} \ve{v}_1 - \frac{19}{12} \ve{v}_2 - \frac{17}{9} \ve{v}_3 + \frac{49}{36} \ve{v}_4. \end{equation*}

Since there exists a solution to equation (2.3.9) for each vector \(\ve{w} \in \mathbb{R}^4\text{,}\) we conclude that \(\bopen \ve{v}_1, \ve{v}_2, \ve{v}_3, \ve{v}_4 \bclose\) spans \(\mathbb{R}^4\text{.}\)

Hence \(\bopen \ve{v}_1, \ve{v}_2, \ve{v}_3, \ve{v}_4 \bclose\) is a basis for \(\mathbb{R}^4\text{,}\) since it is linearly independent and spans \(\mathbb{R}^4\text{.}\)

Voorbeeld 2.3.6

Die lys polinome

\begin{equation*} \ve{p}_0 (x) := 1, \, \ve{p}_1 (x) := x, \, \ve{p}_2 (x) := x^2, \, \ldots, \, \ve{p}_n (x) := x^n \end{equation*}

is 'n basis vir \(\Poly_n\text{,}\) so \(\dim \Poly_n = n+1\text{.}\) Duidelik span hierdie lys \(\Poly_n\) per definisie, so ons moet net seker maak dat hulle line{êr} onafhanklik is. Veronderstel dat

\begin{equation*} a_0 \ve{p}_0 + a_1 \ve{p}_1 + a_2 \ve{p_2} + \cdots + a_n \ve{p}_n = \ve{0}\text{.} \end{equation*}

Dit is 'n vergelyking van funksies, so dit geld vir alle \(x \in \mathbb{R}\text{!}\) Met ander woorde, vir alle \(x \in \mathbb{R}\) het ons het die vergelyking

\begin{equation} a_0 + a_1 x + a_2 x^2 + \cdots + a_n x^n = 0\text{.}\label{root_of_poly}\tag{2.3.16} \end{equation}

Dink versigtig hieraan. Vergelyking (2.3.16) verteenwordig 'n oneindige stelsel van vergelykings vir die onbekendes \(a_0, a_1, \ldots, a_n\text{.}\) Daar is een vergelyking vir elke waarde van \(x \in \mathbb{R}\text{.}\) Byvoorbeeld

\begin{align} (x=1) \amp \amp a_0 + a_1 + a_2 + \cdots + a_n \amp = 0 \label{poly-example-first-eqn}\tag{2.3.17}\\ (x= -1) \amp \amp a_0 - a_1 + a_2 + \cdots + (-1)^n a_n \amp = 0 \tag{2.3.18}\\ (x= 2) \amp \amp a_0 + 2a_1 + 4a_3 + \cdots + 2^n a_n = 0 \tag{2.3.19}\\ (x= 3) \amp \amp a_0 + 3a_1 + 9a_3 + \cdots + 3^n a_n = 0 \tag{2.3.20}\\ \amp \amp \vdots \label{poly-example-last-eqn}\tag{2.3.21} \end{align}

Veronderstel dat ons waardes vir \(a_0, a_1, a_2, \ldots, a_n \) kan vind wat elkeen van hierdie oneindig veel vergelykings (2.3.17)–(2.3.21) op los. Ons kan nou ons standpunt verander. Naamlik, stel hierdie vaste waardes vir \(a_0, a_1, \ldots, a_n\) in Vergelyking (2.3.16) en beskou Vergelyking (2.3.16) as 'n vergelyking vir die onbekende \(x\) (die koëffisiente \(a_0, a_1, \ldots, a_n\) is nou vasgemaak.) Ons trek die gevolg dat elke \(x \in \mathbb{R}\) 'n wortel van hierdie polinoom vergelyking is!

Maar, ons weet uit algebra dat 'n polinoom van die vorm (2.3.16) met nie-nul koëffisiënte meestens \(n\) wortels \(x_1, x_2, \ldots, x_n\) het . So, die koëffisiënte moet nul wees, i.e. \(a_0 = a_1 = a_2 = \cdots = a_n = 0\text{,}\) wat is wat ons moes bewys.

Voorbeeld 2.3.7

Veronderstel \(X\) is 'n eindige versameling. Dan is \(\Fun(X)\) eindig-dimensioneel, met dimensie \(|X|\text{,}\) waar die basis gegee word deur die funksies \(\ve{f}_a\text{,}\) \(a \in X\text{,}\) gedefinieer deur:

\begin{equation} \ve{f}_a (x) := \begin{cases} 1 \amp \mbox{ if } x = a \\ 0 \amp \mbox{ andersins} \end{cases}\label{defn_of_basis_of_functions}\tag{2.3.22} \end{equation}

Ons sal dit in 'n reeks oefeninge bewys.

Die formule aan die regterkant van (2.3.22) kom so gereeld in wiskunde voor dat ons dit 'n spesiale simbool gee, naamlik \(\delta_{ab}\) (die ‘Kronecker-delta’). Hierdie simbool staan vir die formule: “As \(a=b\text{,}\) gee die waarde 1. As \(a \neq b\text{,}\) gee die waarde \(0\)”. In hierdie taal kan ons die definisie van die funksies \(\ve{f}_a\) as

\begin{equation} \ve{f}_a (x) := \delta_{ax}\tag{2.3.23} \end{equation}

herskryf.

Voorbeeld 2.3.8

\(\Trig_n\) is \((2n+1\))-dimensioneel, met basis

\begin{gather*} \ve{T}_0 (x) := 1, \,\,\, \ve{T}_1 (x) := \cos x, \, \ve{T}_2 (x) := \sin x, \, \ve{T}_3 (x) := \cos 2x, \\ \ve{T}_4 (x) := \sin 2x, \, \ldots, \, \ve{T}_{2n-1} (x) := \cos nx , \, \ve{T}_{2n} (x) := \sin nx. \end{gather*}

Hierdie funksies span \(\Trig_n\) per definisie. Hulle is ook lineêr onafhanklik, alhoewel ons dit nie sal bewys nie.

Voorbeeld 2.3.9

Die dimensie van \(\Mat_{n,m}\) is \(nm\text{.}\) 'n Basis word gegee deur die matrikse

\begin{equation*} \mat{E}_{ij} , i=1 \ldots n, \, j = 1 \ldots m \end{equation*}

wat 'n \(1\) in ry \(i\) en kolom \(j\) het, en nulle orals elders.

Normaalweg is \(\mat{A}\) 'n matriks en \(\mat{A}_{ij}\) is die element van die matriks in die posisie \((i,j)\text{.}\) Maar nou is \(\mat{E}_{ij}\) 'n matriks in eie reg! 'n Element in posisie \((k,l)\) sal geskryf word as \((\mat{E}_{ij})_{kl}\text{.}\) Ek hoop jy vind dit nie te verwarrend nie. Trouens, ons kan 'n elegante formule vir die elemente van \(\mat{E}_{ij}\) neerskryf deur gebruik te maak van die Kronecker delta-simbool:

\begin{equation} (\mat{E}_{ij})_{kl} = \delta_{ik} \delta_{jl}\label{kronecker_for_matrix_basis}\tag{2.3.24} \end{equation}
Voorbeeld 2.3.10

Die standaard basis van \(\Mat_{2,2}\) is

\begin{equation*} \mat{E}_{11} = \left[ \begin{array}{cc} 1 \amp 0 \\ 0 \amp 0 \end{array} \right], \, \mat{E}_{12} = \left[ \begin{array}{cc} 0 \amp 1 \\ 0 \amp 0 \end{array} \right], \mat{E}_{21} = \left[ \begin{array}{cc} 0 \amp 0 \\ 1 \amp 0 \end{array} \right], \mat{E}_{22} = \left[ \begin{array}{cc} 0 \amp 0 \\ 0 \amp 1 \end{array} \right]\text{.} \end{equation*}

Die standaard basis van \(\Col_n\) is

\begin{equation*} \ve{e}_1 := \left[ \begin{array}{c} 1 \\ 0 \\ \vdots \\ 0 \end{array} \right], \, \ve{e}_2 := \left[ \begin{array}{c} 0 \\ 1 \\ \vdots \\ 0 \end{array} \right], \, \ldots, \, \ve{e}_n := \left[ \begin{array}{c} 0 \\ 0 \\ \vdots \\ 1 \end{array} \right] \text{.} \end{equation*}

Ons gaan nou deelruimtes van vektorruimtes se dimensie bestudeer.

Stelling 2.3.12

Laat \(W\) 'n deelruimte van 'n eindig-dimensionele vektorruimte \(V\) wees. Dan is \(W\) eindig-dimensioneel, en \(\Dim(W) \leq \Dim(V)\text{.}\)

Bewys

Let

\begin{equation*} \basis{C} = \bopen \ve{v}_1, \ldots, \ve{v}_n \bclose \end{equation*}

be a basis for \(V\text{,}\) so that \(\Dim(V) = n\text{.}\) We just need to show that \(W\) is finite-dimensional, i.e. that there exists a basis

\begin{equation*} \basis{B} = \bopen \ve{w}_1, \ldots, \ve{w}_k \bclose \end{equation*}

for \(W\text{.}\) For then \(\basis{B}\) will be a list of \(k\) linearly independent vectors which live in \(W\) (and hence also in \(V\)) and hence we must have \(k \leq n\) by Stelling 2.2.10, as \(\basis{C}\) spans \(V\text{.}\)

We show that \(W\) is finite-dimensional as follows.

If \(W\) is the zero vector space \(\{ \ve{0} \} \text{,}\) then \(W\) is finite-dimensional by definition.

If \(W\) is not the zero vector space, then there exists a nonzero vector \(\ve{w}_1 \in W\text{.}\) Consider the list \(\basis{B}_1 = \bopen \ve{w}_1 \bclose\text{.}\) Note that \(\basis{B}_1\) is linearly independent, by Item 3 of Stelling 2.2.8. So, if \(\basis{B}_1\) spans \(W\text{,}\) then it is a basis for \(W\text{,}\) and so \(W\) is finite-dimensional and we are done.

If \(\basis{B}_1\) does not span \(W\text{,}\) then there exists a vector \(\ve{w}_2 \in W\) which is not a scalar multiple of \(\ve{e}_1\text{.}\) Now consider the list \(\basis{B}_2 = \left\{ \ve{e}_1, \ve{e}_2 \right\}\text{.}\) Once again, \(\basis{B}_2\) is linearly independent, by Item 3 of Stelling 2.2.8. So, if \(\basis{B}_2\) spans \(W\text{,}\) then it is a basis for \(W\text{,}\) and we are done.

If \(\basis{B}_2\) does not span \(W\text{,}\) then there exists a vector \(\ve{w}_3 \in W\) which is not a linear combination of \(\ve{w}_1\) and \(\ve{w}_2\text{.}\) Now consider the list \(\basis{B}_3 = \left\{ \ve{w}_1, \ve{w}_2, \ve{w}_3 \right\}\text{.}\) Again, \(\basis{B}_3\) is linearly independent, by Item 3 of Stelling 2.2.8. If it does not span \(W\text{,}\) then there exists a vector \(\ve{w}_4 \in W\) which is not a linear combination of \(\ve{w}_1, \ve{w}_2, \ve{w}_3\text{.}\) So consider the list \(\basis{B}_4 = \bopen \ve{w}_1, \ve{w}_2, \ve{w}_3, \ve{w}_4 \bclose \text{.}\)

This process must terminate for some \(k \leq n\text{.}\) If not, then it will produce a list \(\basis{B}_{n+1} = \bopen \ve{w}_1, \ldots, \ve{w}_{n+1} \bclose\text{.}\) This would be a linearly independent list of \(n+1\) vectors from \(V\text{.}\) But \(\Dim V = n\text{,}\) so this is impossible, by Stelling 2.2.10. Hence for some \(k \leq n\) we must have that \(\basis{B}_k\) is a basis for \(W\text{,}\) and we are done.

Dit is goed om 'n voorbeeld van 'n oneindig-dimensionele vektorruimte te hê.

Stelling 2.3.13

\(\Poly\) is oneindigdimensioneel.

Bewys

Veronderstel \(\Poly\) is eindigdimensioneel. Dit beteken daar bestaan 'n eindige versameling polinome \(\ve{p}_1, \ve{p}_2, \ldots, \ve{p}_n\) wat \(\Poly\) span. Maar, laat \(d\) die hoogste graad van al die polinome in die lys \(\ve{p}_1, \ve{p}_2, \ldots, \ve{p}_n\) wees. Dan is \(\ve{p} := x^{d+1}\) 'n polinoom wat nie in die span van \(\ve{p}_1, \ve{p}_2, \ldots, \ve{p}_n\) is nie, want die sommering en skalaarvermenigvuldiging van polinome kan nooit die graad verhoog nie. Ons het 'n teenstelling bereik. So ons aanvanklike aanname kan nie waar wees nie, i.e. \(\Poly\) kan nie eindigdimensioneel wees nie.

Voorbeeld 2.3.14

Ons sal dit nie hier bewys nie, maar die volgende vektorruimtes is ook eindigdimensioneel:

  • \(\mathbb{R}^\infty\text{,}\)

  • \(\Fun(X)\) waar \(X\) 'n eindige versameling is,

  • \(\Cont(I)\) vir enige nie-leë interval \(I\) en

  • \(\Diff(I)\) vir enige oop interval \(I\text{.}\)

Onderafdeling 2.3.1 Sifting

¶

As ons die bewys van Proposisie 2.2.10 (die ‘Afstampproposisie’) noukeurig bestudeer, vind ons dat dit van 'n sif-algoritme gebruik maak. Hierdie algoritme kan op enige lys vektore \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_n\) in 'n vektorruimte toegepas word. Beskou elke vektor \(\ve{v}_i\) in so 'n lys opeenvolgend. As \(\ve{v}_i\) die nul-vektor is, of as dit 'n lineêre kombinasie van die voorafgaande vektore \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_{n-1}\) is, verwyder dit van die lys.

Voorbeeld 2.3.15

Sif die volgende lys vektore in \(\mathbb{R}^3\text{:}\)

\begin{align*} \ve{v}_1 \amp = (1,2,-1), \amp \ve{v}_2 \amp = (0, 0, 0), \amp \ve{v}_3 \amp = (3, 6, -3)\\ \ve{v}_4 \amp = (1, 0, 5), \amp \ve{v}_5 \amp = (5, 4, 13), \amp \ve{v}_6 \amp = (1, 1, 0)\text{.} \end{align*}

Ons begin met \(\ve{v}_1\text{.}\) Aangesien dit nie die nul-vektor is nie en nie 'n lineêre kombinasie van enige voorafgaande vektore is nie, bly dit in die lys. Nou beweeg ons aan na \(\ve{v}_2\text{,}\) wat nul is, so ons verwyder dit. Ons skuif aan na \(\ve{v}_3\text{,}\) wat ons deur inspeksie vasstel dat dit as\(3 \ve{v}_1\) geskryf kan word, so ons verwyder dit. Ons beweeg aan na \(\ve{v}_4\text{.}\) Dit is nie nul nie, en dit kan nie as 'n veelvoud van \(\ve{v}_1\) uitgedruk word nie (bevestig dit vir jouself), so dit bly in die lys. Ons beweeg aan na \(\ve{v}_5\text{.}\) Ons kyk of dit as die lineêre kombinasie

\begin{equation*} \ve{v}_5 = a\ve{v}_1 + b \ve{v}_4 \end{equation*}

geskryf kan word en vind die oplossing \(a=2, b=3\) (bevestig self), so ons verwyder dit. Uiteindelik kom ons by \(\ve{v}_6\text{.}\) Ons ondersoek of dit as 'n lineêre kombinasie

\begin{equation*} \ve{v}_6 = a\ve{v}_1 + b \ve{v}_4 \end{equation*}

geskryf kan word en vind geen oplossings nie (bevestig self), so dit bly in die lys. Ons finale gesifte lys is

\begin{equation*} \ve{v}_1, \ve{v}_4, \ve{v}_6\text{.} \end{equation*}
Verstaanpunt 2.3.16

Doen die drie ‘bevestig self’-bewerkings hierbo.

Die volgende resultate wys dat sifting a baie nuttige manier is om 'n basis vir 'n vektorruimte te konstrueer!

Hulpstelling 2.3.17

As 'n lys vektore \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_n\) 'n vektorruimte \(V\) span, dan sal sifting van die lys in 'n basis vir \(V\) resulteer.

Bewys

By elke stap vind ons dat 'n vektor wat uit die lys verwyder word óf die nul-vektor is, óf 'n lineêre kombinasie van die voorafgaande vektore is wat nié uit die lys verwyder word nie. So as ons die vektor uit die lys verwyder, sal die oorblywende vektore steeds \(V\) span. Daarom span die vektore in die finale lys steeds \(V\text{.}\)

Om te sien dat die finale gesifte lys lineêr onafhanklik is, pas ons Proposisie 2.2.8 toe. Deur konstruksie is geen vektor in die finale lys 'n lineêre kombinasie van die voorafgaande vektore nie (ander sou dit verwyder gewees het!). Daarom is die finale lys nie lineêr afhanklik nie, so dit moet lineêr onafhanklik wees!

Gevolg 2.3.18

Enige lineêr onafhanklike lys vektore \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_k\) in 'n eindig-dimensionele vektorruimte \(V\) kan uitgebrei word tot 'n basis van \(V\text{.}\)

Bewys

Aangesien \(V\) eindig-dimensioneel is, het dit 'n basis \(\ve{e}_1, \ldots, \ve{e}_n\text{.}\) Oorweeg nou die lys

\begin{equation*} L : \ve{v}_1, \ve{v}_2, \ldots, \ve{v}_k, \ve{e}_1, \ve{e}_2, \ldots, \ve{e}_n \end{equation*}

wat duidelik \(V\) span. Deur die lys te sif, sal ons 'n basis vir \(V\) kry, volgens Lemma 2.3.17. Sommige van die \(\ve{e}\)-vektore mag dalk verwyder wees in die proses. Maar geeneen van die \(\ve{v}\)-vektore sal verwyder word nie, aangesien dit sou beteken dat sommige vektore \(\ve{v}_i\) 'n lineêre kombinasie van die voorafgaande vektore \(\ve{v}_1, \ldots, \ve{v}_{i-1}\) is, wat onmoontlik is, omdat \(\ve{v}_1, \ldots, \ve{v}_k\) 'n lineêr onafhanklike lys is. Dus ná sifting van die lys \(L\) brei ons ons die oorspronklike lys \(\ve{v}_1, \ldots, \ve{v}_k\) uit na 'n basis van \(V\text{.}\)

Gevolg 2.3.19

As \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_n\) 'n lineêr onafhanklike lys van \(n\) vektore in 'n \(n\)-dimensionele vektorruimte \(V\) is, dan is dit 'n basis.

Bewys

Volgens Gevolgtrekking 2.3.18 kan ons \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_n\) uitbrei tot 'n basis van \(V\text{.}\) Maar \(V\) het dimensie \(n\text{,}\) so die basis moet slegs \(n\) vektore bevat volgens Stelling 2.3.2 (Onveranderlikheid van Dimensie). Gevolglike het ons geen vektore bygevoeg nie, en ons oorspronklike lys is reeds 'n basis.

Voorbeeld 2.3.20

In Voorbeeld 2.2.7 het ons gewys dat die lys polinome

\begin{equation*} \ve{q}_0 (x) := 1, \,\, \ve{q}_1 (x):= x, \,\, \ve{q}_2(x) := 2x^2 - 1, \,\, \ve{q}_3(x) := 4x^3 - 3x \end{equation*}

lineêr onafhanklik in \(\Poly_3\) is. Aangesien \(\dim \Poly_3 = 4\text{,}\) sien ons dat dit 'n basis vir \(\Poly_3\) is.

In Oefening 2.1.5 het jy gewys dat \(\ve{q}_0, \ldots, \ve{q}_3\) 'n basis vir \(\Poly_3\) is deur 'n ‘brute krag’ benadering. Hierdie nuwe metode is verskillend!

Onderafdeling 2.3.2 Oefeninge

1

Sif die lys vektore

\begin{align*} \ve{v}_1 \amp = (0,0,0), \amp \ve{v}_2 \amp = (1, 0, -1), \amp \ve{v}_3 \amp = (1, 2, 3)\\ \ve{v}_4 \amp = (3, 4, 5), \amp \ve{v}_5 \amp = (4, 8, 12), \amp \ve{v}_6 \amp = (1, 1, 0)\text{.} \end{align*}
2

Laat \(V\) 'n vektorruimte van dimensie \(n\) wees. Besluit (en skryf jou besluit neer!) of die volgende bewerings waar of onwaar is. As dit waar is, bewys dit. Indien nie, gee 'n teenvoorbeeld.

  • Enige lineêr onafhanklike lys vektore in \(V\) bevat hoogstens \(n\) vektore.

  • Enige lys vektore wat \(V\) span bevat ten minste \(n\) vektore.

3

Voltooi die bewys van die volgende Lemma.

Lemma. Veronderstel dat \(V\) 'n vektorruimte van dimensie \(n\) is. Dan is enige lineêr onafhanklike lys van \(n\) vektore in \(V\) 'n basis vir \(V\text{.}\)

Bewys. Laat \(\basis{B} = \left\{ \ve{v}_1, \ldots, \ve{v}_n \right\}\) 'n lineêr onafhanklike lys vektore in \(V\) wees.

Veronderstel dat \(\basis{B}\) nie 'n basis vir \(V\) is nie.

Daarom span \(\basis{B}\) nie \(V\) nie, want... (a)

Daarom bestaan daar \(\ve{v} \in V\) sodanig dat ... (b)

Voeg nou \(\ve{v}\) by die lys \(\basis{B}\) om 'n nuwe lys te vorm, \(\basis{B}' :=\) ... (c)

Die nuwe lys \(\basis{B}'\) is lineêr onafhanklik omdat ... (d)

Dit is 'n teenstelling, omdat ... (e)

Dus \(\basis{B}\) moet 'n basis vir \(V\) wees.

4

Gebruik Oefening 2.3.2.2(a) om aan te toon dat die lys matrikse in \(\Mat_{2,2}\) vanaf Oefening 2.2.2 lineêr onafhanklik is.

5

In elke geval, gebruik die resultate uit Oefeninge 2.3.2.2 en 2.3.2.3 om te bepaal of \(\basis{B}\) 'n basis vir \(V\) is:

  • \(V = \Poly_2\text{,}\) \(\basis{B} = \left\{ 2 + x^2, \, 1-x, \, 1 + x - 3x^2, \, x-x^2 \right\}\)

  • \(V = \Mat_{2,2}\text{,}\)

    \begin{equation*} \basis{B} = \left\{ \begin{bmatrix} 1 & 2 \\ -1 & 3 \end{bmatrix},\, \begin{bmatrix} 0 & 1\\ 3 & -1 \end{bmatrix},\, \begin{bmatrix} 1 & 2\\ 3 & 4 \end{bmatrix} \right\} \end{equation*}

  • \(V = \Trig_2\text{,}\) \(\basis{B} = \left\{ \sin^2 x, \, \cos^2 x, \, 1 - \sin 2x, \, \cos 2x + 3 \sin 2x \right\}\)

6

Laat \(\left\{ \ve{u}, \ve{v}, \ve{w} \right\}\) 'n lineêr onafhanklike lys vektore in 'n vektorruimte \(V\) wees. Skryf neer of die volgende bewerings waar of onwaar is. As dit waar is, bewys dit. As dit onwaar is, gee 'n teenvoorbeeld. Wenk: Gebruik die definisie van lineêre onafhanklikheid.

  • Die lys \(\left\{ \ve{u} + \ve{v}, \, \ve{v} + \ve{w}, \, \ve{u} + \ve{w} \right\}\) is lineêr onafhanklik.

  • Die lys \(\left\{ \ve{u} - \ve{v}, \, \ve{v} - \ve{w}, \, \ve{u} - \ve{w} \right\}\) is lineêr onafhanklik.

7

Vir elkeen van die volgende, toon aan dat \(V\) 'n deelruimte van \(\Poly_2\) is, vind 'n basis vir \(V\text{,}\) en bereken \(\dim V\text{.}\)

  • \(V = \{ p \in \Poly_2 : p(2) = 0 \}\)

  • \(V = \{ p \in \Poly_2 : xp'(x) = p(x) \}\)

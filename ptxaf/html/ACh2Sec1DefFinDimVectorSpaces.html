Afdeling 2.1 Definisie van eindigdimensionele vektorruimtes

¶

Ons begin met 'n paar basiese definisies.

Definisie 2.1.1

'n Lineêre kombinasie van 'n eindige kolleksie vektore \(\ve{v}_1, \ldots, \ve{v}_n\) in 'n vektorruimte \(V\) is 'n vektor van die vorm

\begin{equation} a_1 \ve{v}_1 + a_2 \ve{v}_2 + \cdots + a_b \ve{v}_n\tag{2.1.1} \end{equation}

waar \(a_1, a_2, \ldots, a_n\) skalare is. As al die skalare \(a_i\) gelyk aan nul is, dan sê ons dat dit die triviale lineêre kombinasie is.

Voorbeeld 2.1.2

In \(\mathbb{R}^3\) is \((6, 2, -14)\) 'n lineêre kombinasie van \((-3, 1, 2)\) en \((-2, 0, 3)\text{,}\) want

\begin{equation*} (6, 2, -14) = 2 (-3, 1, 2) -6 (-2, 0, 3)\text{.} \end{equation*}
Voorbeeld 2.1.3

In \(\mathbb{R}^4\text{,}\) is \((2, -1, 3, 0)\) nie 'n lineêre kombinasie van \((1, 3, 2, 0), \, (5, 1, 2, 4)\) en \((-1, 0, 2, 1)\) nie. As dit wel was, dan sou ons vind dat

\begin{equation*} (-2, -1, 3, 0) = a_1 (1, 3, 2, 0) + a_2 (5, 1, 2, 4) + a_3 (-1, 0, 2, 1) \end{equation*}

vir sekere skalare \(a_1, a_2, a_3\text{.}\) In ander woorde, ons sou vind dat

\begin{equation*} (-2, -1, 3, 0) = (a_1 + a_2 + a_3, \, 3a_1 + a_2, \, 2a_1 + 2a_2 + 2a_3, \, 4a_2 + a_3) \end{equation*}

wat ekwivalent is aan die sisteem van vergelykings

\begin{align} a_1 + 5a_2 -a_3 \amp = -2 \label{sim_lin_eqns_first_span_1ex}\tag{2.1.2}\\ 3a_1 + a_2 \amp = -1 \tag{2.1.3}\\ 2a_1 + 2a_2 + 2a_3 \amp = 3\tag{2.1.4}\\ 4a_2 + a_3 \amp = 0 \label{sim_lin_eqns_first_span_4ex}\tag{2.1.5} \end{align}

Verstaanpunt 2.1.4

Bevestig dat die sisteem van vergelykings (2.1.2)–(2.1.5) geen oplossing het nie.

Voorbeeld 2.1.5

Definieer die funksies \(\ve{f}, \ve{f}_1, \ve{f}_2 \in \Diff\) as

\begin{equation*} \ve{f}(x) = \cos^3 x, \ve{f}_1 (x) = \cos (x), \ve{f}_2 (x) = \cos (3x)\text{.} \end{equation*}

Dan is \(\ve{f}\) 'n lineêre kombinasie van \(\ve{f}_1\) en \(\ve{f}_2\text{,}\) vanweë die identiteit \(\cos(3x) = \frac{1}{4} (3 \cos x + \cos(3x))\text{.}\) Sien Voorbeeld 1.6.18. Met ander woorde,

\begin{equation*} \ve{f} = \frac{3}{4} \ve{f}_1 + \frac{1}{4} \ve{f}_2 \text{.} \end{equation*}

Hierdie voorbeeld wys dat \(\ve{f}\) ook 'n trigonometriese polinoom is,selfs al is die oorspronklike formule \(\ve{f}(x) = cos(3x)\) nie van die form (1.6.3) nie.

Definisie 2.1.6

Ons sê dat die lys vektore \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_n\) in 'n vektorruimte \(V\) \(V\) span as elke vektor \(\ve{v} \in V\) 'n lineêre kombinasie van \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_n\) is.

Voorbeeld 2.1.7

\(\mathbb{R}^2\) word deur

\begin{equation*} \ve{e}_1 := (1, \, 0), \, \, \ve{e}_2 := (0, 1) \end{equation*}

gespan, want elke vektor \(\ve{v} = (a_1, \, a_2)\) kan as 'n lineêre kombinasie

\begin{equation*} \ve{v} = a_1 \ve{e}_1 + a_2 \ve{e}_2 \end{equation*}

uitgedruk word.

Voorbeeld 2.1.8

\(\mathbb{R}^2\) word ook deur

\begin{equation*} \ve{f}_1 := (-1, 2), \, \ve{f}_2 := (1, 1), \, \ve{f}_3 := (2, -1) \end{equation*}

gespan.

Let daarop dat elke vektor \(\ve{v}\) geskryf kan vord as 'n lineêre kombinasie van \(\ve{f}_1\text{,}\) \(\ve{f}_2\text{,}\) \(\ve{f}_3\) op meer as een manier (oneindig baie maniere, as't ware). Byvoorbeeld,

\begin{align} (2, 3) \amp = 3 \ve{f}_1 - \frac{1}{3} \ve{f}_2 + \frac{8}{3} \ve{f}_3\label{x1st_f_eqn}\tag{2.1.6}\\ (2,3) \amp = 2 \ve{f}_1 + \frac{2}{3} \ve{f}_2 + \frac{5}{3} \ve{f}_3\label{x2nd_f_eqn}\tag{2.1.7} \end{align}
Verstaanpunt 2.1.10

Bevestig dat vergelykings (2.1.6) en (2.1.7) korrek is.

Voorbeeld 2.1.11

\(\mathbb{R}^n\) word deur

\begin{equation} \ve{e}_1 := (1, 0, \ldots, 0), \,\,\, \ve{e}_2 := (0, 1, \ldots, 0), \,\,\, \ldots, \,\,\, \ve{e}_n := (0, 0, \ldots, 0, 1)\label{defn_of_e_i}\tag{2.1.8} \end{equation}

gespan, want elke vektor \(\ve{v} = (a_1, \,a_2, \, \ldots, \, a_n)\) kan as 'n lineêre kombinasie

\begin{equation} \ve{v} = a_1 \ve{e}_1 + a_2 \ve{e}_2 + \cdots + a_n \ve{e}_n\text{.}\label{standard_basis_R_n}\tag{2.1.9} \end{equation}

uitgedruk word.

Verstaanpunt 2.1.12

Bevestig die korrektheid van (2.1.9).

Onderafdeling Oefeninge

1

Beskou die volgende polinome in \(\Poly_2\text{:}\)

\begin{equation*} \ve{r}_1 (x) := 3x^2 - 2, \, \ve{r}_2 (x) := x^2 + x, \, \ve{r}_3 (x) := x + 1, \, \ve{r}_4 (x) := x-1 \end{equation*}

  1. Kan die polinoom \(\ve{p}\) met \(\ve{p}(x) = x^2 + 1\) as 'n lineêre kombinasie van \(\ve{r}_1\text{,}\) \(\ve{r}_2\text{,}\) \(\ve{r}_3\text{,}\) \(\ve{r}_4\) geskryf word?

  2. Indien wel, in hoeveel verskillende maniere kan dit gedoen word?

2

Veronderstel die vektore \(\ve{e}_1\text{,}\) \(\ve{e}_2\text{,}\) \(\ve{e}_3\) en \(\ve{e}_4\) span 'n vektorruimte \(V\text{.}\) Wys dat die vektore \(\ve{f}_1 := \ve{e}_2 - \ve{e}_1\text{,}\) \(\ve{f}_2 := \ve{e}_3 - \ve{e}_2\text{,}\) \(\ve{f}_3 := \ve{e}_4 - \ve{e}_3\text{,}\) \(\ve{f}_4 := \ve{e}_4\) ook \(V\) span.

3

Wys dat die polinome

\begin{equation*} \ve{q}_0 (x) := 1, \,\, \ve{q}_1 (x) := x, \,\, \ve{q}_2 (x) := 2x^2 - 1, \,\, \ve{q}_3 (x) := 4x^3 - 3x \end{equation*}

\(\Poly_3\) span.

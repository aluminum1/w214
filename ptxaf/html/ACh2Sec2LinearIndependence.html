Afdeling 2.2 Lineêre onafhanklikheid

¶
Definisie 2.2.1

'n Lys van vektore \(\basis{B} = \bopen \ve{v}_1, \ve{v}_2, \ldots, \ve{v}_n \bclose\) in 'n vektorruimte \(V\) is lineêr onafhanklik as die vergelyking

\begin{equation} k_1 \ve{v}_1 + k_2 \ve{e}_2 + \cdots k_n \ve{v}_n = \ve{0}\label{lin_independence_eqn}\tag{2.2.1} \end{equation}

slegs die triviale oplossing \(k_1 = k_2 = \cdots = k_n = 0\) het. Andersins word die lys vektore \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_n\) lineêr afhanklik genoem.

Opmerking 2.2.2 Nulvektor impliseer lineêr onafhanklik

Veronderstel een van die vektore \(\ve{v}_i\) in die lys \(\basis{B} = \bopen \ve{v}_1, \ldots, \ve{v}_n \bclose \) is die nulvektor \(\ve{0}\text{.}\) Dan is die lys \(\basis{B}\) lineêr afhanklik, want die vergelyking (2.2.1) het die nie-triviale oplossing

\begin{equation*} 0 \ve{v}_1 + \cdots + 0 \ve{v}_{i-1} + 1 \ve{v}_i + 0 \ve{v}_{i+1} + \cdots + 0 \ve{v}_n = \ve{0}, \end{equation*}

met ander woorde,

\begin{equation*} k_1 = 0, \ldots, k_{i-1} = 0, k_i = 1, k_{i+1} = 0, \ldots, k_n = 0. \end{equation*}

So: 'n lineêr onafhanklike lys bevat nooit die nulvektor nie!

Voorbeeld 2.2.3

Die lys van vektore \(\ve{f}_1 = (-1, \, 2)\) en \(\ve{f}_2 = (1, \,1)\) uit Voorbeeld 2.1.8 is lineêr onafhanklik, omdat die vergelyking

\begin{equation*} k_1 (-1, \, 2) + k_2 (1, \, 1) = (0, \, 0) \end{equation*}

ekwivalent is aan die sisteem van vergelykings

\begin{equation} -k_1 + k_2 = 0, \quad 2k_1 + k_2 = 0\label{system_from_r2_example}\tag{2.2.2} \end{equation}

wat slegs die trivale oplossing \(k_1 = 0\) en \(k_2 = 0\) het.

Verstaanpunt 2.2.4

Bevestig dat (2.2.2) slegs die triviale oplossing het.

Oplossing
\begin{equation*} (2k_1 + k_2) - (-k_1 + k_2) = 0 = 3k_2 \implies k_2 = 0 \implies k_1 = 0. \end{equation*}
Voorbeeld 2.2.5

Die lys van vektore \(\ve{f}_1 = (-1, \, 2)\text{,}\) \(\ve{f}_2 = (1, \,1)\text{,}\) \(\ve{f}_3 = (2, \, -1)\) uit Voorbeeld 2.1.8 is lineêr afhanklik, want die vergelyking

\begin{equation} k_1 (-1, \, 2) + k_2 (1, \, 1) + k_3 (2, \, -1) = (0, \, 0)\label{eqn_to_check_lin_ind}\tag{2.2.3} \end{equation}

is ekwivalent aan die sisteem van vergelykings

\begin{equation} -k_1 + k_2 + 2k_3 = 0, \quad 2k_1 + k_2 - k_3 = 0\label{system_from_r2_example_new}\tag{2.2.4} \end{equation}

wat 'n een-dimansionele vektorruimte van oplossings het wat deur \(t\) parameteriseer word,

\begin{equation} k_1 = t, \, k_2 = -t, \, k_3 = t, t \in \mathbb{R}\text{.}\label{supposed_soln}\tag{2.2.5} \end{equation}

Byvoorbeeld, vir \(t=2\text{,}\) is

\begin{equation*} 2 (-1, \, 2) - 2 (1, \, 1) + 2 (2, \, -1) = (0, \, 0) \end{equation*}

sodat (2.2.3) non-triviale oplossings het.

Verstaanpunt 2.2.6

Wys dat (2.2.4) die oplossingsversameling (2.2.5) het.

Oplossing

We have a system of consistent homogenous linear equations so we know there exists at least one solution, namely the trivial solution. Since we have 3 unknowns but only 2 equations, we do not have a unique solution. Let \(k_1\) be free, i.e. \(k_1 = t, t \in \mathbb{R}\text{.}\) Then

\begin{equation*} (-k_1 + k_2 + 2k_3) - (2k_1 + k_2 - k_3) = 0 = -3k_1 + 3k_3 \\ \implies -k_1 + k_3 = 0 \implies k_3 = t. -t + k_2 + 2t = 0 \implies k_2 = -t \end{equation*}
Voorbeeld 2.2.7

Die lys polinome

\begin{equation*} \ve{q}_0 (x) := 1, \,\, \ve{q}_1 (x) := x, \,\, \ve{q}_2 (x) := 2x^2 - 1, \,\, \ve{q}_3 (x) := 4x^3 - 3x \end{equation*}

uit Voorbeeld 2.1.5 is lineêr onafhanklik in \(\Poly_3\text{.}\) Dit is omdat die vergelyking

\begin{equation*} k_0 \ve{q}_0 + k_1 \ve{q}_1 + k_2 \ve{q}_2 + k_3 \ve{q}_3 = \ve{0} \end{equation*}

vereenvoudig as die volgende polinoomvergelyking:

\begin{equation*} 4k_3 x^3 + 2k_2 x^2 + (-3k_3 + k_1) x + (-k_2 + k_0) = 0 \end{equation*}

Hierdie is ekwivalent aan die volgende stelsel vergelykings,

\begin{equation*} 4k_3 = 0, \quad 2k_2 = 0, \quad -3 k_3 + k_1 = 0, \quad k_0 - k_2 = 0 \end{equation*}

wat slegs die triviale oplossing \(k_0 = k_1 = k_2 = k_3 = 0\) het.

Hier is twee meer maniere om hoe van lineêre afhanklike lyste vektore te dink.

Stelling 2.2.8 Equivalent Criterions for Linear Dependence

Let \(\basis{B} = \bopen \ve{v}_1, \ldots, \ve{v}_n \bclose\) be a list of vectors in a vector space \(V\text{.}\) The following statements are equivalent:

  1. The list of vectors \(\basis{B}\) is linearly dependent.
  2. (Linear Combination of Other Vectors)

    One of the vectors in the list \(\basis{B}\) is a linear combination of the other vectors in \(\basis{B}\text{.}\)

  3. (Linear Combination of Preceding Vectors)

    Either \(\ve{v}_1 = \ve{0}\text{,}\) or for some \(r \in \{2, 3, \ldots, n\}\text{,}\) \(\ve{v}_r\) is a linear combination of \(\ve{v}_1\text{,}\) \(\ve{v}_2\text{,}\) \(\ldots\text{,}\) \(\ve{v}_{r-1}\text{.}\)

Bewys

We will show that (1) \(\Leftrightarrow\) (2), (1) \(\Rightarrow\) (3) and (3) \(\Rightarrow\) (2), and conclude that each statement implies the others.

(1) \(\Rightarrow\) (2). Suppose that \(\basis{B}\) is linearly dependent. This means that there are scalars \(k_1\text{,}\) \(k_2\text{,}\) \(\ldots\text{,}\) \(k_n\text{,}\) not all zero, such that

\begin{equation} k_1 \ve{v}_1 + k_2 \ve{v}_2 + \cdots + k_n \ve{v}_n = \ve{0}\text{.}\label{lin_dep_lem_another_eqn}\tag{2.2.6} \end{equation}

Let \(k_s\) be one of the nonzero coefficients. Then, by taking the other vectors to the other side of the equation, and multuplying by \(\frac{1}{k_s}\) we can solve for \(\ve{v}_s\) in terms of the other vectors:

\begin{align*} \ve{v}_s \amp = -\frac{k_1}{k_s} \ve{v}_1 - \ldots - \frac{k_n}{k_s} \ve{v}_n \amp \amp \mbox{(No } \ve{v}_i \mbox{ terms on RHS)} \end{align*}

Therefore, (2) is true.

(2) \(\Rightarrow\) (1). Suppose that one of the vectors in the list, say \(\ve{v}_s\text{,}\) is a linear combination of the others vectors. That is,

\begin{align*} \ve{v}_s \amp= k_1\ve{v}_1 + \ldots + k_n \ve{v}_n \amp \amp \mbox{(No } \mbox{term on RHS.)} \end{align*}

Rearranging this equation gives:

\begin{equation} k_1 \ve{v}_1 + \ldots + (-1)\ve{v}_s + \ldots + k_n \ve{v}_n = \ve{0}.\label{rearranged_lin_dep}\tag{2.2.7} \end{equation}

Not all the coefficients on the LHS of (2.2.7) are zero, since the coefficient of \(\ve{v}_s\) is equal to \(-1\text{.}\) Therefore, \(\basis{B}\) is linearly dependent.

(1) \(\Rightarrow\) (3). Suppose that the list \(\basis{B} = \bopen \ve{v}_1, \ldots, \ve{v}_n \bclose\) is linearly dependent. This means that there are scalars \(k_1\text{,}\) \(k_2\text{,}\) \(\ldots\text{,}\) \(k_n\text{,}\) not all zero, such that

\begin{equation} k_1 \ve{v}_1 + k_2 \ve{v}_2 + \cdots + k_n \ve{v}_n = \ve{0}\text{.}\label{lin_dep_lem_first_eqn}\tag{2.2.8} \end{equation}

Let \(r \in \{1, \,2, \ldots, \, n\}\) be the largest index such that \(k_r \neq 0\text{.}\) (We are told that not all the \(k_i\) are zero, so this makes sense.) If \(r=1\text{,}\) then (2.2.8) is simply the equation

\begin{equation*} k_1 \ve{v}_1 = \ve{0}, \mbox{where } k_1 \neq 0\text{.} \end{equation*}

Therefore \(\ve{v}_1 = \ve{0}\) by Lemma 1.5.6, and we are done. On the other hand, suppose \(r \neq 1\text{.}\) Then (2.2.8) becomes the equation

\begin{equation*} k_1 \ve{v}_1 + k_2 \ve{v}_2 + \cdots + k_r \ve{v}_r = \ve{0}, \mbox{where } k_r \neq 0\text{.} \end{equation*}

By dividing by \(k_r\text{,}\) we can now solve for \(\ve{v}_r\) in terms of the preceding vectors \(\ve{v}_1\text{,}\) \(\ve{v}_2\text{,}\) \(\ldots\text{,}\) \(\ve{v}_{r-1}\text{:}\)

\begin{equation*} \therefore \ve{v}_r = - \frac{k_1}{k_r} \ve{v}_1 - \frac{k_2}{k_r} \ve{v}_2 - \cdots - \frac{k_{r-1}}{k_r} \ve{v}_{r-1} \, \end{equation*}

Therefore, (3) is true.

(3) \(\Rightarrow\) (2) Suppose that (3) is true. In other words, either:

  • \(\ve{v}_1 = \ve{0}\text{.}\) Therefore, \(\basis{B}\) is linearly dependent, by Opmerking 2.2.2. In other words, (1) is true. Therefore, since we have already proved that (1) \(\Rightarrow \) (2), we conclude that (2) is true.
  • For some \(r \in \{2, \ldots, n\}\text{,}\) \(\ve{v}_r\) is a linear combination of \(\ve{v}_1, \ldots, \ve{v}_{r-1}\text{.}\) In this case, clearly \(\ve{v}_r\) is a linear combination of the other vectors in \(\basis{B}\text{,}\) so (2) is true.

In both cases, (2) is true. So, (3) \(\Rightarrow\) (2).

Voorbeeld 2.2.9

Ons het in Voorbeeld 2.2.5 gesien dat die lys vektore \(\ve{f}_1 = (-1, \, 2)\text{,}\) \(\ve{f}_2 = (1, \,1)\text{,}\) \(\ve{f}_3 = (2, \, -1)\) in \(\mathbb{R}^3\) lineêr afhanklik is. Gee twee alternatiewe bewyse hiervan, deur gebruik te maak van Stelling 2.2.8.

Oplossing 1

Ons kontrolleer Item 2 uit Stelling 2.2.8. Dit wil sê, ons kontrolleer of een van die vektore in die lys 'n lineêre kombinasie van die ander vektore is. Inderdaad, ons sien deur inspeksie dat

\begin{equation} \ve{f}_2 = \ve{f}_1 + \ve{f}_3.\tag{2.2.9} \end{equation}

Dus, \(\basis{B}\) is lineêr afhanklik.

Oplossing 2

Ons kontrolleer Item 3 uit Stelling 2.2.8. Dit wil sê, ons kontrolleer:

  • \(\ve{f}_1 = \ve{0}\) (nie waar nie), óf

  • \(\ve{f}_2\) is 'n skalaarveelvoud van \(\ve{f}_1\) (nie waar nie), óf

  • \(\ve{f}_3\) is 'n lineêre kombinasie van \(\ve{f}_1\) en \(\ve{f}_2\text{,}\) wat waar is:

    \begin{equation*} \ve{f}_3 = - \ve{f}_1 + \ve{f}_2 \end{equation*}
Stelling 2.2.10 Afstampproposisie

Veronderstel \(\basis{L} = \bopen \ve{l}_1, \ve{l}_2, \ldots, \ve{l}_m \bclose\) is 'n lineêr onafhanklike lys vektore in 'n vektorruimte \(V\) en dat \(\basis{S} = \bopen \ve{s}_1, \ve{s}_2, \ldots, \ve{s}_n \bclose\) vir \(V\) span. Dan is \(m \leq n\text{.}\)

Bewys

Begin met die oorspronglike lys vektore

\begin{equation} \basis{S} = \bopen \ve{s}_1, \ve{s}_2, \ldots, \ve{s}_n \bclose \tag{2.2.10} \end{equation}

wat \(V\) span en oorweeg die ‘opgeblase’ lys

\begin{equation} \basis{S^{\prime}} = \bopen \ve{l}_1, \ve{s}_1, \ve{s}_2, \cdots, \ve{s}_n \bclose\label{bloated_list}\tag{2.2.11} \end{equation}

Nou, omdat \(\basis{S}\) vir \(V\) span, weet ons in besonder dat \(\ve{l}_1\) kan uitgedruk word as 'n lineêre kombinasie van die vektore \(\ve{s}_1, \ldots, \ve{s}_n\text{.}\) Dus, deur Item 2 van Stelling 2.2.8, weet ons dat \(\basis{S}'\) lineêr afhanklik is. Dus, deur Item 3 van Stelling 2.2.8, óf:

  • \(\ve{l}_1 = \ve{0}\text{.}\) Dit kan nie waar wees nie, want dan sou \(\basis{L}\) lineêr afhanklik wees weens Opmerking 2.2.2, wat in teenstryding is met ons oorspronklike aanname.
  • een van die \(\ve{s}\)-vectore, noem dit \(\ve{s}_r\text{,}\) kan uitgedruk word as 'n lineêre kombinasie van die voorafgaande vektore. Ons kan dan \(\ve{s}_r\) uit die lys \(\basis{S}'\) verwyder (‘afstamp’), en die resulterende lys
    \begin{equation} \basis{S}_1 := \left\{\ve{l}_1, \ve{s}_1, \ve{s}_2, \cdots, \hat{\ve{s}_r}, \cdots, \ve{s}_n \right\} \qquad \mbox{(\(\ve{s}_r\) omitted)}\tag{2.2.12} \end{equation}
    sal nog steeds vir \(V\) span, deur Oefening 2.1.7.

In hierdie manier kan ons aangaan: elke keer 'n \(\ve{l}\)-vektor oor te dra en 'n \(\ve{s}\)-vektor te verwyder, en die resulterende lys sal nog steeds vir \(V\) span:

\begin{align*} \basis{L} \amp = \bopen \ve{l}_1, \ldots, \ve{l}_m \bclose \amp \basis{S} \amp = \bopen \ve{s}_1, \ldots, \ve{s}_n \bclose \\ \basis{L}_1 \amp = \bopen \ve{l}_2, \ldots, \ve{l}_m \bclose \amp \basis{S}_1 \amp = \bopen \ve{l}_1, \underbrace{\ve{s}_1, \ldots, \ve{s}_n}_{n-1} \bclose \\ \basis{L}_2 \amp = \bopen \ve{l}_3, \ldots, \ve{l}_m \bclose \amp \basis{S}_2 \amp = \bopen \ve{l}_2, \ve{l}_1, \underbrace{\ve{s}_1, \ldots, \ve{s}_n}_{n-2} \bclose \\ \amp \vdots \amp \amp \vdots \end{align*}

Nou, veronderstel dat \(m > n\text{.}\) Wanneer ons die \(n\)ste stadium van hierdie proses bereik, sal \(\basis{S}_n = \bopen \ve{l}_n, \ldots, \ve{l}_1 \bclose\text{,}\) en dit sal vir \(V\) span. Dus, in besonder, \(\ve{l}_{n+1}\) (let op dat hierdie vektore wel bestaan, want \(m > n \)) sal 'n lineêre kombinasie van \(\ve{l}_1, \ldots, \ve{l}_n\) wees. Maar dan, deur Item 2 van Stelling 2.2.8, lei ons af dat \(\basis{L}\) lineêr afhanklik is. Maar ons het aangeneem van die begin af dat \(\basis{L}\) lineêr onafhanklik is. So ons het 'n teenstryding. Dus, ons aanname dat \(m > n\) moet vals wees. Dus, dit moet waar wees dat \(m \leq n\text{.}\)

Onderafdeling Oefeninge

1

Wys dat die lys vektore \((2, \, 3, \, 1)\text{,}\) \((1, \, -1, \, 2)\text{,}\) \((7, \, 3, \, c)\) lineêr afhanklik in \(\mathbb{R}^3\) is as en slegs as \(c=8\text{.}\)

Oplossing

We set up a linear equation and find the necessary conditions on \(c\text{.}\) Suppose some linear combination of the vectors equals 0:

\begin{equation*} k_1(2, \, 3, \, 1) + k_2(1, \, -1, \, 2) + k_3(7, \, 3, \, c) = \ve 0 = (0,0,0) \end{equation*}

This vector equation gives rise to a system of 3 linear equations:

\begin{equation*} 2k_1 + k_2 + 7k_3 = 0,\\ 3k_1 - k_2 + 3k_3 = 0,\\ k_1 + 2k_2 + ck_3 = 0. \end{equation*}

The corresponding matrix equation is

\begin{equation*} \begin{bmatrix} 2 \mamp 1 \mamp 7 \\ 3 \mamp -1 \mamp 3\\ 1 \mamp 2 \mamp c \end{bmatrix} \begin{bmatrix} k_1 \\ k_2 \\ k_3\\ \end{bmatrix} = \begin{bmatrix} 0 \\ 0 \\ 0 \end{bmatrix} \end{equation*}

This matrix is non-invertible if and only if its determinant is 0. Furthermore, the matrix being non-invertible will mean we can find a non-trivial solution to the intial equation. We compute the determinant:s

\begin{equation*} \det\begin{bmatrix} 2 \mamp 1 \mamp 7 \\ 3 \mamp -1 \mamp 3\\ 1 \mamp 2 \mamp c \end{bmatrix} = -5c + 40 \end{equation*}

which is 0 if and only if \(c =8 \text{.}\)

2

Die lys vektore in \(\Mat_{2,2}\) gegee deur

\begin{equation*} \ve{v}_1 = \begin{bmatrix} 1 & 2 \\ 1 & 1 \end{bmatrix},\, \ve{v}_2 = \begin{bmatrix} 1 & 0 \\ -2 & 1 \end{bmatrix},\, \ve{v}_3 = \begin{bmatrix} 1 & 0 \\ 2 & 3 \end{bmatrix},\, \ve{v}_4 = \begin{bmatrix} 0 & 3 \\ 1 & -1 \end{bmatrix},\, \ve{v}_5 = \begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix} \end{equation*}

is lineêr onafhanklik (ons sal dit in Oefening 2.3.2.4 bewys, maar ter wille van hierdie vraag kan jy aanvaar dat dit waar is). Herhaal dieselfde stappe as in Voorbeeld 2.2.9 om die eerste vektor in die lys te vind wat of die nulvektor of 'n lineêre kombinasie van die voorafgaande vektore is.

Oplossing

Firstly note that \(\ve v_1 \) is non-zero, so we consider \(\ve v_2 \text{.}\) \(\ve v_2 \) cannnot be a scalar multiple of \(\ve v_1\) by considering the matrix entry in position (1,2). We now consider \(\ve v_3 \text{.}\) Suppose

\begin{equation*} a\begin{bmatrix} 1 & 2 \\ 1 & 1 \end{bmatrix} + b \begin{bmatrix} 1 & 0 \\ -2 & 1 \end{bmatrix} = \begin{bmatrix} 1 & 0 \\ 2 & 3 \end{bmatrix} \end{equation*}

This gives rise to a system of four linear equations. In particular, we have the equation for the matrix entry in position (1,2):

\begin{equation*} 2a + 0b = 0 \end{equation*}

And hence \(a = 0 \text{.}\) But clearly

\begin{equation*} b \begin{bmatrix} 1 & 0 \\ -2 & 1 \end{bmatrix} \neq \begin{bmatrix} 1 & 0 \\ 2 & 3 \end{bmatrix} \end{equation*}

for any choice of \(b\text{.}\) Hence \(\ve v_3 \) is not a scalar multiple of \(\ve v_1 \) and \(\ve v_2\text{.}\) We consider \(\ve v_4 \) next. Suppose

\begin{equation*} a\begin{bmatrix} 1 & 2 \\ 1 & 1 \end{bmatrix} + b \begin{bmatrix} 1 & 0 \\ -2 & 1 \end{bmatrix} + c \begin{bmatrix} 1 & 0 \\ 2 & 3 \end{bmatrix} = \begin{bmatrix} 0 & 3 \\ 1 & -1 \end{bmatrix} \end{equation*}

The equation for the entry in position (1,2) is simply

\begin{equation*} 2a = 3 \end{equation*}

and so \(a = \frac{3}{2}\text{.}\) The corresponding equation for the entry in position (1,1) is thus

\begin{equation*} \frac{3}{2} + b + c = 0. \end{equation*}

Using this result, we consider the equation for the entry in position (2,2) and compute:

\begin{equation*} \frac{3}{2} + b + 3c = -1 \\ \implies \frac{3}{2} + b + c + 2c = -1 \\ \implies 2c = -1 \implies c = -\frac{1}{2} \end{equation*}

and so \(b = -1 \text{.}\) SHOW THAT THIS IS INCONSISTENT WITH (2,1).

3

Laat \(\basis{B} = \left\{ \ve{v}_1, \ldots, \ve{v}_n \right\}\) 'n lineêr onafhanklike lys vektore in 'n vektorruimte \(V\) wees. Veronderstel dat \(\ve{v}\) 'n vektor in \(V\) is wat nie as 'n lineêre kombinasie van die vektore uit \(\basis{B}\) geskryf kan word nie. Toon aan dat die lys \(\basis{B}' = \left\{ \ve{v}_1, \ldots, \ve{v}_n, \ve{v} \right\}\) lineêr onafhanklik is. (Wenk: Gebruik die Lineêre Kombinasie van Voorafgaande Vektore Proposisie.)

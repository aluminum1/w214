Section 2.4 Coordinate vectors

¶

There is a more direct way to think about a basis of a vector space.

Proposition 2.4.1 Bases give coordinates

A list of vectors\(\basis{B} = \left\{ \ve{e}_1, \ve{e}_2,\ldots, \ve{e}_n\right\}\) in a vector space \(V\) is a basis for \(V\) if and only if every vector \(\ve{v} \in V\) can be written as a linear combination

\begin{equation} \ve{v} = a_1 \ve{e}_1 + a_2 \ve{e}_2 + \cdots + a_n \ve{e}_n\label{trying_to_formulate_basis}\tag{2.4.1} \end{equation}

in precisely one way. (That is, for each \(\ve{v} \in V\) there exist scalars \(a_1\text{,}\) \(a_2\text{,}\) \(\ldots\text{,}\) \(a_n\) satisfying (2.4.1), and that moreover these scalars are unique).

It is important to understand the mathematical phrase ‘There exists a unique \(X\) satisfying \(Y\)’. It means two things. Firstly, that there does exist an \(X\) which satisfies \(Y\text{.}\) And secondly, that there is no more than one \(X\) which satisfies \(Y\text{.}\)

We call the scalars \(a_1, a_2, \ldots, a_n\) appearing in (2.4.1) the coordinates of \(\ve{v}\) in the basis \(\ve{e}_1, \ve{e}_2, \ldots, \ve{e}_n\text{.}\)

Proof

\(\Rightarrow\text{.}\) Suppose that the list of vectors\(\basis{B} = \left\{ \ve{e}_1, \ve{e}_2,\ldots, \ve{e}_n\right\}\) is a basis for \(V\text{.}\) Suppose \(\ve{v} \in V\text{.}\) Since the list of vectors \(\basis{B}\) spans \(V\text{,}\) we know that we can write \(\ve{v}\) as a linear combination of the vectors in the list in at least one way,

\begin{equation} \ve{v} = a_1 \ve{e}_1 + a_2 \ve{e}_2 + \cdots + a_n \ve{e}_n\text{.}\label{one_way_of_expanding}\tag{2.4.2} \end{equation}

We need to show that this is the only way to express \(\ve{v}\) as a linear combination of the vectors \(\ve{e}_i\text{.}\) Indeed, suppose that we also have

\begin{equation} \ve{v} = b_1 \ve{e}_1 + b_2 \ve{e}_2 + \cdots + b_n \ve{e}_n\text{.}\label{second_way_of_expanding}\tag{2.4.3} \end{equation}

Subtracting these two equations gives

\begin{equation*} \ve{0} = (a_1 - b_1) \ve{e}_1 + (a_2 - b_2) \ve{e}_2 + \cdots + (a_n - b_n) \ve{e}_n\text{.} \end{equation*}

Since the list of vectors \(\ve{e}_1\text{,}\) \(\ve{e}_2\text{,}\) \(\ldots\text{,}\) \(\ve{e}_n\) is linearly independent, we conclude that

\begin{equation*} a_1 - b_1 = 0, \,\, a_2 - b_2 = 0, \,\, \cdots, \,\, a_n - b_n = 0\text{.} \end{equation*}

That is, \(a_1 = b_1\text{,}\) \(a_2 = b_2\text{,}\) and so on up to \(a_n = b_n\text{,}\) and hence the expansion (2.4.2) is unique.

\(\Leftarrow\text{.}\) Conversely, suppose that every vector \(\ve{v}\) can be written as a unique linear combination

\begin{equation*} \ve{v} = a_1 \ve{e}_1 + a_2 \ve{e}_2 + \cdots + a_n \ve{e}_n\text{.} \end{equation*}

The fact that each \(\ve{v}\) can be written as a linear combination of the vectors \(\ve{e}_1\text{,}\) \(\ve{e}_2\text{,}\) \(\ldots\text{,}\) \(\ve{e}_n\) means that \(\basis{B}\) spans \(V\text{.}\) We still need to show that this list \(\basis{B}\) is linearly independent. So, suppose that there exist scalars \(b_1\text{,}\) \(b_2\text{,}\) \(\ldots\text{,}\) \(b_n\) such that

\begin{equation} b_1 \ve{e}_1 + b_2 \ve{e}_2 + \cdots + b_n \ve{e}_n = \ve{0}\text{.}\label{finding_bs_making_zero}\tag{2.4.4} \end{equation}

We need to show that all the \(b_i\) must equal zero. We already know one possible solution of (2.4.4) : simply set each \(b_i = 0\text{.}\) But we are told that each vector (in particular, the vector \(\ve{0}\)) can be expressed as a linear combination of the \(\ve{e}_i\) in exactly one way. Hence this must be the only solution, i.e. we must have \(b_1 = b_2 = \cdots = b_n = 0\text{,}\) and so the list \(\basis{B}\) is linearly independent.

Definition 2.4.2

Let \(\basis{B} = \left\{ \ve{b}_1, \ve{b}_2, \ldots, \ve{b}_n \right\}\) be a basis for a vector space \(V\text{,}\) and let \(\ve{v} \in V\text{.}\) Write

\begin{equation*} \ve{v} = a_1 \ve{b}_1 + a_2 \ve{b}_2 + \cdots + \ve{a}_n \ve{b}_n \, \text{.} \end{equation*}

Then the column vector

\begin{equation*} [\ve{v}]_\basis{B} := \left[ \begin{array}{c} a_1 \\ a_2 \\ \vdots \\ a_n \end{array} \right] \in \Col_n \end{equation*}

is called the coordinate vector of \(\ve{v}\) with respect to the basis \(\basis{B}\).

I indicate that a collection of things is a list (where the order matters) and not merely a set (where the order does not matter) using my own home-made symbols \(\{ \}\text{.}\) A basis \(\basis{B} = \left\{ \ve{b}_1, \ve{b}_2, \ldots, \ve{b}_n \right\}\) is a list of vectors. The order of the vectors matters because it affects the coordinate vector \([\ve{v}]_\basis{B}\text{.}\)

Example 2.4.3

Find the coordinate vector of \(\ve{p} = 2x^2 - 2x + 3\) with respect to the basis \(\basis{B} = \left\{ 1+x, x^2 + x -1, x^2 + x + 1\right\}\) of \(\Poly_3\text{.}\)

Solution

We write \(\ve{p}\) as a linear combination of polynomials from the basis \(\basis{B}\text{:}\)

\begin{equation*} 2x^2 - 2x + 3 = -4(1+x) - \frac{5}{2} (x^2 + x -1) + \frac{9}{2}(x^2 + x +1) \end{equation*}
\begin{equation*} [\ve{p}]_\basis{B} := \left[ \begin{array}{c} -4 \\ -\frac{5}{2}\\ \frac{9}{2} \end{array} \right] \end{equation*}

Example 2.4.7

Find the coordinate vector of the function \(\ve{f}\) given by

\begin{equation*} \ve{f}(x) = \sin^2 x - \cos^3 x \end{equation*}

with respect to the standard basis

\begin{equation*} \basis{S} = \left\{ 1, \cos x, \sin x, \cos 2x, \sin 2x, \cos 3x, \sin 3x\right\} \end{equation*}

of \(\Trig_3\text{.}\)

Solution

Using the addition formulae for \(\sin\) and \(\cos\) as in Exercise 1.6.20, we compute

\begin{equation*} \sin^2 x - \cos^3 x = \frac{1}{2} - \frac{3}{4} \cos x - \frac{1}{2} \cos 2x - \frac{1}{4} \cos 3x\text{.} \end{equation*}

Hence

\begin{equation*} [\ve{f}]_\basis{S} = \left[ \begin{array}{c} \frac{1}{2} \\ -\frac{3}{4} \\ 0 \\ -\frac{1}{2} \\ 0 \\ -\frac{1}{4} \\ 0 \end{array} \right] \end{equation*}
Checkpoint 2.4.8

Check this!

Solution
\begin{equation*} \cos 2x = 1 - 2\sin^2x \implies \sin^2x = \frac{1}{2} - \frac{1}{2}\cos 2x \end{equation*}
\begin{equation*} \cos^3 x = \cos x \cos^2 x = \cos x (1 - \sin^2 x) = \cos x(\frac{1}{2} + \frac{1}{2}\cos 2x) \\ = \frac{1}{2}\cos x + \frac{1}{2}\cos x \cos 2x \end{equation*}
\begin{equation*} \cos x \cos 2x = \cos 3x + \sin x \sin 2x = \cos 3x + 2 \sin^2 x \cos x \\ = \cos 3x + (1 - \cos 2x) \cos x = \cos 3x + \cos x - \cos 2x \cos x \\ \implies \cos x \cos 2x = \frac{1}{2}\cos 3x + \frac{1}{2}\cos x \end{equation*}

Thus

\begin{equation*} \sin^2 x - \cos^3 x = \frac{1}{2} - \frac{1}{2}\cos 2x - (\frac{1}{2}\cos x + \frac{1}{2}\cos x \cos 2x) \\ = \frac{1}{2} - \frac{1}{2}\cos 2x - \frac{1}{2}\cos x - \frac{1}{2}(\frac{1}{2}\cos 3x + \frac{1}{2}\cos x) \\ = \frac{1}{2} -\frac{3}{4}\cos x - \frac{1}{2}\cos 2x - \frac{1}{4}\cos 3x. \end{equation*}
Lemma 2.4.9

Let \(\basis{B} = \left\{ \ve{e}_1, \ve{e}_2, \ldots, \ve{e}_n \right\}\) be a basis for a vector space \(V\text{.}\) Then for all vectors \(\ve{v}, \ve{w} \in V\) and all scalars \(k\) we have

  1. \([\ve{v} + \ve{w}]_\basis{B} = [\ve{v}]_\basis{B} + [\ve{w}]_\basis{B}\)

  2. \([k \ve{v}]_\basis{B} = k [ \ve{v}]_\basis{B}\)

Proof

(a) Suppose that

\begin{align*} \ve{v} \amp = a_1 \ve{e}_1 + a_2 \ve{e}_2 + \cdots + a_n \ve{e}_n\\ \end{align*}

and

\begin{align*} \ve{w} \amp = b_1 \ve{e}_1 + b_2 \ve{e}_2 + \cdots + b_n \ve{e}_n \, .\\ \end{align*}

Then, using the rules of a vector space, we compute

\begin{align*} \ve{v} + \ve{w} \amp = (a_1 + b_1) \ve{e}_1 + (a_2 + b_2) \ve{e}_2 + \cdots + (a_n + b_n) \ve{e}_n \, \text{.} \end{align*}

From this we read off that

\begin{equation*} [\ve{v} + \ve{w}]_\basis{B} = \left[ \begin{array}{c} a_1 + b_1 \\ a_2 + b_2 \\ \vdots \\ a_n + b_n \end{array} \right]\text{.} \end{equation*}

The proof of (b) is similar.

Subsection Exercises

1

Prove Lemma 2.4.9(b) in the case where \(V\) is two-dimensional, so that \(\basis{B} = \left\{ \ve{e}_1, \ve{e}_2\right\}\text{.}\) Justify each step using the rules of a vector space.

Solution

As before, suppose that

\begin{align*} \ve{v} \amp = a_1 \ve{e}_1 + a_2 \end{align*}

Which gives

\begin{equation*} [\ve v]_{\basis B} =\left[\begin{array}{c} a_1 \\ a_2 \end{array}\right]. \end{equation*}

Then

\begin{align*} k\ve v \amp = k(a_1 \ve e_1 + a_2 \ve e_2) \\ \amp = k(a_1 \ve e_1) + k(a_2 \ve e_2) \amp (\knowl{./knowl/R4.html}{\text{R4 }})\\ \amp = (ka_1)\ve e_1 + (ka_2) \ve e_2 \amp (\knowl{./knowl/R6.html}{\text{R6 }}) \end{align*}

Reading off the coefficients, we obtain

\begin{equation*} k\left[\begin{array}{c} a_1 \\ a_2 \end{array}\right] = \left[\begin{array}{c} ka_1 \\ ka_2 \end{array}\right], \end{equation*}

as desired.

2

Let \(\basis{B}\) be the basis of \(\Mat_{2,2}\) given by

\begin{equation*} \mat{B}_1 = \begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix}, \mat{B}_2 = \begin{bmatrix} 1 & 0\\ 0& -1 \end{bmatrix}, \mat{B}_3 = \begin{bmatrix} 1 & 1\\ 1 & 1 \end{bmatrix}, \mat{B}_4 = \begin{bmatrix} 0 & 1\\ -1 & 0 \end{bmatrix}\text{.} \end{equation*}

Determine \([\mat{A}]_\basis{B}\text{,}\) where

\begin{equation*} \mat{A} = \begin{bmatrix} 1 & 2\\ 3 & 4 \end{bmatrix}. \end{equation*}
Solution

To determine \([\mat{A}]_\basis{B}\text{,}\) we must find the scalars \(a_1 , a_2 , a_3, a_4 \) satisfying

\begin{equation*} a_1 \mat{B_1} + a_2\mat{B_2} + a_3\mat{B_3} + a_4\mat{B_4} = \mat A. \end{equation*}

This results in a system of 4 linear equations in 4 variables, one equation for each entry in \(\mat A \text{:}\)

\begin{gather*} a_1 + a_2 + a_3 = 1 \\ a_3 + a_4 = 2 \\ a_3 - a_4 = 3 \\ a_1 - a_2 + a_3 = 4 \end{gather*}

Solving this equation, we get

\begin{equation*} a_1 = 0, \quad a_2 = -\frac{3}{2}, \quad a_3 = \frac{5}{2}, \quad a_4 = -\frac{1}{2}. \end{equation*}

Hence

\begin{equation*} [\mat{A}]_\basis{B} = \begin{bmatrix} 0 \\ -\frac{3}{2} \\ \frac{5}{2} \\ -\frac{1}{2} \end{bmatrix} \end{equation*}

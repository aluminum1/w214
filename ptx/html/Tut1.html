Tutorial 1 covers Section 1.1 up until the end of Section 1.5. The following exercises have been selected.

Subsection Exercises

1

Prove that set \(C\) from Section 1.1 together with the addition operation (1.1.6), the zero vector (1.1.9) and the scalar multiplication operation (1.1.12) forms a vector space.

Solution

Firstly, we note that the addition operator, the zero vector, and scalar multiplication are all well-defined. We are required to check R1-R8.

R1: Let

\begin{gather*} p = a_4 x^4 + a_3 x^3 + a_2 x^2 + a_1 x + a _0 \\ q = b_4 x^4 + b_3 x^3 + b_2 x^2 + b_1 x + b _0. \end{gather*}

Then

\begin{align*} p + q \amp = (a_4 +b_4)x^4 + \ldots + (a_0 +b_0) \amp \mbox{(defn of addition in C)}\\ \amp = (b_4 + a_4)x^4 + \ldots (b_0 + a_0) \amp \mbox{(s+t = t+s for real numbers s,t)}\\ \amp = q +p \amp \mbox{(defn of addition in C)} \end{align*}

R2: Let \(r = c_4 x^4 + \ldots c_0 \) Then

\begin{align*} (p+q) +r \amp = \left[a_4 +b_4)x^4 + \ldots + (a_0 +b_0)\right] + (c_4 x^4 + \ldots c_0) \amp \mbox{(defn of addition in C)} \\ \amp = ((a_4 +b_4) + c_4)x^4 + \ldots + ((a_0 +b_0) + c_4) \amp \mbox{(defn of addition in C)}\\ \amp = (a_4 +(b_4 + c_4)x^4 + \ldots + (a_0 +(b_0 + c_4)) \amp \mbox{((s+t)+u = s +(t+u) for real numbers s,t,u)}\\ \amp = \left[ (a_4 x^4 + \ldots a_0 \right] + \left[(b_4+c_4) x^4 + \ldots (b_0 + c_0) \right] \amp \mbox{(defn of addition in C)}\\ \amp = p + (q +r) \amp \mbox{(defn of addition in C)} \end{align*}

R3: Let \(z = 0x^4 \ldots 0 \text{.}\) R3a:

\begin{align*} z + p \amp = (0x^4 \ldots 0) + (a_4 x^4 + \ldots a_0) \amp \mbox{(defn of addition in C)}\\ \amp = (0 + a_4) x^4 + \ldots (0 + a_0) \amp \mbox{(defn of addition in C)} \\ \amp = a_4 x^4 + \ldots a_0 \amp \mbox{(0 + t = t for real numbers t)}\\ \amp = p \end{align*}
The rest of the rules are similar.

2

Define the set \(C^\prime\) consisting of all polynomials of degree exactly 4. Show that if \(C^\prime\) is given the addition operation (1.1.6), the zero vector (1.1.9) and the scalar multiplication operation (1.1.12) then \(C^\prime\) does not form a vector space.

Hint

Give a counterexample!

Solution

We shall show that \(C^\prime\) is not closed under addition. Let

\begin{align*} p \amp = x^4 \\ q \amp = -x^4 + x^3. \end{align*}

Then

\begin{equation*} p + q = x^3. \end{equation*}

But \(x^3 \) is not in \(C^\prime \text{.}\) Hence \(C^\prime \) is not closed under addition and so cannot be a vector space.

3

Consider the set

\begin{equation*} X := \{(a_1, a_2) \in \mathbb{R}^2 : a_1 \geq 0, a_2 \geq 0\} \end{equation*}

equipped with the same addition operation (1.1.4), zero vector (1.1.8) and scalar multiplication operation (1.1.10) as in \(A\text{.}\) Does \(X\) form a vector space? If not, why not?

Solution

\(X \) is not a vector space since scalar multiplication is not defined! For example, consider \((1,1)\text{.}\) \((1,1) \in X\) but \((-1)\smul (1,1) = (-1,-1)\) is not.

5

Let \(X = \{a, b, c\}\text{.}\)

  1. Write down three different functions \(f, g, h\) in \(\Fun(X)\text{.}\)

  2. For each of the functions you wrote down in Item 3.1.5.a, calculate (i) \(f + g\) and (ii) \(3 \smul h\text{.}\)

Solution
  1. \begin{gather*} f(a)= 4 \\ f(b) = 0 \\ f(c) = 2 \\ \\ g(a)= 1 \\ g(b) = 1 \\ g(c) = 1 \\ \\ h(a)= 0 \\ h(b) = 3\\ h(c) = 0 \end{gather*}
  2. \begin{gather*} (f+g)(a) =5\\ (f+g)(b) =1 \\ (f+g)(c) = 3\\ \\ ( 3 \smul h)(a) = 0 \\ ( 3 \smul h)(b) = 9 \\ ( 3 \smul h)(c) = 0 \end{gather*}
6

Define a strange new addition operation \(\hat{+}\) on \(\mathbb{R}\) by

\begin{equation*} x \hat{+} y := x - y, \qquad x, y \in \mathbb{R}\text{.} \end{equation*}

Does \(\hat{+}\) satisfy R2? If it does, prove it. If it does not, give a counterexample.

Solution

No, for example:

\begin{equation*} (1 \hat{+} 2) \hat{+} 3 = (1-2)-3 = -4\text{.} \end{equation*}

But

\begin{equation*} 1 \hat{+} (2 \hat{+}3) = 1 - (2-3) = 2\text{.} \end{equation*}
7

Construct an operation \(\boxplus\) on \(\mathbb{R}\) satisfying R1 but not R2.

Hint

Try adjusting the formula from Exercise 3.1.6.

Solution

Define \(x\boxplus y = | x - y | \text{.}\) R1 is satisfied since \(x \boxplus y = | x - y | = | y -x | = y \boxplus x\text{.}\) However, R2 is not satisfied since \((1 \boxplus 2) \boxplus 3 = | |1-2| -3| = 2 \) but \(1 \boxplus ( 2 \boxplus 3) = | 1 - |2-3| = 0 \)

8

Prove that for all vectors \(\ve{v}\) in a vector space, \(-(-\ve{v}) = \ve{v}\text{.}\)

Solution

The proof is as follows:

\begin{align*} -(-v) \amp = (-1)\smul\left( (-1) \smul v\right) \amp \mbox{(defn of -v applied twice)}\\ \amp = \left((-1)(-1)\right) \smul v \amp \mbox{(R6)}\\ \amp = 1 \smul v \\ \amp = v \amp \mbox{(R7)} \end{align*}
9

Let \(V\) be a vector space. Suppose that a vector \(\ve{v} \in V\) satisfies

\begin{equation} 5\smul \ve{v} = 2 \smul \ve{v}.\tag{3.1.1} \end{equation}

Prove that \(\ve{v}=\ve{0}\text{.}\)

Solution
\begin{align*} \amp 5 \smul v = 2\smul v \\ \amp \implies 5.v + (-2).v = 2.v + (-2).v \\ \amp \implies (5 - 2).v = (2 - 2).v \\ \amp \implies 3.v = 0.v \\ \amp \implies (\frac{1}{3} 3).v = (\frac{1}{3} 0) v \\ \amp \implies 1v = 0v \\ \amp \implies v = 0 \end{align*}
10

If \(k \smul \ve{v} = \ve{0}\) in a vector space, then it necessarily follows that \(k=0\text{.}\)

Solution

False. Take \(\mathbb{R}^2\) as an example. If \(v = (0,0)\) then \(2.(0,0) = (0,0) \) but, of course, \(2 \neq 0\text{.}\)

11

The empty set can be equipped with data D1 , D2 , D3  satisfying the rules of a vector space.

Solution

False. In order for the empty set to be a vector space, it must have a zero vector. That is, we must be able to find some \(v \in \text{the empty set} \) satisfying the axioms for the zero vector. However, since the empty set has no elements in it, by definition, we cannot ever hope to find such a \(v \text{.}\) Hence the empty set can never be a vector space.

12

Rule R7  of a vector space follows automatically from the other rules.

Solution

False. Let \(V\) be a non-zero vector space (such as \(\mathbb{R}^2\)). Redefine scalar multiplication as follows

\begin{equation*} k.v := 0 \mbox{ for all scalars k and all vectors v}. \end{equation*}

Then \(V\) will satisfy all the rules of a vector space except R7. Thus it is not the case that R7 follows from the other rules.

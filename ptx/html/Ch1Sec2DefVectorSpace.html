Section 1.2 Definition of an abstract vector space

Â¶

Mathematics is about identifying patterns. We have found three different sets, \(A\text{,}\) \(B\) and \(C\text{,}\) which look very different on the surface but have much in common. In each set, there is an addition operation, a zero vector, and a scalar multiplication operation. Moreover, in each set, these operations satisfy the same rules. Let us now record this pattern by giving it a name and writing down the rules explicitly.

Definition 1.2.1

A vector space is a set \(V\) equipped with the following data:

D1.

An addition operation. (That is, for every pair of elements \(\ve{u}, \ve{v} \in V\text{,}\) a new element \(\ve{u} + \ve{v} \in V\) is defined.)

D2.

A zero vector. (That is, a special vector \(\ve{0} \in V\) is marked out.)

D3.

A scalar multiplication operation. (That is, for each real number \(k\) and each element \(\ve{v} \in V\text{,}\) a new element \(k \smul\ve{v} \in V\) is defined).

This data should satisfy the following rules for all \(\ve{u}, \ve{v}, \ve{w}\) in \(V\) and for all real numbers \(k\) and \(l\text{:}\)

R1.

\(\ve{v} + \ve{w} = \ve{w} + \ve{v}\)

R2.

\((\ve{u} + \ve{v}) + \ve{w} = \ve{u} + (\ve{v} + \ve{w})\)

R3a.

\(\ve{0} + \ve{v} = \ve{v}\)

R3b.

\(\ve{v} + \ve{0} = \ve{v}\)

R4.

\(k \smul(\ve{v} + \ve{w}) = k \smul\ve{v} + k \smul\ve{w}\)

R5.

\((k + l) \smul\ve{v} = k \smul\ve{v} + l \smul{\ve{v}}\)

R6.

\(k \smul(l \smul\ve{v}) = (kl) \smul\ve{v}\)

R7.

\(1 \smul\ve{v} = \ve{v}\)

R8.

\(0 \smul\ve{v} = \ve{0}\)

We will call the elements of a vector space vectors, and we will write them in boldface eg. \(\ve{v} \in V\text{.}\) We do this to distinguish vectors from real numbers, which we will call scalars, and which don't have a boldface. It is difficult to use boldface in handwriting, so you should write them with an arrow on top, like so: \(\vec{v}\text{.}\)

Also, in this chapter we will write scalar multiplication with a \(\cdot\text{,}\) for instance \(k \smul\ve{v}\text{,}\) but in later chapters we will simply write it as \(k \ve{v}\) for brevity, so be careful!

To prove that a certain set can be given the structure of a vector space, one therefore needs to do the following:

  1. Define a set \(V\text{.}\)

  2. Define the data of an addition operation (D1), a zero vector (D2), and a scalar multiplication operation (D3) on \(V\text{.}\)

  3. Check that this data satisfies the rules (R1) - (R8).

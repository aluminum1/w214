Definition 3.0.1

Let \(f : \mathbb{R}^{n+1} \rightarrow \mathbb{R}\) be a function. The zero set of \(f\) is the subset \(M \subset \mathbb{R}^{n+1}\) given by

\begin{equation*} M := \{ \ve{x} \in \mathbb{R}^{n+1} : f(\ve{x}) = 0 \} \text{.} \end{equation*}
Definition 3.0.2

We say that a subset \(M \subset \mathbb{R}^{n+1}\) is an \(n\)-dimensional hypersurface if there exists a function \(f : \mathbb{R}^{n+1} \rightarrow \mathbb{R}\) such that \(M\) is the zero set of \(f\text{,}\) and if for all \(\ve{p} \in M\text{,}\)

\begin{equation*} \nabla_{\ve{p}} f \mbox{ exists and is not equal to } \ve{0}\text{.} \end{equation*}
Example 3.0.3 Hyperbola

Let \(f : \mathbb{R}^2 \rightarrow \mathbb{R}\) be the function

\begin{equation*} f(x,y) = y^2 - x^2 - 1 \end{equation*}

Draw a picture of the zero set \(M\) of \(f\text{,}\) and show that it is a hypersurface.

Solution

The zero level set of \(f\) is

\begin{equation*} M = \{ (x,y) \in \mathbb{R}^2 : y^2 - x^2 - 1 = 0 \} \end{equation*}

which is a hyperbola: tikzalign

To show that \(M\) is a hypersurface, we first compute the gradient of \(f\) at a general point \(\ve{r} = (x,y) \in \mathbb{R}^2\text{:}\)

\begin{equation*} \nabla_\ve{r} f = (2y, -2x) \end{equation*}

The gradient clearly exists everywhere. Moreover, for a point \(\ve{p} = (a,b)\) on \(M\text{,}\) we have

\begin{equation*} \nabla_\ve{p} f = (2b, -2a) \mbox{where } b^2 = 1 + a^2 \end{equation*}

Suppose \(\nabla_\ve{p} f = (0,0)\text{.}\) Then in particular, \(2b = 0\) so that \(b=0\text{,}\) which means that \(0 = 1 + a^2\) using the equation for \(M\text{.}\) This is impossible. Hence \(\nabla_\ve{p} f \neq (0,0)\) for all \(\ve{p} \in M\text{,}\) so that \(M\) is a hypersurface.

Example 3.0.4 Two-sphere

Let \(f : \mathbb{R}^3 \rightarrow \mathbb{R}\) be the function

\begin{equation*} f(x,y,z) = x^2 + y^2 + z^2 -1\text{.} \end{equation*}

Draw a picture of the level set \(M\) of \(f\text{,}\) and show that it is a differentiable hypersurface.

Solution

The level set of \(f\) is

\begin{equation*} M = \{ (x,y,z) \in \mathbb{R}^3 : x^2 + y^2 + z^2 - 1 = 0 \} \end{equation*}

This is called the 2-sphere, and is pictured below: tikzalign

To show it is a hypersurface, we first compute \(\nabla f\) at a general point \(\ve{r} = (x,y,z) \in \mathbb{R}^3\text{:}\)

\begin{equation*} \nabla_\ve{r} f = (2x, 2y, 2z) \end{equation*}

This clearly exists everywhere. Moreover, for a point \(\ve{p} = (a, b, c)\) on \(M\text{,}\) we have

\begin{equation*} \nabla_\ve{p} f = (2a, 2b, 2c) \neq (0, 0, 0) \end{equation*}

since \(a^2 + b^2 + c^2 = 1\text{.}\) Hence \(M\) is a hypersurface.

Example 3.0.5

Let \(f : \mathbb{R}^3 \rightarrow \mathbb{R}\) be the function

\begin{equation*} f(x,y,z) = (2-\sqrt{x^2 + y^2})^2 + z^2 - 1\text{.} \end{equation*}

Draw a picture of the level set \(M\) of \(f\text{,}\) and show that it is a hypersurface.

Solution

The zero set of \(f\) is

\begin{equation*} M = \{ (x,y,z) \in \mathbb{R}^3 : (2-\sqrt{x^2 + y^2})^2 + z^2 - 1 = 0 \} \end{equation*}

This is a torus, and is pictured below: tikzalign

To show \(M\) is a hypersurface, we first compute \(\nabla f\) at a general point \(\ve{r} = (x,y,z) \in \mathbb{R}^3\text{:}\)

\begin{equation*} \nabla_\ve{r} f = \left( \frac{2x(2 - \sqrt{x^2 + y^2)}}{\sqrt{x^2 + y^2}}, \, \frac{2y(2 - \sqrt{x^2 + y^2)}}{\sqrt{x^2 + y^2}}, \, 2z \right) \end{equation*}

Note that \(\nabla f\) doesn't exist everywhere in \(\mathbb{R}^3\text{,}\) since there is a problem when \(x^2 + y^2 = 0\) (i.e. on the \(z\)-axis). But for a point \(\ve{p} = (a, b, c)\) on \(M\text{,}\) we cannot have \(a^2 + b^2 = 0\) since that implies, using the equation for \(M\text{,}\) that

\begin{equation*} (2 - \sqrt{0})^2 + c^2 - 1 = 0\text{,} \end{equation*}

in other words, \(c^2 = -3\) which is impossible. So \(\nabla_\ve{p} f\) exists everywhere on \(M\text{.}\) Moreover, suppose \(\nabla_\ve{p} f = (0,0,0)\) and \(\ve{p}\) lies on \(M\text{.}\) Then in particular \(2c = 0\) so \(c=0\text{,}\) and hence, using the equation for \(M\text{,}\)

\begin{equation*} (2 - \sqrt{a^2 + b^2})^2 = 1 \end{equation*}

which implies that \(a\) and \(b\) cannot both be zero. This is a contradiction, hence \(\nabla_\ve{p} \neq \ve{0}\) everywhere on \(M\text{.}\) So \(M\) is a hypersurface.

Definition 3.0.6

Let \(M \subset \mathbb{R}^{n+1}\) be a hypersurface associated to a function \(f : \mathbb{R}^{n+1} \rightarrow \mathbb{R}\text{.}\) The tangent space of \(M\) at \(\ve{p} \in M\) is:

\begin{equation*} T_\ve{p} M := \{ \ve{v} \in \mathbb{R}^{n+1} \, : \, \nabla_\ve{p} f \dotp \ve{v} = 0 \} \end{equation*}

From Multivariable Calculus, you know that we can equivalently express the tangent space to \(M\) at \(p\) as

\begin{align*} T_\ve{p} M \amp = \{ \ve{v} \in \mathbb{R}^{n+1} \, : \, D_\ve{v} f (\ve{p}) = 0 \}\\ \amp = \left\{ \ve{v} \in \mathbb{R}^{n+1} \, : \, \frac{d}{dt}\Big|_{t=0} f(\ve{p} + t \ve{v}) = 0 \right\} \end{align*}

Solution

\begin{equation*} \nabla_\ve{p} f = (2 \sqrt{2}, -2)\text{.} \end{equation*}

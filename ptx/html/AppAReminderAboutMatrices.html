Appendix A Reminder about matrices

¶

Let us recall a few things about matrices, and set up our notation.

An \(n \times m\) matrix \(\mat{A}\) is just a rectangular array of numbers, with \(n\) rows and \(m\) columns:

\begin{equation*} \mat{A} = \left[ \begin{array}{cccc} \mat{A}_{11} \amp \mat{A}_{12} \amp \cdots \amp \mat{A}_{1m} \\ \mat{A}_{21} \amp \mat{A}_{22} \amp \cdots \amp \mat{A}_{2m} \\ \vdots \amp \amp \ddots \amp \vdots \\ \mat{A}_{n1} \amp \mat{A}_{n2} \amp \cdots \amp \mat{A}_{nm} \end{array} \right] \end{equation*}

I will always write matrices in ‘sans serif’ font, eg. \(\mat{A}\text{.}\) It is difficult to ‘change fonts’ in handwritten text, but I encourage you to at least reserve the letters \(\mat{A}\text{,}\) \(\mat{B}\text{,}\) \(\mat{C}\text{,}\) etc. for matrices, and \(S\text{,}\) \(T\text{,}\) etc. for linear maps!

Two \(n \times m\) matrices \(\mat{A}\) and \(\mat{B}\) can be added, to get a new \(n \times m\) matrix \(\mat{A} + \mat{B}\text{:}\)

\begin{equation*} (\mat{A} + \mat{B})_{ij} := \mat{A}_{ij} + \mat{B}_{ij} \end{equation*}

There is the zero \(n \times m\) matrix:

\begin{equation*} \mat{0} = \left[ \begin{array}{cccc} 0 \amp 0 \amp \cdots \amp 0 \\ 0 \amp 0 \amp \cdots \amp 0 \\ \vdots \amp \amp \ddots \amp \vdots \\ 0 \amp 0 \amp \cdots \amp 0 \end{array} \right] \end{equation*}

You can also multiply an \(n \times m\) matrix \(\mat{A}\) by a scalar \(k\text{,}\) to get a new \(n \times m\) matrix \(k \mat{A}\text{:}\)

\begin{equation*} (k\mat{A})_{ij} := k \mat{A}_{ij} \end{equation*}
Lemma A.0.1

  1. Equipped with these operations, the set \(\Mat_{n, m}\) of all \(n \times m\) matrices is a vector space.

  2. The dimension of \(\Mat_{nm}\) is \(nm\text{,}\) with basis given by the matrices

    \begin{equation*} \mat{E}_{ij} , i=1 \ldots n, \, j = 1 \ldots m \end{equation*}

    which have a \(1\) in the \(i\)th row and \(j\)th column and zeroes everywhere else.

Proof

Left as an exercise.

Example A.0.2

\(\Mat_{2,2}\) has basis

\begin{equation*} \mat{E}_{11} = \left[ \begin{array}{cc} 1 \amp 0 \\ 0 \amp 0 \end{array} \right], \, \mat{E}_{12} = \left[ \begin{array}{cc} 0 \amp 1 \\ 0 \amp 0 \end{array} \right], \mat{E}_{21} = \left[ \begin{array}{cc} 0 \amp 0 \\ 1 \amp 0 \end{array} \right], \mat{E}_{22} = \left[ \begin{array}{cc} 0 \amp 0 \\ 0 \amp 1 \end{array} \right]\text{.} \end{equation*}

Usually \(\mat{A}\) is a matrix, and \(\mat{A}_{ij}\) is the element of the matrix at position \((i,j)\text{.}\) But now \(\mat{E}_{ij}\) is a matrix in its own right! Its element at position \((k,l)\) will be written as \((\mat{E}_{ij})_{kl}\text{.}\) I hope you don't find this too confusing. In fact, we can write down an elegant formula for the elements of \(\mat{E}_{ij}\) using the Kronecker delta symbol:

\begin{equation} (\mat{E}_{ij})_{kl} = \delta_{ik} \delta_{jl}\label{kronecker_for_matrix_basis-2}\tag{A.0.1} \end{equation}
Example A.0.3

We will write \(\Col_n\) for the vector space \(\Mat_{n,1}\) of \(n\)-dimensional column vectors, and we will write the standard basis vectors \({\mathbf \mat{E}}_{i1}\) of \(\Col_n\) more simply as \(\col{e}_i\text{:}\)

\begin{equation*} \col{e}_1 := \left[ \begin{array}{c} 1 \\ 0 \\ \vdots \\ 0 \end{array} \right], \, \col{e}_2 := \left[ \begin{array}{c} 0 \\ 1 \\ \vdots \\ 0 \end{array} \right], \, \ldots, \, \col{e}_n := \left[ \begin{array}{c} 0 \\ 0 \\ \vdots \\ 1 \end{array} \right] \text{.} \end{equation*}

Vectors in \(\Col_n\) will be written in bold sans-serif font eg. \(\col{v} \in \Col_n\text{.}\)

Equipped with these operations, the set \(\Mat_{n, m}\) of all \(n \times m\) matrices is a vector space with dimension \(nm\) . We write \(\Col_n\) for the vector space \(\Mat_{n,1}\) of \(n\)-dimensional column vectors.

The most important operation is matrix multiplication. An \(n \times k\) matrix \(\mat{A}\) can be multiplied from the right with a \(k \times m\) matrix \(\mat{B}\) to get a \(n \times m\) matrix \(\mat{AB}\text{,}\)

\begin{align*} \end{align*}

by defining the entries of \(\mat{AB}\) to be

\begin{equation*} (\mat{AB})_{ij} := \mat{A}_{i1} \mat{B}_{1j} + \mat{A}_{i2} \mat{B}_{2j} + \cdots + \mat{A}_{ik} \mat{B}_{kj}\text{.} \end{equation*}
Proposition A.0.4

The above operations on matrices satisfy the following rules whenever the sums and products involved are defined:

  1. \((\mat{A} + \mat{B}) \mat{C} = \mat{A} \mat{C} + \mat{B} \mat{C}\)

  2. \(\mat{A} (\mat{B} + \mat{C}) = \mat{A} \mat{B} + \mat{A} \mat{C}\)

  3. \((k \mat{A}) \mat{B} = \mat{A} (k \mat{B}) = k (\mat{A} \mat{B})\)

  4. \((\mat{A} \mat{B}) \mat{C} = \mat{A} (\mat{B} \mat{C})\)

Proof

The proofs of (1) - (3) are all routine checks which you have hopefully done before. Let us prove (4), to practice using \(\Sigma\)-notation! Suppose \(\mat{A}\text{,}\) \(\mat{B}\) and \(\mat{C}\) have sizes \(n \times k\text{,}\) \(k \times r\) and \(r \times m\) respectively, so that the matrix products make sense. Then:

\begin{align*} ((\mat{A}\mat{B})\mat{C})_{ij} \amp = \sum_{p=1}^r (\mat{A} \mat{B})_{ip} \mat{C}_{pj}\\ \amp = \sum_{p=1}^r \left(\sum_{q=1}^k \mat{A}_{iq} \mat{B}_{qp} \right) \mat{C}_{pj}\\ \amp = \sum_{p,q} \mat{A}_{iq} \mat{B}_{qp} \mat{C}_{pj}\\ \amp = \sum_{q=1}^k \mat{A}_{iq} \left( \sum_{p=1}^r \mat{B}_{qp} \mat{C}_{pj}\right)\\ \amp = \sum_{q=1}^k \mat{A}_{iq} (\mat{B} \mat{C} )_{qj}\\ \amp = (\mat{A} (\mat{B} \mat{C}))_{ij}\text{.} \end{align*}

I hope the \(\Sigma\)-notation does not confuse you in the above proof! Let me write out the exact same proof without \(\Sigma\)-notation, in the simple case where \(\mat{A}\text{,}\) \(\mat{B}\) and \(\mat{C}\) are all \(2 \times 2\) matrices, and we are trying to calculate, say, the entry at position \(11\text{.}\)

\begin{align*} ((\mat{A}\mat{B})\mat{C})_{11} \amp = (\mat{A}\mat{B})_{11} \mat{C}_{11} + (\mat{A} \mat{B})_{12} \mat{C}_{21}\\ \amp = (\mat{A}_{11} \mat{B}_{11} + \mat{A}_{12} \mat{B}_{21}) \mat{C}_{11} + (\mat{A}_{11} \mat{B}_{12} + \mat{A}_{12} \mat{B}_{22}) \mat{C}_{21}\\ \amp = \mat{A}_{11} \mat{B}_{11} \mat{C}_{11} + \mat{A}_{12} \mat{B}_{21} \mat{C}_{11} + \mat{A}_{11} \mat{B}_{12} \mat{C}_{21} + \mat{A}_{12} \mat{B}_{22} \mat{C}_{21}\\ \amp = \mat{A}_{11} (\mat{B}_{11} \mat{C}_{11} + \mat{B}_{12} \mat{C}_{21}) + \mat{A}_{12} ( \mat{B}_{21} \mat{C}_{11} + \mat{B}_{22} \mat{C}_{21})\\ \amp = \mat{A}_{11} (\mat{B} \mat{C})_{11} + \mat{A}_{12} (\mat{B} \mat{C})_{21}\\ \amp = (\mat{A} (\mat{B} \mat{C}))_{11}\text{.} \end{align*}

Do you understand the \(\Sigma\)-notation proof now? The crucial step (going from the second to the fourth lines) is called exchanging the order of summation.

The transpose of an \(n \times m\) matrix \(\mat{A}\) is the \(m \times n\) matrix \(\mat{A}^T\) whose entries are given by

\begin{equation*} (\mat{A}^T)_{ij} := \mat{A}_{ji}\text{.} \end{equation*}

Section 1.5 Some results about abstract vector spaces

Â¶

It is time to use the rules of a vector space to prove some general results.

We are about to do our first formal proof in the course!

Our first lemma shows that the zero vector \(\ve{0}\) is the unique vector in \(V\) which 'behaves like a zero vector'. More precisely:

Lemma 1.5.1

Suppose \(V\) is a vector space with zero vector \(\ve{0}\text{.}\) If \(\ve{0}'\) is a vector in \(V\) satisfying

\begin{equation} \ve{0}' + \ve{v} = \ve{v} \text{ for all }\ve{v} \in V\label{uniqueness_zero}\tag{1.5.1} \end{equation}

then \(\ve{0}' = \ve{0}\text{.}\)

Proof
\begin{align*} \ve{0} \amp = \ve{0}' + \ve{0} \amp \amp \text{using } \knowl{./knowl/uniqueness_zero.html}{\text{(1.5.1)}} \text{ with } \ve{v}=\ve{0} \\ \amp = \ve{0}' \amp \amp \text{( } \knowl{./knowl/R3b.html}{\text{R3bÂ }}\text{)} \end{align*}
Definition 1.5.2

If \(V\) is a vector space, we define the additive inverse of a vector \(\ve{v} \in V\) as

\begin{equation*} - \ve{v} := (-1) \smul\ve{v} \end{equation*}
Lemma 1.5.3

If \(V\) is a vector space, then for all \(\ve{v} \in V\text{,}\)

\begin{equation} -\ve{v} + \ve{v} = \ve{0} \mbox{ and } \ve{v} + (-\ve{v}) = \ve{0}\text{.}\tag{1.5.2} \end{equation}
Proof
\begin{align*} -\ve{v} + \ve{v} \amp = (-1) \smul\ve{v} + \ve{v} \amp \amp \mbox{(using defn of \(- \ve{v}\))}\\ \amp = (-1) \smul\ve{v} + 1 \smul\ve{v} \amp \amp \mbox{(R7)}\\ \amp = (-1 + 1) \smul\ve{v} \amp \amp \mbox{(R5)}\\ \amp = 0 \smul\ve{v}\\ \amp = \ve{0} \amp \amp \mbox{(R8)} \end{align*}

In addition,

\begin{align*} \ve{v} + (-\ve{v}) \amp = -\ve{v} + \ve{v} \amp \amp \mbox{(R1)}\\ \amp = \ve{0} \amp \amp \mbox{(by previous proof)} \end{align*}
Lemma 1.5.4

Suppose that two vectors \(\ve{w}\) and \(\ve{v}\) in a vector space satisfy \(\ve{w} + \ve{v} = \ve{0}\text{.}\) Then \(\ve{w} = -\ve{v}\text{.}\)

Proof
\begin{align*} \ve{w} \amp = \ve{w} + \ve{0} \amp \amp \mbox{(R3b)}\\ \amp = \ve{w} + (\ve{v} + -\ve{v}) \amp \amp \mbox{by } \knowl{./knowl/add_inv.html}{\text{LemmaÂ 1.5.3}}\\ \amp = (\ve{w} + \ve{v}) + -\ve{v}) \amp \amp \mbox{(R2)}\\ \amp = \ve{0} + -\ve{v} \amp \amp \mbox{(by assumption)}\\ \amp = -\ve{v} \amp \amp \mbox{(R3a)} \text{.} \end{align*}

Let us prove two more lemmas, for practice.

Lemma 1.5.5

Let \(V\) be a vector space and \(k\) any scalar. Then

\begin{equation*} k \smul\ve{0} = \ve{0} \end{equation*}
Proof
\begin{align*} k \smul\ve{0} \amp = k \smul(0 \smul\ve{0}) \amp \text{(R8 for } \ve{v}=\ve{0}\text{)} \\ \amp = ((k)(0))\smul\ve{0} \amp \text{(R6)} \\ \amp = 0 \smul\ve{0} \amp \text{(} (k)(0)=0 \text{ for any real number } k \text{)} \\ \amp = \ve{0} \amp \text{(R8 for } \ve{v}=\ve{0}\text{)} \end{align*}
Lemma 1.5.6

Suppose that \(\ve{v}\) is a vector in a vector space \(V\) and that \(k\) is a scalar. Then

\begin{equation*} k \smul\ve{v} = \ve{0} \Leftrightarrow k = 0 \mbox{ or } \ve{v} = \ve{0}\text{.} \end{equation*}
Proof

(Proof of \(\Leftarrow\)). Suppose \(k=0\text{.}\) Then \(k \smul\ve{v} = 0 \smul\ve{v} = \ve{0}\) by R8 of a vector space. On the other hand, suppose \(\ve{v} = \ve{0}\text{.}\) Then \(k \smul\ve{v} = k \smul\ve{0} = \ve{0}\) by ExerciseÂ 1.5.2.

(Proof of \(\Rightarrow\)). Suppose \(k \smul\ve{v} = \ve{0}\text{.}\) There are two possibilities: either \(k = 0\text{,}\) or \(k \neq 0\text{.}\) If \(k=0\text{,}\) then we are done. If \(k \neq 0\text{,}\) then \(\frac{1}{k}\) exists and we can multiply both sides by it:

\begin{align*} k \smul\ve{v} \amp = \ve{0}\\ \therefore \frac{1}{k} \smul(k \smul\ve{v}) \amp = \frac{1}{k} \smul\ve{0} \amp \amp \mbox{(Multiplied both sides by \(\frac{1}{k}\))}\\ \therefore \left(\frac{1}{k} k\right) \smul\ve{v} \amp = \ve{0} \amp \amp \mbox{(On the LHS, we used R6. On the RHS, we used } \knowl{./knowl/mult_by_zero_ex.html}{\text{ExerciseÂ 1.5.2}})\\ \therefore 1 \smul\ve{v} \amp = \ve{0} \amp \amp \mbox{(using \(\frac{1}{k} k = 1\))}\\ \therefore \ve{v} \amp = \ve{0} \amp \amp \mbox{(R7)} \end{align*}

Hence in the case \(k \neq 0\) we must have \(\ve{v} = \ve{0}\text{,}\) which is what we wanted to show.

Example 1.5.7

Let us practice using the rules of a vector space to perform everyday calculations. For instance, suppose that we are trying to solve for the vector \(\ve{x}\) appearing in the following equation:

\begin{equation} \ve{v} + 7 \smul\ve{x} = \ve{w}\tag{1.5.3} \end{equation}

We do this using the rules as follows:

\begin{align*} \ve{v} + 7 \smul\ve{x} \amp = \ve{w}\\ \therefore -\ve{v} + (\ve{v} + 7 \smul\ve{x}) \amp = -\ve{v} + \ve{w} \amp \amp \mbox{(Added \(-\ve{v}\) on left to both sides)}\\ \therefore (-\ve{v} + \ve{v}) + 7 \smul\ve{x} \amp = -\ve{v} + \ve{w} \amp \amp \mbox{(used R2 on LHS)}\\ \therefore \ve{0} + 7 \smul\ve{x} \amp = -\ve{v} + \ve{w} \amp \amp \mbox{(used } \knowl{./knowl/add_inv.html}{\text{LemmaÂ 1.5.3}} \mbox{ on LHS)}\\ \therefore 7 \smul\ve{x} \amp = -\ve{v} + \ve{w} \amp \amp \mbox{(used R3a on LHS)}\\ \therefore \frac{1}{7} \smul(7 \smul\ve{x}) \amp = \frac{1}{7} \smul(-\ve{v} + \ve{w}) \amp \amp \mbox{(scalar multiplied both sides by \(\frac{1}{7}\))}\\ \therefore (\frac{1}{7} 7) \smul \ve{x} \amp = \frac{1}{7} \smul(-\ve{v} + \ve{w}) \amp \amp \mbox{(used R6 on LHS)}\\ \therefore 1 \smul\ve{x} \amp = \frac{1}{7} \smul(-\ve{v} + \ve{w}) \amp \amp \mbox{(multiplied \(\frac{1}{7}\) with \(7\))}\\ \therefore \ve{x} \amp = \frac{1}{7} \smul(-\ve{v} + \ve{w}) \amp \amp \mbox{(R7)} \end{align*}

As the course goes on we will leave out all these steps. But it is important for you to be able to reproduce them all, if asked to do so!

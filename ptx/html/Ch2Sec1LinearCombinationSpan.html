Section 2.1 Linear combinations and span

Â¶

We start with some basic definitions.

Definition 2.1.1

A linear combination of a finite collection \(\ve{v}_1, \ldots, \ve{v}_n\) of vectors in a vector space \(V\) is a vector of the form

\begin{equation} a_1 \ve{v}_1 + a_2 \ve{v}_2 + \cdots + a_b \ve{v}_n\tag{2.1.1} \end{equation}

where \(a_1, a_2, \ldots, a_n\) are scalars. If all the scalars \(a_i\) are \(0\text{,}\) we say that it is the trivial linear combination.

Example 2.1.2 First example of a linear combination

In \(\mathbb{R}^3\text{,}\) \((6, 2, -14)\) is a linear combination of \((-3, 1, 2)\) and \((-2, 0, 3)\) because

\begin{equation*} (6, 2, -14) = 2 (-3, 1, 2) -6 (-2, 0, 3)\text{.} \end{equation*}
Example 2.1.3 Checking if a vector is a linear combination of other vectors

In \(\mathbb{R}^4\text{,}\) is \(\ve{v}=(2, -1, 3, 0)\) a linear combination of

\begin{equation*} \ve{v}_1 = (1, 3, 2, 0), \ve{v}_2=(5, 1, 2, 4), \text{ and } \ve{v}_3 = (-1, 0, 2, 1) \text{?} \end{equation*}

To check this, we need to check if the equation

\begin{equation} \ve{v} = a_1 \ve{v}_1 + a_2 \ve{v}_2 + a_3 \ve{v}_3,\label{first_span_ex_eqn}\tag{2.1.2} \end{equation}

which is an equation in the unknowns \(a_1, a_2, a_3\text{,}\) has any solutions. Let us write out (2.1.2) explicitly:

\begin{align} (2, -1, 3, 0) \amp = a_1 (1, 3, 2, 0) + a_2 (5, 1, 2, 4) + a_3 (-1, 0, 2, 1) \tag{2.1.3}\\ \therefore (2, -1, 3, 0) \amp = (a_1 + 5a_2 -a_3, 3a_1 + a_2, 2a_1 + 2a_2 + 2a_3, 4a_2 + a_3) \label{last_eqn_in_lin_comb_ex}\tag{2.1.4} \end{align}

(2.1.4) is an equation between two vectors in \(\mathbb{R}^4\text{.}\) Two vectors in \(\mathbb{R}^4\) are equal if and only if their corresponding coefficients are equal. So, (2.1.2) is equivalent to the following system of simultaneous linear equations:

\begin{align} a_1 + 5a_2 -a_3 \amp = -2 \label{sim_lin_eqns_first_span_1ex}\tag{2.1.5}\\ 3a_1 + a_2 \amp = -1 \tag{2.1.6}\\ 2a_1 + 2a_2 + 2a_3 \amp = 3\tag{2.1.7}\\ 4a_2 + a_3 \amp = 0 \label{sim_lin_eqns_first_span_4ex}\tag{2.1.8} \end{align}

In other words, our question becomes: do equations (2.1.5)â(2.1.8) have a solution?

This is the kind of problem you already know how to solve by hand, from first year. We can also use SageMath to do it for us. We simply tell it what our unknown variables are, and then ask it to solve the equation. Press Evaluate (Sage) to see the result.

SageMath returns an empty list []. In other words, there are no solutions to equations (2.1.5)â(2.1.8). Therefore \(\ve{v}\) cannot be expressed as a linear combination of \(\ve{v}_1, \ve{v}_2, \ve{v}_3\text{.}\)

Example 2.1.4 Checking if a polynomial is a linear combination of other polynomials

In \(\Poly_2\text{,}\) can \(p=x^2 - 1\) be expressed as a linear combination of

\begin{equation*} p_1 = 1+x^2, \, p_2 = x-3, \, p_3 = x^2+x+1, \, p_4 = x^2 + x - 1 \text{?} \end{equation*}

To check this, we need to check if the equation

\begin{equation} p = a_1 p_1 + a_2 p_2 + a_3 p_3 + a_4 p_4,\label{second_span_ex_eqn}\tag{2.1.9} \end{equation}

which is an equation in the unknowns \(a_1, a_2, a_3, a_4\text{,}\) has any solutions. Let us write out (2.1.9) explicitly, grouping together powers of \(x\text{:}\)

\begin{align*} p \amp = a_1 p_1 + a_2 p_2 + a_3 p_3 + a_4 p_4 \\ \therefore x^2 - 1 \amp = a_1 (1+x^2) + a_2 (x-3) + a_3 (x^2 + x + 1) + a_4 (x^2 + x - 1) \\ \therefore -1 + x^2 \amp = (a_1 -3a_2 + a_3 - a_4) + (a_2 + a_3 + a_4) x + (a_1 + a_3 + a_4)x^2 \end{align*}

Now, two polynomials are equal if and only if all their coefficients are equal. So, (2.1.9) is equivalent to the following system of simultaneous linear equations:

\begin{align} a_1 - 3a_2 + a_3 - a_4 \amp = -1 \label{sim_lin_eqns_second_span_1ex}\tag{2.1.10}\\ a_2 + a_3 + a_4 \amp = 0\tag{2.1.11}\\ a_1 + a_3 + a_4 \amp = 1 \label{sim_lin_eqns_second_span_4ex}\tag{2.1.12} \end{align}

In other words, our question becomes: do equations (2.1.10)â(2.1.12) have a solution? We ask SageMath.

On my computer, Sage returns:

[[a1 == 2*r1 + 2/3, a2 == (2/3), a3 == -r1 + 1/3, a4 == r1]]

Here, r1 and r2 are to be interpreted as free paramters. I'm going to call them \(s\) and \(t\) instead, because that's what we usually call our free parameters! So, equations (2.1.10)â(2.1.12) have infinitely many solutions, parameterized by two free parameters \(s\) and \(t\text{.}\) In particular, there exists at least one solution. For instance, if we take \(s=2\) and \(t = 1\) (a totally arbitrary choice!), we get the following solution:

\begin{gather} a_1 = \frac{8}{3}, \, a_2 = \frac{2}{3}, a_3 = -\frac{5}{3}, a_4 = 1 \tag{2.1.13}\\ \text{i.e. } p = \frac{8}{3} p_1 + \frac{2}{3} p_2 - \frac{5}{3} p_3 + p_4 \label{soln_poly_span_prob}\tag{2.1.14} \end{gather}

You should expand out the right hand side of (2.1.14) by hand and check that it indeed is equal to \(p\text{.}\)

We conclude that \(p\) can indeed be expressed as a linear combination of \(p_1\text{,}\) \(p_2\text{,}\) \(p_3\) and \(p_4\text{.}\)

Example 2.1.5

Define the functions \(\ve{f}, \ve{f}_1, \ve{f}_2 \in \Diff\) by

\begin{equation*} \ve{f}(x) = \cos^3 x, \ve{f}_1 (x) = \cos (x), \ve{f}_2 (x) = \cos (3x)\text{.} \end{equation*}

Then \(\ve{f}\) is a linear combination of \(\ve{f}_1\) and \(\ve{f}_2\) because of the identity \(\cos(3x) = \frac{1}{4} (3 \cos x + \cos(3x))\text{.}\) See ExampleÂ 1.6.18. In other words,

\begin{equation*} \ve{f} = \frac{3}{4} \ve{f}_1 + \frac{1}{4} \ve{f}_2 \text{.} \end{equation*}

This example shows that \(\ve{f}\) is also a trigonometric polynomial, even though its original formula \(\ve{f}(x) = \cos(3x)\) was not in the form (1.6.3).

Definition 2.1.6

We say that a list of vectors \(\basis{B} = \bopen \ve{v}_1, \ve{v}_2, \ldots, \ve{v}_n \bclose \) in a vector space \(V\) spans \(V\) if every vector \(\ve{v} \in V\) is a linear combination of the vectors from \(\basis{B}\text{.}\)

Example 2.1.7

\(\mathbb{R}^2\) is spanned by

\begin{equation*} \ve{e}_1 := (1, \, 0), \, \, \ve{e}_2 := (0, 1) \end{equation*}

because every vector \(\ve{v} = (a_1, \, a_2)\) can be written as the linear combination

\begin{equation*} \ve{v} = a_1 \ve{e}_1 + a_2 \ve{e}_2\text{.} \end{equation*}
Example 2.1.8 Checking if a list of vectors space the vector space

Is \(\mathbb{R}^2\) spanned by the following list of vectors?

\begin{equation*} \ve{f}_1 := (-1, 2), \, \ve{f}_2 := (1, 1), \, \ve{f}_3 := (2, -1) \end{equation*}

Solution

To check this, we need check if every vector \(\ve{v} \in V\) can be written as a linear combination of \(\ve{f}_1, \ve{f}_2\) and \(\ve{f}_3\text{.}\)

So, let \(\ve{v} = (v_1,v_2) \) be a fixed, but arbitrary, vector in \(\mathbb{R}^2\text{.}\) We need to check if the following equation has a solution for \(a_1, a_2, a_3\text{:}\)

\begin{equation} \ve{v} = a_1 \ve{f}_1 + a_2 \ve{f}_2 + a_3 \ve{f}_3\label{eqn_for_R2_span}\tag{2.1.15} \end{equation}

Let us write this equation out explicitly:

\begin{align} \ve{v} \amp = a_1 \ve{f}_1 + a_2 \ve{f}_2 + a_3 \ve{f}_3 \tag{2.1.16}\\ \therefore (v_1, v_2) \amp = a_1 (-1, 2) + a_2 (1, 1) + a_3 (2, -1) \tag{2.1.17}\\ \therefore (v_1, v_2) \amp = (-a_1 + a_2 + 2a_3, 2a_1 + a_2 - a_3) \label{last_eqn_span_example}\tag{2.1.18} \end{align}

(2.1.18) is an equation between two vectors in \(\mathbb{R}^2\text{.}\) Two vectors in \(\mathbb{R}^2\) are equal if and only if their corresponding coefficients are equal. So, (2.1.18) is equivalent to the following system of simultaneous linear equations:

\begin{align} -a_1 + a_2 + 2a_3 \amp = v_1 \label{sim_lin_eqns_3ex}\tag{2.1.19}\\ 2a_1 + a_2 - a_3 \amp = v_2\label{sim_lin_eqns_3ex2}\tag{2.1.20} \end{align}

In other words, the original question

Is \(\mathbb{R}^2\) spanned by \(\ve{f}_1, \ve{f}_2, \ve{f}_3 \) ?

is equivalent to the question

Can we always solve (2.1.19)â(2.1.20) for \(a_1, a_2, a_3\text{,}\) no matter what the fixed constants \(\ve{v}_1, \ve{v}_2\) are?

You already know how to solve simultaneous linear equations such as (2.1.19)â(2.1.20) by hand:

\begin{align} -a_1 + a_2 + 2a_3 \amp = v_1 \tag{2.1.21}\\ 2a_1 + a_2 - a_3 \amp = v_2\tag{2.1.22}\\ \tag{2.1.23}\\ \therefore -a_1 + a_2 + 2a_3 \amp = v_1 \tag{2.1.24}\\ 3a_2 + 3a_3 \amp = 2v_1 + v_2 \amp \amp R2 \rightarrow R2 + 2R1 \tag{2.1.25}\\ \tag{2.1.26}\\ \text{Let } a_3 \amp = t \label{span_ex_aa1}\tag{2.1.27}\\ \therefore a_2 \amp = \frac{1}{3}(2v_1 + v_2) - t \tag{2.1.28}\\ \therefore a_1 \amp = -\frac{1}{3} (-v_1 + v_2) + t \label{span_ex_aa2}\tag{2.1.29} \end{align}

In other words, no matter what \(v_1, v_2\) are, there are always infinitely many solutions (they are parameterized a free parameter \(t\)) to (2.1.19)â(2.1.20), and hence to our original equation (2.1.15). That is, we can express any \(\ve{v} \in \mathbb{R}^2\) as a linear combination of the vectors \(\ve{f}_1, \ve{f}_2, \ve{f}_3\text{...}\) and in fact there are infinitely many ways to do it, parameterized by a free parameter \(t\text{!}\)

For instance, suppose we try to write \(\ve{v} = (2,3)\) as a linear combination of \(\ve{f}_1, \ve{f}_2, \ve{f}_3\text{.}\) If we take our general solution ((2.1.27)â(2.1.29)), and set \(t=0\text{,}\) then we get

\begin{gather*} a_1 = \frac{1}{3}, \, a_2 = \frac{7}{3}, \, a_3 = 0 \\ \text{i.e. } \ve{v} = \frac{1}{3} \ve{f}_1 + \frac{7}{3} \ve{f}_2 \end{gather*}

Or we could take, say, \(t = 1\text{.}\) Then our solution will be

\begin{gather*} a_1 = \frac{4}{3}, \, a_2 = \frac{4}{3}, \, a_3 = 1 \\ \text{i.e. } \ve{v} = \frac{4}{3} \ve{f}_1 + \frac{4}{3} \ve{f}_2 + \ve{f}_3 \end{gather*}

There are infinitely many solutions. But the important point is that there is always a solution to (2.1.15), no matter what \(\ve{v}\) is. Therefore, the vectors \(\ve{f}_1, \ve{f}_2, \ve{f}_3\) indeed span \(\mathbb{R}^2\text{.}\)

Finally, let us solve this problem using SageMath. Working by hand, we arrive at the simultanous linear equations (2.1.19)â(2.1.20), and then put it into a Sage cell:

Note that I needed to tell Sage that v1 and v2 are variables, and that I am asking it to solve for a1, a2 and a3. On my computer, Sage outputs:

[[a1 == r1 - 1/3*v1 + 1/3*v2, a2 == -r1 + 2/3*v1 + 1/3*v2, a3 == r1]]

Here, r1 is to be interpreted as our free parameter \(t\text{.}\) So Sage is giving us the same solution as we found by hand, (2.1.27)â(2.1.29).

Example 2.1.10

\(\mathbb{R}^n\) is spanned by

\begin{equation} \ve{e}_1 := (1, 0, \ldots, 0), \,\,\, \ve{e}_2 := (0, 1, \ldots, 0), \,\,\, \ldots, \,\,\, \ve{e}_n := (0, 0, \ldots, 0, 1)\label{defn_of_e_i}\tag{2.1.30} \end{equation}

because every vector \(\ve{v} = (a_1, \,a_2, \, \ldots, \, a_n)\) can be written as the linear combination

\begin{equation} \ve{v} = a_1 \ve{e}_1 + a_2 \ve{e}_2 + \cdots + a_n \ve{e}_n\text{.}\label{standard_basis_R_n}\tag{2.1.31} \end{equation}
Checkpoint 2.1.11

Check equation (2.1.31).

Solution
\begin{equation*} \ve{v} = a_1 \ve{e}_1 + a_2 \ve{e}_2 + \cdots + a_n \ve{e}_n = a_1(1, \ldots, 0) + \cdots a_n(0,\ldots,1) \\ = (a_1 , \ldots , 0 ) + \cdots + (0 ,\ldots, a_n) = (a_1, \ldots, a_n ) \end{equation*}

Subsection Exercises

1

Recall from \(1^\text{st}\) year that a function \(f: \mathbb R \to \mathbb R \) is even if \(f(-x) = f(x) \) and odd if \(f(-x) = -f(x) \text{.}\) Show that every vector in the vector space \(\Fun(\mathbb R)\) is a linear combination of an even function and an odd function.

Solution

The solution is relatively straightforward. Define the following two functions:

\begin{equation*} f_{\text{even}} (x)= \frac{1}{2}\left(f(x) + f(-x) \right), \quad f_{\text{odd}} (x)= \frac{1}{2}\left( f(x) - f(-x) \right) \end{equation*}

It is easy to see that, as the names suggest, \(f_{\text{even}} \) is an even function and \(f_{\text{odd}} (x) \) is an odd function. We simply sum \(f_{\text{even}} \) and \(f_{\text{odd}} \text{:}\)

\begin{equation*} f_{\text{even}} (x) + f_{\text{odd}} (x) = \frac{1}{2}\left(f(x) + f(-x) \right) + \frac{1}{2}\left( f(x) - f(-x) \right) = f(x). \end{equation*}
2

Suppose \(\ve v_1, \ve v_2 , \ve v_3, \ve v_4\) spans \(V\text{.}\) Prove that \(\ve v_1 - \ve v_2, \ve v_2 - \ve v_3, \ve v_3 - \ve v_4 , \ve v_4\) also spans \(V\text{.}\)

Solution

If we are given that \(\ve v_1, \ve v_2 , \ve v_3, \ve v_4\) spans \(V\) then to show that any other collection of vectors in \(V\) spans \(V\) it suffices to show that each of \(\ve v_1, \ve v_2 , \ve v_3, \ve v_4\) can be written has a linear combination of the new collection - you should try prove this yourself!

With this observation in hand, the exercise has an easy solution.

\begin{align*} \ve v_1 \amp = (\ve v_1 - \ve v_2) + (\ve v_2 - \ve v_3) + (\ve v_3 - \ve v_4) + \ve v_4 \\ \ve v_2 \amp= (\ve v_2 - \ve v_3) + (\ve v_3 - \ve v_4) + \ve v_4 \\ \ve v_3 \amp= (\ve v_3 - \ve v_4) + \ve v_4 \\ \ve v_4 \amp= \ve v_4 \end{align*}
3

Consider the following polynomials in \(\Poly_2\text{:}\)

\begin{equation*} \ve{r}_1 (x) := 3x^2 - 2, \, \ve{r}_2 (x) := x^2 + x, \, \ve{r}_3 (x) := x + 1, \, \ve{r}_4 (x) := x-1 \end{equation*}

  1. Can the polynomial \(\ve{p}\) with \(\ve{p}(x) = x^2 + 1\) be written as a linear combination of \(\ve{r}_1\text{,}\) \(\ve{r}_2\text{,}\) \(\ve{r}_3\text{,}\) \(\ve{r}_4\text{?}\)

  2. If so, in how many ways can this be done?

Solution
  1. We must set up the appropriate system of linear equations:
    \begin{align*} \amp a\ve r_1(x) + b \ve r_2(x) + c \ve r_3(x) + d \ve r_4(x) \amp = \amp\ve p(x)\\ \implies \amp a(3x^2 - 2) + b(x^2 + x) + c(x+1) + d(x-1) \amp = \amp x^2 + 1 \end{align*}
    After grouping like powers of \(x\) we obtain
    \begin{equation*} x^2(3a + b) + x(b + c + d) + (-2a + c -d ) = x^2 + 1 . \end{equation*}
    We equate coefficients on both sides of the equation to obtain the following system of linear equations:
    \begin{gather*} 3a + b + 0c + 0d = 1,\\ 0a + 1b + 1c + 1d = 0, \\ -2a + 0b + 1c + -1d = 1 . \end{gather*}
    Using your preferred method for solving a system of linear equations (such as Gauss reduction), we obtain a solution set of the form:
    \begin{align*} \amp d \text{ is free},\\ \amp a = 2 + 2d, \\ \amp b = -5 - 6d, \\ \amp c = 5 + 5d. \end{align*}
    And so \(\ve p (x)\) is indeed a linear combination of \(\ve r_1(x),\ve r_2(x),\ve r_3(x),\ve r_4(x)\text{.}\)
  2. Since \(d\) is free in the above solution set, we can write \(\ve p (x)\) as a linear combination of \(\ve r_1(x),\ve r_2(x),\ve r_3(x),\ve r_4(x)\) in an uncountably infinite number of ways (one for each real number!).
4

Suppose that the vectors \(\ve{e}_1\text{,}\) \(\ve{e}_2\text{,}\) \(\ve{e}_3\) and \(\ve{e}_4\) span a vector space \(V\text{.}\) Show that the vectors \(\ve{f}_1 := \ve{e}_2 - \ve{e}_1\text{,}\) \(\ve{f}_2 := \ve{e}_3 - \ve{e}_2\text{,}\) \(\ve{f}_3 := \ve{e}_4 - \ve{e}_3\text{,}\) \(\ve{f}_4 := \ve{e}_4\) also span \(V\text{.}\)

Solution

You could choose to show this directly or we could use a clever approach based on \(\mathbf 2 \text{.}\) From \(\mathbf 2 \text{,}\) we know that \(\ve e_1 - \ve e_2, \ve e_2 - \ve ev_3, \ve e_3 - \ve e_4 , \ve e_4\) must span \(V \text{.}\) But if these vectors span \(V \text{,}\) then non-zero multiples of the vectors also span \(V\text{.}\) Thus \(\ve{f}_1, \ve{f}_2, \ve{f}_3 ,\ve{f}_4 \) must span \(V\text{.}\)

5

Show that the polynomials

\begin{equation*} \ve{q}_0 (x) := 1, \,\, \ve{q}_1 (x):= x, \,\, \ve{q}_2(x) := 2x^2 - 1, \,\, \ve{q}_3(x) := 4x^3 - 3x \end{equation*}

span \(\Poly_3\text{.}\)

Solution

Once again we base our strategy on \(2\text{.}\) Pick a spanning set for \(\Poly_3\text{.}\) We'll use \(1, x , x^2 , x^3\text{,}\) since it's the simplest. \(1, x\) are certainly spanned by \(\ve q_0x), \ve q_1, \ve q_2, \ve q_3\) since \(1 = \ve q_0(x)\) and \(x = \ve q_1(x)\text{.}\) It can easily be seen that

\begin{equation*} x^2 = \frac{1}{2}\ve q_2(x) + \frac{1}{2}\ve q_0(x) \\ x^3 = \frac{1}{4}\ve q_3(x) + \frac{3}{4}\ve q_1(x), \end{equation*}

.completing the proof.

6

Let \(\basis{S} = \bopen \ve{v}_1, \ldots, \ve{v}_n \bclose \) be a list of vectors in a vector space \(V\text{.}\) Suppose that \(\basis{S}\) spans \(V\text{.}\) Suppose that \(w\) is another vector in \(V\text{.}\) Prove that the list of vectors \(\basis{S}' = \bopen \ve{w}, \ve{v}_1, \ldots, \ve{v}_n \bclose\) also spans \(V\text{.}\)

7

Let \(\basis{S} = \bopen \ve{v}_1, \ldots, \ve{v}_n \bclose \) be a list of vectors in a vector space \(V\text{.}\) Suppose that \(\basis{S}\) spans \(V\text{.}\) Suppose that one of the vectors in the list, say \(\ve{v}_r\text{,}\) can be expressed as a linear combination of the preceding vectors:

\begin{equation} \ve{v}_r = a_1 \ve{v}_1 + \cdots + a_{r-1} \ve{v}_{r-1}\label{exercise-span-omission-1}\tag{2.1.32} \end{equation}

Suppose that we remove \(\ve{v}_r\) from \(\basis{S}\text{,}\) to arrive at a new list

\begin{equation*} \basis{T} = \bopen \ve{v}_1, \ldots, \hat{\ve{v}_r}, \ldots, \ve{s}_n \bclose \end{equation*}

Prove that \(\basis{T}\) also spans \(V\text{.}\)

Solution

We must show that every vector \(\ve{v} \in V\) can be written as a linear combination of the vectors from \(\basis{T}\text{.}\) So let \(\ve{v} \in V\text{.}\) Since \(\basis{S}\) spans \(V\text{,}\) we know we can write \(\ve{v}\) as a linear combination of the vectors from \(\basis{S}\text{:}\)

\begin{equation} \ve{v} = b_1 \ve{v}_1 + \cdots + b_r \ve{v}_r + \cdots + b_{n} \ve{v}_n\label{exercise-span-omission-2}\tag{2.1.33} \end{equation}

Substituting (2.1.32) into (2.1.33) gives

\begin{align} \ve{v} \amp = b_1 \ve{v}_1 + \cdots + b_r (a_1 \ve{v}_1 + \cdots + a_{r-1} \ve{v}_{r-1}) + b_{r+1} \ve{v}_{r+1} + \cdots + b_n \ve{v}_n \tag{2.1.34}\\ \amp = (b_1 + b_r a_1) \ve{v}_1 + \cdots + (b_r + b_r a_{r-1}) \ve{v}_{r-1} + b_{r+1} \ve{v}_{r+1} + \cdots + b_n \ve{v}_n \label{exercise-span-omission-3}\tag{2.1.35} \end{align}

Equation (2.1.35) shows that we can express \(\ve{v}\) as a linear combination of the vectors from \(\basis{T}\text{.}\) Hence \(\basis{T}\) spans V.

Section 2.3 Basis and dimension

¶
Definition 2.3.1

A list of vectors \(\basis{B} = \bopen \ve{e}_1, \ve{e}_2,\ldots, \ve{e}_n\bclose\) in a vector space \(V\) is called a basis for \(V\) if it is linearly independent and spans \(V\text{.}\)

Theorem 2.3.2 Invariance of dimension

If \(\basis{B} = \left\{ \ve{b}_1, \ve{b}_2, \ldots, \ve{b}_m \right\}\) and \(\basis{C} = \left\{ \ve{c}_1, \ve{c}_2, \ldots, \ve{c}_n\right\}\) are bases of a vector space \(V\text{,}\) then \(m=n\text{.}\)

Proof

This is a consequence of Proposition 2.2.10 (the Bumping Off Proposition). Since the \(\ve{b}\)-vectors are linearly independent and the \(\ve{c}\)-vectors span \(V\text{,}\) we have \(m \leq n\text{.}\) On the other hand, since the \(\ve{c}\)-vectors are linearly independent and the \(\ve{b}\)-vectors span \(V\text{,}\) we have \(n \leq m\text{.}\) Hence \(m=n\text{.}\)

Definition 2.3.3

A vector space \(V\) is finite-dimensional if there exists a basis \(\basis{B}\) for \(V\text{.}\) In that case, the dimension of \(V\) is the number of vectors in the basis \(\basis{B}\text{.}\) A vector space is infinte-dimensional if it is not finite-dimensional.

Note that the concept of ‘dimension of a vector space’ is only well-defined because of Theorem 2.3.2.

The case of the zero vector space \(Z = \{\ve{0}\}\) is not explicitly handled in Definition 2.3.3 and we treat it as a special case. Namely, we define the dimension of the zero vector space \(Z\) to be \(0\text{.}\) So, by definition, \(Z\) is finite-dimensional, and its dimension equals \(0\text{.}\)

Example 2.3.4 Standard basis for \(\mathbb{R}^n\)

The list of vectors

\begin{equation*} \ve{e}_1 := (1, 0, \ldots, 0), \,\,\, \ve{e}_2 := (0, 1, \ldots, 0), \,\,\, \ldots, \,\,\, \ve{e}_n := (0, 0, \ldots, 0, 1) \end{equation*}

is a basis for \(\mathbb{R}^n\text{.}\) We already saw in Example 2.1.10 that this list spans \(\mathbb{R}^n\text{.}\) We need to check that it is linearly independent. So, suppose that

\begin{equation*} a_1 \ve{e}_1 + a_2 \ve{e}_2 + \cdots + a_n \ve{e}_n = \ve{0}\text{.} \end{equation*}

Expanding out the left hand side in components using the definition of the standard basis vectors \(\ve{e}_i\text{,}\) this becomes the equation

\begin{equation*} (a_1, 0, 0, \ldots, 0) + (0, a_2, 0, \ldots, 0) + \cdots + (0, 0, 0, \ldots, a_n) = (0, 0, 0, \ldots, 0)\text{.} \end{equation*}

In other words, we have

\begin{equation*} (a_1, a_2, a_3, \ldots, a_n) = (0, 0, 0, \ldots, 0) \end{equation*}

which says precisely that \(a_1 = a_2 = a_3 = \cdots = a_n = 0\text{,}\) which is what we needed to prove. Thus the list of vectors \(\ve{e}_1, \ve{e}_2, \ldots, \ve{e}_n\) is linearly independent, and is hence a basis for \(\mathbb{R}^n\text{.}\) So \(\mathbb{R}^n\) has dimension \(n\text{.}\)

Example 2.3.5 A basis for \(\mathbb{R}^4\)

Check whether the following list of vectors

\begin{equation} \ve{v}_1 = (1, 0, 2, -3), \, \ve{v}_2 = (1, 3, -1, 2), \, \ve{v}_3 = (0, 1, 2, -1), \ve{v}_4 = (1, 2, 3, 4)\tag{2.3.1} \end{equation}

is a basis for \(\mathbb{R}^4\text{.}\)

Solution

First we check if the list of vectors is linearly independent. Consider the equation

\begin{align} a_1 \ve{v}_1 + a_2 \ve{v}_2 + a_3 \ve{v}_3 + a_4 \ve{v}_4 \amp = \ve{0} \tag{2.3.2}\\ \therefore a_1 (1, 0, 2,-3) + a_2 (1, 3, -1, 2) + a_3 (0, 1, 2, -1) + a_4 (1, 2, 3, 4) \amp = (0, 0, 0, 0) \tag{2.3.3}\\ \therefore (a_1 - a_2 + a_4, 3a_2 + a_3 + 2a_4, 2a_1 - a_2 + 2a_3 + 3a_4, -3a_1 + 2a_2 -a_3 + 4a_4) \amp = (0,0,0,0) \tag{2.3.4} \end{align}

So the list of vectors is linearly indepedent if and only if the following equations have only the trivial solution \(a_1 = 0, a_2 = 0, a_3 = 0, a_4 = 0\text{:}\)

\begin{align} a_1 - a_2 + a_4 \amp = 0 \label{R4-example-eqn1}\tag{2.3.5}\\ 3a_1 + a_3 + 2a_4 \amp = 0 \tag{2.3.6}\\ 2a_1 - a_2 + 2a_3 + 3a_4 \amp = 0 \tag{2.3.7}\\ -3a_1 + 2a_2 - a_3 + 4a_4 \amp = 0 \label{R4-example-eqn4}\tag{2.3.8} \end{align}

We can compute the solutions to equations (2.3.5)–(2.3.8) by hand, or using SageMath.

SageMath outputs:

[[a1 == 0, a2 == 0, a3 == 0, a4 == 0]]

So indeed, equations (2.3.5)–(2.3.8) have only the trivial solution. Therefore the list of vectors is linearly independent.

Next, we need to check that the list of vectors spans \(\mathbb{R}^4\text{.}\) (There is a shorter way of doing this, using Corollary 2.3.19 below, but for now we prove it from first principles.) So, let \(\ve{w} = (w_1, w_2, w_3, w_4)\) be an arbitrary vector in \(\mathbb{R}^4\text{.}\) We need to show that there exists at least one way to express \(\ve{w}\) as a linear combination of the vectors \(\ve{v}_1, \ve{v}_2, \ve{v}_3, \ve{v}_4\text{.}\) In other words, we need to check if there exists at least one solution to the following equation:

\begin{align} a_1 \ve{v}_1 + a_2 \ve{v}_2 + a_3 \ve{v}_3 + a_4 \ve{v}_4 \amp = \ve{w} \label{fund_eqn_span_r4_ex}\tag{2.3.9}\\ \therefore a_1 (1, 0, 2,-3) + a_2 (1, 3, -1, 2) + a_3 (0, 1, 2, -1) + a_4 (1, 2, 3, 4) \amp = (w_1, w_2, w_3, w_4) \tag{2.3.10}\\ \therefore (a_1 - a_2 + a_4, 3a_2 + a_3 + 2a_4, 2a_1 - a_2 + 2a_3 + 3a_4, -3a_1 + 2a_2 -a_3 + 4a_4) \amp = (w_1, w_2, w_3, w_4) \tag{2.3.11} \end{align}

So the list of vectors spans \(\mathbb{R}^4\) if and only if the following equations for \(a_1, a_2, a_3, a_4\) always have a solution, no matter what the values of \(w_1, w_2, w_3, w_4\) are:

\begin{align} a_1 - a_2 + a_4 \amp = w_1 \label{R4-example-span-eqn1}\tag{2.3.12}\\ 3a_1 + a_3 + 2a_4 \amp = w_2 \tag{2.3.13}\\ 2a_1 - a_2 + 2a_3 + 3a_4 \amp = w_3 \tag{2.3.14}\\ -3a_1 + 2a_2 - a_3 + 4a_4 \amp = w_4 \label{R4-example-span-eqn4}\tag{2.3.15} \end{align}

We can compute the solutions to equations (2.3.12)–(2.3.15) by hand, or using SageMath:

Note that we ask SageMath to solve for a1, a2, a3, a4, since w1, w2, w3, w4 are regarded as constants in the equation... we are not trying to solve for them, they are fixed, but arbitrary! SageMath outputs:

[[a1 == 1/9*w1 + 7/18*w2 - 2/9*w3 - 1/18*w4, a2 == -2/3*w1 + 5/12*w2 - 1/6*w3 + 1/12*w4, a3 == -7/9*w1 - 2/9*w2 + 5/9*w3 - 1/9*w4, a4 == 2/9*w1 + 1/36*w2 + 1/18*w3 + 5/36*w4]]

In other words, there does indeed exist a solution, no matter what \((w_1, w_2, w_3, w_4)\) is. For instance, if \((w_1, w_2, w_3, w_4) = (3, 1, 2, 4) \text{,}\) then the solution is

\begin{equation*} a_1 = \frac{1}{18}, a_2 = - \frac{19}{12}, a_3 = -\frac{17}{9}, a_4 = \frac{49}{36}. \end{equation*}

In other words,

\begin{equation*} (3, 1, 2, 4) = \frac{1}{18} \ve{v}_1 - \frac{19}{12} \ve{v}_2 - \frac{17}{9} \ve{v}_3 + \frac{49}{36} \ve{v}_4. \end{equation*}

Since there exists a solution to equation (2.3.9) for each vector \(\ve{w} \in \mathbb{R}^4\text{,}\) we conclude that \(\bopen \ve{v}_1, \ve{v}_2, \ve{v}_3, \ve{v}_4 \bclose\) spans \(\mathbb{R}^4\text{.}\)

Hence \(\bopen \ve{v}_1, \ve{v}_2, \ve{v}_3, \ve{v}_4 \bclose\) is a basis for \(\mathbb{R}^4\text{,}\) since it is linearly independent and spans \(\mathbb{R}^4\text{.}\)

Example 2.3.6

The list of polynomials

\begin{equation*} \ve{p}_0 (x) := 1, \, \ve{p}_1 (x) := x, \, \ve{p}_2 (x) := x^2, \, \ldots, \, \ve{p}_n (x) := x^n \end{equation*}

is a basis for \(\Poly_n\text{,}\) so \(\Dim \Poly_n = n+1\text{.}\) Indeed, this list spans \(\Poly_n\) by definition, so we just need to check that it is linearly independent. Suppose that

\begin{equation*} a_0 \ve{p}_0 + a_1 \ve{p}_1 + a_2 \ve{p_2} + \cdots + a_n \ve{p}_n = \ve{0}\text{.} \end{equation*}

This is an equation between functions, so it holds for all \(x \in \mathbb{R}\text{!}\) In other words, for all \(x \in \mathbb{R}\text{,}\) the following equation holds:

\begin{equation} a_0 + a_1 x + a_2 x^2 + \cdots + a_n x^n = 0\label{root_of_poly}\tag{2.3.16} \end{equation}

Think about this carefully. Equation (2.3.16) represents an infinite family of equations for the unknowns \(a_0, a_1, \ldots, a_n\text{.}\) There is one equation for each value of \(x \in \mathbb{R}\text{.}\) For example:

\begin{align} (x=1) \amp \amp a_0 + a_1 + a_2 + \cdots + a_n \amp = 0 \label{poly-example-first-eqn}\tag{2.3.17}\\ (x= -1) \amp \amp a_0 - a_1 + a_2 + \cdots + (-1)^n a_n \amp = 0 \tag{2.3.18}\\ (x= 2) \amp \amp a_0 + 2a_1 + 4a_3 + \cdots + 2^n a_n = 0 \tag{2.3.19}\\ (x= 3) \amp \amp a_0 + 3a_1 + 9a_3 + \cdots + 3^n a_n = 0 \tag{2.3.20}\\ \amp \amp \vdots \label{poly-example-last-eqn}\tag{2.3.21} \end{align}

Suppose we find values for \(a_0, a_1, a_2, \ldots, a_n \) which solve all these infinitely many equations (2.3.17)–(2.3.21). We can now change our point of view. Namely, substitute these fixed values for \(a_0, a_1, \ldots, a_n\) into Equation (2.3.16) and regard Equation (2.3.16) as an equation for the unknown \(x\) (the coefficients \(a_0, a_1, \ldots, a_n\) are now fixed.) We conclude that every \(x \in \mathbb{R}\) is a root of this polynomial equation!

But, we know from algebra that a polynomial equation of the form (2.3.16) with nonzero coefficients has at most \(n\) roots \(x_1, x_2, \ldots, x_n\text{.}\) So, in order for (2.3.16) to hold for all real numbers \(x\text{,}\) the coefficients must be zero, i.e. \(a_0 = a_1 = a_2 = \cdots = a_n = 0\text{,}\) which is what we needed to show.

Example 2.3.7

Suppose \(X\) is a finite set. Then \(\Fun(X)\) is finite-dimensional, with dimension \(|X|\text{,}\) with basis given by the functions \(\ve{f}_a\text{,}\) \(a \in X\text{,}\) defined by:

\begin{equation} \ve{f}_a (x) := \begin{cases} 1 \amp \mbox{ if } x = a \\ 0 \amp \mbox{ otherwise} \end{cases}\label{defn_of_basis_of_functions}\tag{2.3.22} \end{equation}

We will prove this in a series of exercises.

The formula on the right hand side of (2.3.22) occurs so often in mathematics we give it a symbol of its own, \(\delta_{ab}\) (the ‘Kronecker delta’). This symbol stands for the formula: “If \(a=b\text{,}\) return a 1. If \(a \neq b\text{,}\) return a \(0\)”. In this language, we can rewrite the definition of the functions \(\ve{f}_a\) as

\begin{equation} \ve{f}_a (x) := \delta_{ax}\text{.}\tag{2.3.23} \end{equation}
Example 2.3.8

\(\Trig_n\) is \((2n+1)\)-dimensional, with basis

\begin{gather*} \ve{T}_0 (x) := 1, \,\,\, \ve{T}_1 (x) := \cos x, \, \ve{T}_2 (x) := \sin x, \, \ve{T}_3 (x) := \cos 2x, \\ \ve{T}_4 (x) := \sin 2x, \, \ldots, \, \ve{T}_{2n-1} (x) := \cos nx , \, \ve{T}_{2n} (x) := \sin nx. \end{gather*}

You know that these functions span \(\Trig_n\text{,}\) by definition. They are also linearly independent, though we will not prove this.

Example 2.3.9

The dimension of \(\Mat_{n,m}\) is \(nm\text{,}\) with basis given by the matrices

\begin{equation*} \mat{E}_{ij} , i=1 \ldots n, \, j = 1 \ldots m \end{equation*}

which have a \(1\) in the \(i\)th row and \(j\)th column and zeroes everywhere else.

Usually \(\mat{A}\) is a matrix, and \(\mat{A}_{ij}\) is the element of the matrix at position \((i,j)\text{.}\) But now \(\mat{E}_{ij}\) is a matrix in its own right! Its element at position \((k,l)\) will be written as \((\mat{E}_{ij})_{kl}\text{.}\) I hope you don't find this too confusing. In fact, we can write down an elegant formula for the elements of \(\mat{E}_{ij}\) using the Kronecker delta symbol:

\begin{equation} (\mat{E}_{ij})_{kl} = \delta_{ik} \delta_{jl}\label{kronecker_for_matrix_basis}\tag{2.3.24} \end{equation}
Example 2.3.10

The standard basis of \(\Mat_{2,2}\) is

\begin{equation*} \mat{E}_{11} = \left[ \begin{array}{cc} 1 \amp 0 \\ 0 \amp 0 \end{array} \right], \, \mat{E}_{12} = \left[ \begin{array}{cc} 0 \amp 1 \\ 0 \amp 0 \end{array} \right], \mat{E}_{21} = \left[ \begin{array}{cc} 0 \amp 0 \\ 1 \amp 0 \end{array} \right], \mat{E}_{22} = \left[ \begin{array}{cc} 0 \amp 0 \\ 0 \amp 1 \end{array} \right]\text{.} \end{equation*}
Example 2.3.11

The standard basis of \(\Col_n\) is

\begin{equation*} \col{e}_1 := \left[ \begin{array}{c} 1 \\ 0 \\ \vdots \\ 0 \end{array} \right], \, \col{e}_2 := \left[ \begin{array}{c} 0 \\ 1 \\ \vdots \\ 0 \end{array} \right], \, \ldots, \, \col{e}_n := \left[ \begin{array}{c} 0 \\ 0 \\ \vdots \\ 1 \end{array} \right] \text{.} \end{equation*}

We now consider dimensions of subspaces of vector spaces.

Proposition 2.3.12

Let \(W\) be a subspace of a finite-dimensional vector space \(V\text{.}\) Then \(W\) is finite-dimensional, and \(\Dim(W) \leq \Dim(V)\text{.}\)

Proof

Let

\begin{equation*} \basis{C} = \bopen \ve{v}_1, \ldots, \ve{v}_n \bclose \end{equation*}

be a basis for \(V\text{,}\) so that \(\Dim(V) = n\text{.}\) We just need to show that \(W\) is finite-dimensional, i.e. that there exists a basis

\begin{equation*} \basis{B} = \bopen \ve{w}_1, \ldots, \ve{w}_k \bclose \end{equation*}

for \(W\text{.}\) For then \(\basis{B}\) will be a list of \(k\) linearly independent vectors which live in \(W\) (and hence also in \(V\)) and hence we must have \(k \leq n\) by Proposition 2.2.10, as \(\basis{C}\) spans \(V\text{.}\)

We show that \(W\) is finite-dimensional as follows.

If \(W\) is the zero vector space \(\{ \ve{0} \} \text{,}\) then \(W\) is finite-dimensional by definition.

If \(W\) is not the zero vector space, then there exists a nonzero vector \(\ve{w}_1 \in W\text{.}\) Consider the list \(\basis{B}_1 = \bopen \ve{w}_1 \bclose\text{.}\) Note that \(\basis{B}_1\) is linearly independent, by Item 3 of Proposition 2.2.8. So, if \(\basis{B}_1\) spans \(W\text{,}\) then it is a basis for \(W\text{,}\) and so \(W\) is finite-dimensional and we are done.

If \(\basis{B}_1\) does not span \(W\text{,}\) then there exists a vector \(\ve{w}_2 \in W\) which is not a scalar multiple of \(\ve{e}_1\text{.}\) Now consider the list \(\basis{B}_2 = \left\{ \ve{e}_1, \ve{e}_2 \right\}\text{.}\) Once again, \(\basis{B}_2\) is linearly independent, by Item 3 of Proposition 2.2.8. So, if \(\basis{B}_2\) spans \(W\text{,}\) then it is a basis for \(W\text{,}\) and we are done.

If \(\basis{B}_2\) does not span \(W\text{,}\) then there exists a vector \(\ve{w}_3 \in W\) which is not a linear combination of \(\ve{w}_1\) and \(\ve{w}_2\text{.}\) Now consider the list \(\basis{B}_3 = \left\{ \ve{w}_1, \ve{w}_2, \ve{w}_3 \right\}\text{.}\) Again, \(\basis{B}_3\) is linearly independent, by Item 3 of Proposition 2.2.8. If it does not span \(W\text{,}\) then there exists a vector \(\ve{w}_4 \in W\) which is not a linear combination of \(\ve{w}_1, \ve{w}_2, \ve{w}_3\text{.}\) So consider the list \(\basis{B}_4 = \bopen \ve{w}_1, \ve{w}_2, \ve{w}_3, \ve{w}_4 \bclose \text{.}\)

This process must terminate for some \(k \leq n\text{.}\) If not, then it will produce a list \(\basis{B}_{n+1} = \bopen \ve{w}_1, \ldots, \ve{w}_{n+1} \bclose\text{.}\) This would be a linearly independent list of \(n+1\) vectors from \(V\text{.}\) But \(\Dim V = n\text{,}\) so this is impossible, by Proposition 2.2.10. Hence for some \(k \leq n\) we must have that \(\basis{B}_k\) is a basis for \(W\text{,}\) and we are done.

It is good to have an example of an infinite-dimensional vector space.

Proposition 2.3.13

\(\Poly\) is infinite-dimensional.

Proof

Suppose \(\Poly\) is finite-dimensional. This means there exists a finite collection of polynomials \(\ve{p}_1, \ve{p}_2, \ldots, \ve{p}_n\) which spans \(\Poly\text{.}\) But, let \(d\) be the highest degree of all the polynomials in the list \(\ve{p}_1, \ve{p}_2, \ldots, \ve{p}_n\text{.}\) Then \(\ve{p} := x^{d+1}\) is a polynomial which is not in the span of \(\ve{p}_1, \ve{p}_2, \ldots, \ve{p}_n\text{,}\) since adding polynomials together and multiplying them by scalars can never increase the degree. We have arrived at a contradiction. So our initial assumption cannot be correct, i.e. \(\Poly\) cannot be finite-dimensional.

Example 2.3.14

We will not prove this here, but the following vector spaces are also infinite-dimensional:

  • \(\mathbb{R}^\infty\text{,}\)

  • \(\Fun(X)\) where \(X\) is an infinite set,

  • \(\Cont(I)\) for any nonempty interval \(I\text{,}\) and

  • \(\Diff(I)\) for any open interval \(I\text{.}\)

Subsection 2.3.1 Sifting

¶

If we consider the proof of Proposition 2.2.10 (the ‘Bumping off’ Proposition) carefully, we find that it makes use of a sifting algorithm. This algorithm can actually be applied to any list of vectors \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_n\) in a vector space. Consider each vector \(\ve{v}_i\) in the list consecutively. If \(\ve{v}_i\) is the zero vector, or if it is a linear combination of the preceding vectors \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_{n-1}\text{,}\) remove it from the list.

Example 2.3.15

Sift the following list of vectors in \(\mathbb{R}^3\text{:}\)

\begin{align*} \ve{v}_1 \amp = (1,2,-1), \amp \ve{v}_2 \amp = (0, 0, 0), \amp \ve{v}_3 \amp = (3, 6, -3)\\ \ve{v}_4 \amp = (1, 0, 5), \amp \ve{v}_5 \amp = (5, 4, 13), \amp \ve{v}_6 \amp = (1, 1, 0)\text{.} \end{align*}

We start with \(\ve{v}_1\text{.}\) Since it is not the zero vector, and is not a linear combination of any preceding vectors, it remains. We move on to \(\ve{v}_2\text{,}\) which is zero, so we remove it. We move on to \(\ve{v}_3\text{,}\) which by inspection is equal to \(3 \ve{v}_1\text{,}\) so we remove it. We move on to \(\ve{v}_4\text{.}\) It is not zero, and cannot be expressed as a multiple of \(\ve{v}_1\) (check this), so it remains. We move on to \(\ve{v}_5\text{.}\) We check if it can be written as a linear combination

\begin{equation*} \ve{v}_5 = a\ve{v}_1 + b \ve{v}_4 \end{equation*}

and find the solution \(a=2, b=3\) (check this), so we remove it. Finally we move on to \(\ve{v}_6\text{.}\) We check if it can be written as a linear combination

\begin{equation*} \ve{v}_6 = a\ve{v}_1 + b \ve{v}_4 \end{equation*}

and find no solutions (check this), so it remains. Our final sifted list is

\begin{equation*} \ve{v}_1, \ve{v}_4, \ve{v}_6\text{.} \end{equation*}
Checkpoint 2.3.16

Do the three ‘check this’ calculations above.

Sifting is a very useful way to construct a basis of a vector space!

Lemma 2.3.17

If a list of vectors \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_n\) spans a vector space \(V\text{,}\) then sifting the list will result in a basis for \(V\text{.}\)

Proof

At each step, the vector that is removed from the list is either the zero vector, or a linear combination of the vectors before it. So if we remove this vector, the resulting list will still span \(V\text{.}\) Thus by the end of the process, the final sifted list of vectors still spans \(V\text{.}\)

To see that the final sifted list is linearly independent, we can apply Proposition 2.2.8. By construction, no vector in the final sifted list is a linear combination of the preceding vectors (if it was, it would have been removed!). Hence the final sifted list is not linearly dependent, so it must be linearly independent!

Corollary 2.3.18

Any linearly independent list of vectors \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_k\) in a finite-dimensional vector space \(V\) can be extended to a basis of \(V\text{.}\)

Proof

Since \(V\) is finite-dimensional, it has a basis \(\ve{e}_1, \ldots, \ve{e}_n\text{.}\) Now consider the list

\begin{equation*} L : \ve{v}_1, \ve{v}_2, \ldots, \ve{v}_k, \ve{e}_1, \ve{e}_2, \ldots, \ve{e}_n \end{equation*}

which clearly spans \(V\text{.}\) By sifting this list, we will arrive at a basis for \(V\text{,}\) by Lemma 2.3.17. Some of the \(\ve{e}\)-vectors may have been removed. But none of the \(\ve{v}\)-vectors will have been removed, since that would mean some \(\ve{v}_i\) is a linear combination of the preceding vectors \(\ve{v}_1, \ldots, \ve{v}_{i-1}\text{,}\) which is impossible, as \(\ve{v}_1, \ldots, \ve{v}_k\) is linearly independent list. Hence after sifting the list \(L\) we indeed extend our original list \(\ve{v}_1, \ldots, \ve{v}_k\) to a basis of \(V\text{.}\)

Corollary 2.3.19

If \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_n\) is a linearly independent list of \(n\) vectors in an \(n\)-dimensional vector space \(V\text{,}\) then it is a basis.

Proof

By Corollary 2.3.18, we can extend \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_n\) to a basis for \(V\text{.}\) But \(V\) has dimension \(n\text{,}\) so the basis must contain only \(n\) vectors by Theorem 2.3.2 (Invariance of Dimension). So we have not added any vectors at all! Hence our original list was already a basis.

Example 2.3.20

In Example 2.2.7 we showed that the list of polynomials

\begin{equation*} \ve{q}_0 (x) := 1, \,\, \ve{q}_1 (x):= x, \,\, \ve{q}_2(x) := 2x^2 - 1, \,\, \ve{q}_3(x) := 4x^3 - 3x \end{equation*}

is linearly independent in \(\Poly_3\text{.}\) Since \(\Dim \Poly_3 = 4\text{,}\) we see that it is a basis of \(\Poly_3\text{.}\)

In Exercise 2.1.5, you showed that \(\ve{q}_0, \ldots, \ve{q}_3\) is a basis for \(\Poly_3\) by ‘brute force’. This new method is different!

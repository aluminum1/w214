Subsection Exercises

1.6 Subspaces
1

Read through the webpage version of SubsectionÂ 1.6.3 (Solutions to homogenous linear differential equations), which is new and contains a lot of SageMath examples.

2

Show that the set

\begin{equation*} V := \{ (a, -a, b, -b) : a, b \in \mathbb{R} \} \end{equation*}

is a subspace of \(\mathbb{R}^4\text{.}\)

3

Consider the set

\begin{equation*} V := \{ \mathbf{f} \in \Diff((-1, 1)) : f'(0) = 2 \} \end{equation*}

Is \(V\) a subspace of \(\Diff((-1,1))\text{?}\) If you think it is, prove that it is. If you think it is not, prove that it is not!

4

Is \(\mathbb{R}^+ := \{ \ve{x} \in \mathbb{R} : \ve{x} \geq 0 \}\) a subspace of \(\mathbb{R}\text{?}\) If you think it is, prove that it is. If you think it is not, prove that it is not!

5

Give an example of a nonempty subset \(V\) of \(\mathbb{R}^2\) which is closed under scalar multiplication, but \(V\) is not a subspace of \(\mathbb{R}^2\text{.}\)

\begin{equation*} p_1 = 1 + x, p_2 = x^3 + x^2 + x - 1, p_3 = x^3 -x^2 + 1, p_4 = 1-x \text{ ?} \end{equation*}

Setup the appropriate system of simultaneous linear equations. Then solve these by hand, or using SageMath, as in Example ExampleÂ 2.1.4.

Show that the polynomials

\begin{equation*} p_1 = 1 + x, p_2 = x^3 + x^2 + x - 1, p_3 = x^3 -x^2 + 1, p_4 = 1-x \end{equation*}

from the previous question span \(\Poly_3\text{.}\) Setup the appropriate system of simultaneous linear equations. Then solve these by hand, or using SageMath, as in Example ExampleÂ 2.1.8.

2.2 (Linear Independence)
9

Read through the webpage version of SectionÂ 2.1. I have added some new material, and gave examples of how to use SageMath to solve systems of linear equations.

10
Let \(\basis{S} = \bopen \ve{v}_1, \ldots, \ve{v}_n \bclose \) be a list of vectors in a vector space \(V\text{.}\) Suppose that \(\basis{S}\) spans \(V\text{.}\) Suppose that \(w\) is another vector in \(V\text{.}\) Prove that the list of vectors \(\basis{S}' = \bopen \ve{w}, \ve{v}_1, \ldots, \ve{v}_n \bclose\) also spans \(V\text{.}\)

Consider the following list of matrices, thought of as vectors in \(\Mat_{2,2}\text{:}\)

\begin{equation*} \ve{v}_1 = \begin{bmatrix} 1 & 2 \\ 1 & 1 \end{bmatrix},\, \ve{v}_2 = \begin{bmatrix} 1 & 0 \\ -2 & 1 \end{bmatrix},\, \ve{v}_3 = \begin{bmatrix} 1 & 0 \\ 2 & 3 \end{bmatrix},\, \ve{v}_4 = \begin{bmatrix} 0 & 3 \\ 1 & -1 \end{bmatrix},\, \ve{v}_5 = \begin{bmatrix} 1 & 0 \\ 0 & 1 \end{bmatrix} \end{equation*}
11

Show that the list is linearly dependent. You are welcome to use SageMath (you will first need to setup the appropriate system of linear equations).

12

Go through the same steps as in ExampleÂ 2.2.9 to find the first vector in the list which is either the zero vector or a linear combination of the preceding vectors. You are welcome to use SageMath at the points in your calculation when you need to solve a system of simultaneous linear equations.

Indeed, recall that \(L\) can be specified by a homogenous linear equation of the form:

\begin{equation} L = \{ (x,y) \in \mathbb{R}^2 : ax + by = 0 \}\tag{1.6.1} \end{equation}

for some constants \(a\) and \(b\text{.}\) So, if \(\ve{v} = (x, \, y)\) and \(\ve{v}' = (x', \, y')\) lie on \(L\text{,}\) then their sum \(\ve{v} + \ve{v}' = (x + x', \, y + y')\) also lies on \(L\text{,}\) because its components satisfy the defining equation (1.6.1):

\begin{align*} \amp a(x + x') + b (y + y')\\ \amp = (ax + by) + (ax' + by')\\ \amp = 0 + 0 \amp \amp \mbox{(since \(ax + by = 0\) and \(ax' + by' = 0\))}\\ \amp = 0\text{.} \end{align*}

Example 1.6.13 Continuous functions as a subspace

The set

\begin{equation*} \Cont(I) := \{ \ve{f}: I \rightarrow \mathbb{R}, \, \ve{f} \mbox{ continuous} \} \end{equation*}

of all continuous functions on an interval \(I\) is a subspace of the set \(\Fun(I)\) of all functions on \(I\text{.}\) Let us check that it satisfies the definition. You know from earlier courses that:

  • If \(\ve{f}\) and \(\ve{g}\) are continuous functions on \(I\text{,}\) then \(\ve{f} + \ve{g}\) is also a continuous function.

  • The zero function \({0}\) defined by \(\ve{0}(x) = 0\) for all \(x \in I\) is a continuous function.

  • If \(\ve{f}\) is a continuous function, and \(k\) is a scalar, then \(k \smul\ve{f}\) is also continuous.

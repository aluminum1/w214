Example 1.6.13 Hyperplanes orthogonal to a fixed vector

This example generalizes Example 1.6.6 to higher dimensions. Let \(\ve{v} \in \mathbb{R}^n \) be a fixed nonzero vector. The hyperplane orthogonal to \(\ve{v}\) is the set \(W\) of all vectors orthogonal to \(\ve{v}\text{,}\) that is,

\begin{equation} W := \{ \ve{w} \in \mathbb{R}^n : \ve{v} \dotp \ve{w} = 0\}.\tag{1.6.2} \end{equation}

You will prove in Exercise Checkpoint 1.6.14 that \(W\) is a subspace of \(\mathbb{R}^n\text{.}\)

For example, consider the vector \(\ve{v} = (1,2,3) \in \mathbb{R}^3\text{.}\) Then the hyperplane orthogonal to \(\ve{v}\) is

\begin{equation} W = \{ \ve{w} \in \mathbb{R}^3 : \ve{v} \dotp \ve{w} = 0 \}.\tag{1.6.3} \end{equation}

If we write \(\ve{w} = (w_1, w_2, w_3)\) then \(\ve{v} \dotp \ve{w} = 0\) translates into the equation

\begin{equation} w_1 + 2 w_2 + 3w_3 = 0.\tag{1.6.4} \end{equation}

So, \(W\) can be regarded as the set of vectors in \(\mathbb{R}^3\) whose components satisfy (1.6.4).

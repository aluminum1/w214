Solution 2.2.9.1

We check Item 2 of Proposition 2.2.8. That is, we check if one of the vectors in the list is a linear combination of the other vectors. Indeed, we observe by inspection that

\begin{equation} \ve{f}_2 = \ve{f}_1 + \ve{f}_3.\tag{2.2.9} \end{equation}

Hence, \(\basis{B}\) is linearly dependent.

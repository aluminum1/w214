Example 1.5.7

Let us practice using the rules of a vector space to perform everyday calculations. For instance, suppose that we are trying to solve for the vector \(\ve{x}\) appearing in the following equation:

\begin{equation} \ve{v} + 7 \smul\ve{x} = \ve{w}\tag{1.5.3} \end{equation}

We do this using the rules as follows:

\begin{align*} \ve{v} + 7 \smul\ve{x} \amp = \ve{w}\\ \therefore -\ve{v} + (\ve{v} + 7 \smul\ve{x}) \amp = -\ve{v} + \ve{w} \amp \amp \mbox{(Added \(-\ve{v}\) on left to both sides)}\\ \therefore (-\ve{v} + \ve{v}) + 7 \smul\ve{x} \amp = -\ve{v} + \ve{w} \amp \amp \mbox{(used R2 on LHS)}\\ \therefore \ve{0} + 7 \smul\ve{x} \amp = -\ve{v} + \ve{w} \amp \amp \mbox{(used } \knowl{./knowl/add_inv.html}{\text{LemmaÂ 1.5.3}} \mbox{ on LHS)}\\ \therefore 7 \smul\ve{x} \amp = -\ve{v} + \ve{w} \amp \amp \mbox{(used R3a on LHS)}\\ \therefore \frac{1}{7} \smul(7 \smul\ve{x}) \amp = \frac{1}{7} \smul(-\ve{v} + \ve{w}) \amp \amp \mbox{(scalar multiplied both sides by \(\frac{1}{7}\))}\\ \therefore (\frac{1}{7} 7) \smul \ve{x} \amp = \frac{1}{7} \smul(-\ve{v} + \ve{w}) \amp \amp \mbox{(used R6 on LHS)}\\ \therefore 1 \smul\ve{x} \amp = \frac{1}{7} \smul(-\ve{v} + \ve{w}) \amp \amp \mbox{(multiplied \(\frac{1}{7}\) with \(7\))}\\ \therefore \ve{x} \amp = \frac{1}{7} \smul(-\ve{v} + \ve{w}) \amp \amp \mbox{(R7)} \end{align*}

As the course goes on we will leave out all these steps. But it is important for you to be able to reproduce them all, if asked to do so!

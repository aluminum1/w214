Example 1.4.1 A non-example

Define the set \(V\) by

\begin{equation} V := \{ \mathbf{a}, \mathbf{b} \} \text{.}\tag{1.4.1} \end{equation}

Define the addition operation by

\begin{align*} \ve{a} + \ve{a} := \ve{a} \amp \amp \ve{a} + \ve{b} := \ve{a}\\ \ve{b} + \ve{a} := \ve{b} \amp \amp \ve{b} + \ve{b} := \ve{c} \end{align*}

For this to be a well-defined addition operation, we need to check that whenever you add two elements of \(V\) together, you get out a well-defined element of \(V\text{.}\) But \(\ve{b} + \ve{b} = \ve{c}\text{,}\) so adding \(\ve{b} \in V\) to itself outputs something, namely \(\ve{c}\text{,}\) which is not an element of \(V\text{.}\) So \(V\) does not form a vector space since it does not even have a well-defined addition operation.

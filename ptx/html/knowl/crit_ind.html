Remark 2.2.3

The criterion for a list of vectors \(\ve{v}_1, \ve{v}_2, \ldots, \ve{v}_n\) to be linearly independent,

\begin{equation*} k_1 \ve{v}_1 + k_2 \ve{v}_2 + \cdots k_n \ve{v}_n = \ve{0} \Rightarrow k_1 = k_2 = \cdots = k_n = 0 \end{equation*}

is clearly equivalent to the criterion:

All of the \((\ve{v}_i)\) are nonzero and none of them can be expressed as a nontrivial linear combination of the others. In fact, this seemingly obvious remark will be the starting point for many future proofs concerning linear independence!

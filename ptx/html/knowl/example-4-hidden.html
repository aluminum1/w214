Define the set \(Z\) by

\begin{equation} Z := \{ \ve{z} \}\text{.}\tag{1.4.5} \end{equation}

Note that it contains just a single element, \(\ve{z}\text{.}\) Define the addition operation as

\begin{equation} \ve{z} + \ve{z} := \ve{z}\tag{1.4.6} \end{equation}

Define the zero element as

\begin{equation} \ve{0} := \ve{z}\text{.}\tag{1.4.7} \end{equation}

Finally define scalar multiplication by a scalar \(k \in \mathbb{R}\) as:

\begin{equation} k \smul\ve{z} := \ve{z}\text{.}\tag{1.4.8} \end{equation}

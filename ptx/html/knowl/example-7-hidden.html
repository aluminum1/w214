Let \(X\) be any set. Define the set \(\Fun(X)\) of real-valued functions on \(X\) by

\begin{equation} \Fun(X) := \{ f : X \rightarrow \mathbb{R} \}\text{.}\tag{1.4.17} \end{equation}

Note that the functions can be arbitrary; there is no requirement for them to be continuous, or differentiable. Such a requirement would not make sense, since \(X\) could be an arbitrary set. For instance \(X\) could be the set \(\{a, b, c\}\) â€”without any further informaiton, it does not make sense to say that a function \(f : X \rightarrow \mathbb{R}\) is continuous.

Define the addition operation by

\begin{equation} (f + g) (x) := f(x) + g(x), x \in X\tag{1.4.18} \end{equation}

Make sure you understand what this formula is saying! We start with two functions \(f\) and \(g\text{,}\) and we are defining their sum \(f + g\text{.}\) This is supposed to be another function on \(X\text{.}\) To define a function on \(X\text{,}\) I am supposed to write down what value it assigns to each \(x \in X\text{.}\) And that is what the formula says: the value that the function \(\mathbf{f} + \mathbf{g}\) assigns to an element \(x \in X\) is defined to be the number \(f(x)\) plus the number \(g(x)\text{.}\) Remember: \(f\) is a function, while \(f(x)\) is a number!

Define the zero vector, which we will call \(z\) in this example, to be the function which outputs the number \(0\) for every input value of \(x \in X\text{:}\)

\begin{equation} z (x) := 0 \text{ for all \(x \in X\).}\tag{1.4.19} \end{equation}

Define scalar multiplication by

\begin{equation} (k \smul f)(x) := k f(x)\text{.}\tag{1.4.20} \end{equation}

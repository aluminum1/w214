(a) Suppose that

\begin{align*} \ve{v} \amp = a_1 \ve{e}_1 + a_2 \ve{e}_2 + \cdots + a_n \ve{e}_n\\ \end{align*}

and

\begin{align*} \ve{w} \amp = b_1 \ve{e}_1 + b_2 \ve{e}_2 + \cdots + b_n \ve{e}_n \, .\\ \end{align*}

Then, using the rules of a vector space, we compute

\begin{align*} \ve{v} + \ve{w} \amp = (a_1 + b_1) \ve{e}_1 + (a_2 + b_2) \ve{e}_2 + \cdots + (a_n + b_n) \ve{e}_n \, \text{.} \end{align*}

From this we read off that

\begin{equation*} [\ve{v} + \ve{w}]_\basis{B} = \left[ \begin{array}{c} a_1 + b_1 \\ a_2 + b_2 \\ \vdots \\ a_n + b_n \end{array} \right]\text{.} \end{equation*}

The proof of (b) is similar.

Exercise 3

Let \(\basis{B} = \left\{ \ve{v}_1, \ldots, \ve{v}_n \right\}\) be a linearly independent list of vectors in a vector space \(V\text{.}\) Suppose that \(\ve{v}\) is a vector in \(V\) which cannot be written as a linear combination of the vectors from \(\basis{B}\text{.}\) Show that the list \(\basis{B}' = \left\{ \ve{v}_1, \ldots, \ve{v}_n, \ve{v} \right\}\) is still linearly independent. (Hint: Use the Linear Combination of Preceding Vectors Proposition.)

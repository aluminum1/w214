Definition 1.2.1

A vector space is a set \(V\) equipped with the following data:

D1.

An addition operation. (That is, for every pair of elements \(\ve{u}, \ve{v} \in V\text{,}\) a new element \(\ve{u} + \ve{v} \in V\) is defined.)

D2.

A zero vector. (That is, a special vector \(\ve{0} \in V\) is marked out.)

D3.

A scalar multiplication operation. (That is, for each real number \(k\) and each element \(\ve{v} \in V\text{,}\) a new element \(k \smul\ve{v} \in V\) is defined).

This data should satisfy the following rules for all \(\ve{u}, \ve{v}, \ve{w}\) in \(V\) and for all real numbers \(k\) and \(l\text{:}\)

R1.

\(\ve{v} + \ve{w} = \ve{w} + \ve{v}\)

R2.

\((\ve{u} + \ve{v}) + \ve{w} = \ve{u} + (\ve{v} + \ve{w})\)

R3a.

\(\ve{0} + \ve{v} = \ve{v}\)

R3b.

\(\ve{v} + \ve{0} = \ve{v}\)

R4.

\(k \smul(\ve{v} + \ve{w}) = k \smul\ve{v} + k \smul\ve{w}\)

R5.

\((k + l) \smul\ve{v} = k \smul\ve{v} + l \smul{\ve{v}}\)

R6.

\(k \smul(l \smul\ve{v}) = (kl) \smul\ve{v}\)

R7.

\(1 \smul\ve{v} = \ve{v}\)

R8.

\(0 \smul\ve{v} = \ve{0}\)

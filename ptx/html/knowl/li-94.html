Item 2.3.2.8.b
Once again, we omit the check that \(V \) is a subspace of \(\Poly_2 \text{.}\) Pick any non-zero vector \(\ve v_1 \) in \(V \text{.}\) Let's choose \(\ve v_1 = x \text{.}\) It is not as obvious as before whether there are indeed any vectors in \(V \) not in the span of \(\left\{ \ve v_1 \right\} \text{,}\) and so we must do some computations. If \(p(x) = ax^2 + bx + c\) is in \(V \text{,}\) \(p(x)\) must satisfy
\begin{equation*} x(2ax + b) = ax^2 + bx + c \\ \implies 2ax^2 + bx = ax^2 + bx + c. \end{equation*}
Hence \(a = c = 0 \text{.}\) Thus all vectors in \(V \) are scalar multiples of \(\ve v_1 = x \text{.}\) Hence \(\bopen \ve v_1 \bclose \) is a basis for \(V \) and so \(\dim V = 1\text{.}\)

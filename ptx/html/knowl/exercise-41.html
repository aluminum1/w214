Exercise 5

Consider the set

\begin{equation*} V := \{ (x_1, \, x_2, \, x_3, \, \ldots) \in \mathbb{R}^\infty : \lim_{n \rightarrow \infty} x_n = 0 \} \end{equation*}

Is \(V\) a subspace of \(\mathbb{R}^\infty\text{?}\) If you think it is, prove that it is. If you think it is not, prove that it is not!

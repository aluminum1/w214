At each step, the vector that is removed from the list is either the zero vector, or a linear combination of the vectors before it. So if we remove this vector, the resulting list will still span \(V\text{.}\) Thus by the end of the process, the final sifted list of vectors still spans \(V\text{.}\)

To see that the final sifted list is linearly independent, we can apply PropositionÂ 2.2.8. By construction, no vector in the final sifted list is a linear combination of the preceding vectors (if it was, it would have been removed!). Hence the final sifted list is not linearly dependent, so it must be linearly independent!

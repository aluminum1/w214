\(\mathbb{R}^n\) is spanned by

\begin{equation} \ve{e}_1 := (1, 0, \ldots, 0), \,\,\, \ve{e}_2 := (0, 1, \ldots, 0), \,\,\, \ldots, \,\,\, \ve{e}_n := (0, 0, \ldots, 0, 1)\tag{2.1.30} \end{equation}

because every vector \(\ve{v} = (a_1, \,a_2, \, \ldots, \, a_n)\) can be written as the linear combination

\begin{equation} \ve{v} = a_1 \ve{e}_1 + a_2 \ve{e}_2 + \cdots + a_n \ve{e}_n\text{.}\tag{2.1.31} \end{equation}

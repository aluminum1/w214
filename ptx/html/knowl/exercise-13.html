Exercise 5

Let \(\mathbb{R}^+\) be the set of positive real numbers. Define an addition operation \(\oplus\text{,}\) a zero vector \(z\) and a scalar multiplication \(.\)on \(\mathbb{R}^+\) by

\begin{align*} x \oplus y \amp := xy\\ z \amp := 1\\ k \smul x \amp := x^k \end{align*}

where \(x, y \in \mathbb{R}^+\text{,}\) and \(k\) is a scalar (i.e. an arbitrary real number).

  1. Check that these operations are well-defined. For instance, is \(x+ y \in \mathbb{R}^+\text{,}\) as it should be?

  2. Check that these operations satisfy R1 to R8.

We conclude that \(\mathbb{R}^+\text{,}\) equipped with these operations, forms a vector space.

Mathematics is about identifying patterns. We have found three different sets, \(A\text{,}\) \(B\) and \(C\text{,}\) which look very different on the surface but have much in common. In each set, there is an addition operation, a zero vector, and a scalar multiplication operation. Moreover, in each set, these operations satisfy the same rules. Let us now record this pattern by giving it a name and writing down the rules explicitly.

Example 1.4.6 \(\mathbb{R}^n\)

Define the set \(\mathbb{R}^n\) by

\begin{equation} \mathbb{R}^n := \{ (x_1, \, x_2, \, \ldots, \, x_n) : x_i \in \mathbb{R} \mbox{ for all } i = 1 \ldots n\}\text{.}\tag{1.4.9} \end{equation}

Define the addition operation as

\begin{equation} (x_1, \, x_2, \, \ldots, \, x_n) + (y_1, \, y_2, \, \ldots, \, y_n) := (x_1 + y_1, \, x_2 + y_2, \, \ldots, \, x_n + y_n)\text{.}\tag{1.4.10} \end{equation}

Define the zero element as

\begin{equation} \ve{0} := (0, \, 0, \, \ldots, \, 0)\text{.}\tag{1.4.11} \end{equation}

Define scalar multiplication by

\begin{equation} k \smul(x_1, \, x_2, \, \ldots, \, x_n) := (k x_1, \, k x_2, \, \ldots, \, k x_n)\text{.}\tag{1.4.12} \end{equation}

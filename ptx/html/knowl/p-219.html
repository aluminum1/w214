Since \(U\) is a subspace, we know that it actually makes sense to “equip it with the same addition operation, zero vector and scalar multiplication as in \(V\)”. (If \(U\) was not a subspace, then we might have for instance \(\ve{u}, \ve{u}' \in U\) but \(\ve{u}+\ve{u}' \notin U\text{,}\) so the addition operation would not make sense.)

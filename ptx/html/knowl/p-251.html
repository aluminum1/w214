Suppose we are given an \(n\)th order homogenous linear differential equation of the form (1.6.4) on some interval \(I \subseteq \mathbb{R}\text{.}\) Write \(V\) for the set of all solutions to the differential equation. That is,

\begin{equation} V := \{ y : a_n(x) y^{(n)}(x) + \cdots + a_1(x) y'(x) + a_0(x) y(x) = 0 \}\tag{1.6.8} \end{equation}

A list of vectors\(\basis{B} = \left\{ \ve{e}_1, \ve{e}_2,\ldots, \ve{e}_n\right\}\) in a vector space \(V\) is a basis for \(V\) if and only if every vector \(\ve{v} \in V\) can be written as a linear combination

\begin{equation} \ve{v} = a_1 \ve{e}_1 + a_2 \ve{e}_2 + \cdots + a_n \ve{e}_n\tag{2.4.1} \end{equation}

in precisely one way. (That is, for each \(\ve{v} \in V\) there exist scalars \(a_1\text{,}\) \(a_2\text{,}\) \(\ldots\text{,}\) \(a_n\) satisfying (2.4.1), and that moreover these scalars are unique).

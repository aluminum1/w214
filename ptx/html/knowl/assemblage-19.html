I indicate that a collection of things is a list (where the order matters) and not merely a set (where the order does not matter) using my own home-made symbols \(\{ \}\text{.}\) A basis \(\basis{B} = \left\{ \ve{b}_1, \ve{b}_2, \ldots, \ve{b}_n \right\}\) is a list of vectors. The order of the vectors matters because it affects the coordinate vector \([\ve{v}]_\basis{B}\text{.}\)

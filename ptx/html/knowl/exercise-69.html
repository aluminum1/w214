If it is true, prove it. If it is false, give a counterexample. (Hint: Use the definition of linear independence.)

  2. The list \(\left\{ \ve{u} - \ve{v}, \, \ve{v} - \ve{w}, \, \ve{u} - \ve{w} \right\}\) is linearly independent.

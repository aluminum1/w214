Solution 2.4.3.1

We write \(\ve{p}\) as a linear combination of polynomials from the basis \(\basis{B}\text{:}\)

\begin{equation*} 2x^2 - 2x + 3 = -4(1+x) - \frac{5}{2} (x^2 + x -1) + \frac{9}{2}(x^2 + x +1) \end{equation*}
\begin{equation*} [\ve{p}]_\basis{B} := \left[ \begin{array}{c} -4 \\ -\frac{5}{2}\\ \frac{9}{2} \end{array} \right] \end{equation*}

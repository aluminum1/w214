We will call the elements of a vector space vectors, and we will write them in boldface eg. \(\ve{v} \in V\text{.}\) We do this to distinguish vectors from real numbers, which we will call scalars, and which don't have a boldface. It is difficult to use boldface in handwriting, so you should write them with an arrow on top, like so: \(\vec{v}\text{.}\)

Item 2.3.2.7.a

True. Suppose there is a linear relation on \(\left\{ \ve{u} + \ve{v}, \, \ve{v} + \ve{w}, \, \ve{u} + \ve{w} \right\}\text{:}\)

\begin{equation*} a( \ve{u} + \ve{v}) + b(\ve{v} + \ve{w}) + c(\ve{u} + \ve{w}). \end{equation*}

This induces a linear relation on \(\left\{ \ve{u}, \ve{v}, \ve{w} \right\}\text{:}\) \((a + c)\ve u + (a +b )\ve v + (b + c)w = \ve 0\) Since \(\left\{ \ve{u}, \ve{v}, \ve{w} \right\}\) is linearly independent, we must have that

\begin{equation*} a + c = 0 \qquad (1) \\ a + b = 0 \qquad (2) \\ b + c = 0 \qquad (3) \end{equation*}

\((1) + (2) \) combined with \((3) \) gives us that \(a = 0 \) and so \(b = c = 0 \) too. We conclude that \(\left\{ \ve{u} + \ve{v}, \, \ve{v} + \ve{w}, \, \ve{u} + \ve{w} \right\}\) is linearly independent.

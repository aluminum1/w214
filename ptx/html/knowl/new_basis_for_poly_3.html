Example 2.2.7

The list of polynomials

\begin{equation*} \ve{q}_0(x) := 1, \,\, \ve{q}_1(x) := x, \,\, \ve{q}_2(x) := 2x^2 - 1, \,\, \ve{q}_3(x) := 4x^3 - 3x \end{equation*}

from ExampleÂ 2.1.5 is linearly independent in \(\Poly_3\text{.}\) This is because the equation

\begin{equation*} k_0 \ve{q}_0 + k_1 \ve{q}_1 + k_2 \ve{q}_2 + k_3 \ve{q}_3 = \ve{0} \end{equation*}

becomes the following equation between polynomials:

\begin{equation*} 4k_3 x^3 + 2k_2 x^2 + (-3k_3 + k_1) x + (-k_2 + k_0) = 0 \end{equation*}

This is equivalent to the following system of equations,

\begin{equation*} 4k_3 = 0, \quad 2k_2 = 0, \quad -3 k_3 + k_1 = 0, \quad k_0 - k_2 = 0 \end{equation*}

which has only the trivial solution \(k_0 = k_1 = k_2 = k_3 = 0\text{.}\)

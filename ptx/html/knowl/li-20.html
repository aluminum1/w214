D3. Scalar multiplication

We define scalar multiplication on \(B\) as follows. Let \(k\) be a real number and \(\ve{u} = (u_1, \, u_2, \, u_3)\) be an element of \(B\text{.}\) We define

\begin{equation} k \smul\ve{u} := (ku_1, \, ku_2, \, ku_3)\text{.}\tag{1.3.4} \end{equation}

We need to check that this makes sense. When I multiply a multiply a vector \(\ve{u}\) in \(B\) by a scalar \(k\text{,}\) the result \(k \smul\ve{u}\) is supposed to be an element of \(B\text{.}\) Does \((ku_1, \, ku_2, \, ku_3)\) really belong to \(B\text{?}\) Let us check that it satisfies the defining equation (1.3.1):

\begin{align*} \amp ku_1 - ku_2 + ku_3\\ \amp = k (u_1 - u_2 + u_3) \amp \amp \mbox{(this algebra step is true for ordinary numbers)}\\ \amp = k 0 \amp \amp \mbox{(since \(\ve{u}\) is in \(B\))}\\ \amp = 0\text{.} \end{align*}

Therefore, \(k \smul\ve{u}\) is indeed an element of \(B\text{,}\) so we have written down a well-defined scalar multiplication operation on \(B\text{.}\)

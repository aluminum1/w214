Exercise 1

Define an addition operation on the set \(X := \{ \ve{0}, \, \ve{a}, \, \ve{b} \}\) by the following table:

\begin{equation*} \begin{array}{c|ccc} + \amp \ve{0} \amp \ve{a} \amp \ve{b} \\ \hline \ve{0} \amp \ve{0} \amp \ve{a} \amp \ve{b} \\ \ve{a} \amp \ve{a} \amp \ve{0} \amp \ve{a} \\ \ve{b} \amp \ve{b} \amp \ve{a} \amp \ve{0} \end{array} \end{equation*}

This table works as follows. To calculate, for example, \(\ve{b} + \ve{a}\text{,}\) find the intersection of the row labelled by \(\ve{b}\) with the column labelled by \(\ve{a}\text{.}\) We see that \(\ve{b} + \ve{a} := \ve{a}\text{.}\)

Prove that this addition operation satisfies R1.

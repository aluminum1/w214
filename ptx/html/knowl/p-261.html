A homogenous \(n\)th order linear ordinary differential equation on an interval \(I\) is a differential equation of the form

\begin{equation} a_n(x) y^{(n)}(x) + a_{n-1}(x) y^{(n-1)}(x) + \cdots + a_1(x)y'(x) + a_0(x) y(x) = 0, \quad x \in I\tag{1.6.8} \end{equation}

where \(y^{(k)}\) means the \(n\)th derivative of \(y\text{.}\) A solution to the differential equation is just some function \(y(x)\) defined on the interval \(I\) which satisfies (1.6.8).

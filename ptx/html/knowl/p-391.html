The list of polynomials

\begin{equation*} \ve{p}_0 (x) := 1, \, \ve{p}_1 (x) := x, \, \ve{p}_2 (x) := x^2, \, \ldots, \, \ve{p}_n (x) := x^n \end{equation*}

is a basis for \(\Poly_n\text{,}\) so \(\Dim \Poly_n = n+1\text{.}\) Indeed, this list spans \(\Poly_n\) by definition, so we just need to check that it is linearly independent. Suppose that

\begin{equation*} a_0 \ve{p}_0 + a_1 \ve{p}_1 + a_2 \ve{p_2} + \cdots + a_n \ve{p}_n = \ve{0}\text{.} \end{equation*}

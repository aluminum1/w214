Solution 1.4.3.1

R1: We must check whether \(\ve{v} + \ve{w} = \ve{w} + \ve{v} \text{ for all } \ve{v}, \ve{w} \in \{\ve{a},\ve{b}\}\text{.}\) Clearly \(\ve{a} + \ve{a} = \ve{a} + \ve{a}\) and likewise for \(\ve{b}\text{.}\) And finally \(\ve{b} = \ve{a} +\ve{b} = \ve{b} + \ve{a}\text{.}\)

R2: We must check whether \((\ve{u} + \ve{v}) + \ve{w} = \ve{u} + (\ve{v} + \ve{w}) \text{ for all } \ve{u} , \ve{v},\ve{w} \in \{\ve{a},\ve{b}\}\text{.}\) This requires we check 8 equations in total. For brevity, we shall only present the solution for one of them, the rest are virtually identical. We check whether

\begin{gather*} (\ve{a} + \ve{b}) + \ve{b} = \ve{a} +(\ve{b} + \ve{b} \end{gather*}

To that end, consider:

\begin{align*} \text{LHS} \amp = (\ve{a} + \ve{b}) + \ve{b}\\ \amp = \ve{b} + \ve{b}\\ \amp = \ve{a}\text{.} \end{align*}

By a similar method:

\begin{align*} \text{RHS} \amp = \ve{a} +(\ve{b} + \ve{b})\\ \amp = \ve{a} + \ve{a}\\ \amp = \ve{a}\text{.} \end{align*}

R3, R4, R6, and R7 all follow from routine checks.

We shall demonstrate why R5 is not satisfied. For that, we need to find a counterexample. Take \(k=2=l, \ve v = \ve b\text{.}\) Then

\begin{align*} \text{LHS} \amp = (2+2).\ve{b}\\ \amp = 4.\ve{b}\\ \amp = \ve{b} \end{align*}

whereas

\begin{align*} \text{RHS} \amp = 2.\ve{b} + 2.\ve{b}\\ \amp = \ve{b} + \ve{b}\\ \amp =\ve{a} \end{align*}

Since \(\text{LHS} \neq \text{RHS}\text{,}\) R5 cannot be true.

In \(\mathbb{R}^4\text{,}\) \((2, -1, 3, 0)\) is not a linear combination of \((1, 3, 2, 0), \, (5, 1, 2, 4)\) and \((-1, 0, 2, 1)\text{.}\) If it was, then we would have

\begin{equation*} (-2, -1, 3, 0) = a_1 (1, 3, 2, 0) + a_2 (5, 1, 2, 4) + a_3 (-1, 0, 2, 1) \end{equation*}

for some scalars \(a_1, a_2, a_3\text{.}\) In other words, we would have

\begin{equation*} (-2, -1, 3, 0) = (a_1 + a_2 + a_3, \, 3a_1 + a_2, \, 2a_1 + 2a_2 + 2a_3, \, 4a_2 + a_3) \end{equation*}

which is equivalent to the system of equations

\begin{align*} a_1 + 5a_2 -a_3 \amp = -2, \\ 3a_1 + a_2 \amp = -1, \\ 2a_1 + 2a_2 + 2a_3 \amp = 3,\\ 4a_2 + a_3 \amp = 0. \end{align*}

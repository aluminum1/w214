If we use the Linear Combination of Preceding Vectors Proposition, the proof is very slick! \(\ve v_i \) (for any \(i \in \left\{1,\ldots n\right\}\)) cannot be a linear combination of \(\ve v_1 \ldots \ve v_{i-1} \) or else we would contradict the Linear Combination of Preceding Vectors Proposition. It is given that \(\ve v \) is not a linear combination of \(\ve v_1 \ldots \ve v_{n} \text{.}\) Hence no vector in \({B}' = \left\{ \ve{v}_1, \ldots, \ve{v}_n, \ve{v} \right\}\) is a linear combination of the previous vectors. Thus by the Linear Combination of Preceding Vectors Proposition, \({B}' \) must be linearly independent.

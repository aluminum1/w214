Fix constants \(a,b\) and \(c\text{.}\) Let \(V \subseteq \Diff(\mathbb{R})\) be the set of all solutions to the following differential equation:

\begin{equation*} a \frac{d^2f}{dx^2} + b \frac{df}{dx} + cf = 0 \end{equation*}

To be completely explicit, let us consider the ordinary differential equation

\begin{equation} 5y^{\prime \prime} + y^\prime - 3y = 0.\tag{1.7.6} \end{equation}

You can use methods learnt in first year to show that

\begin{equation*} y_1 = e^{-\frac{1}{10}(1 + \sqrt{61})x}, \quad y_2 = e^{\frac{1}{10}(-1 + \sqrt{61})x} \end{equation*}

are two solutions to (1.7.6).

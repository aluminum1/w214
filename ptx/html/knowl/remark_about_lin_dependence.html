Remark 2.2.2 Zero vector implies linear dependence

Suppose one of the vectors \(\ve{v}_i\) in the list \(\basis{B} = \bopen \ve{v}_1, \ldots, \ve{v}_n \bclose \) is the zero vector \(\ve{0}\text{.}\) Then \(\basis{B}\) is linearly dependent, since the equation (2.2.1) has the nontrivial solution

\begin{equation*} 0 \ve{v}_1 + \cdots + 0 \ve{v}_{i-1} + 1 \ve{v}_i + 0 \ve{v}_{i+1} + \cdots + 0 \ve{v}_n = \ve{0}, \end{equation*}

in other words,

\begin{equation*} k_1 = 0, \ldots, k_{i-1} = 0, k_i = 1, k_{i+1} = 0, \ldots, k_n = 0. \end{equation*}

So: a linearly independent list ofvectors never contains the zero vector!

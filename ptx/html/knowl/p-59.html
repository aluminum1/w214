Let us see. Let \(\ve{a} = (a_1, \,a_2)\) and \(\ve{a}' = (a'_1, \, a'_2)\) be elements of \(A\text{.}\) We can add them in two different orders,

\begin{equation*} \ve{a} + \ve{a}' = (a_1 + a'_1, \, a_2 + a'_2) \end{equation*}

and

\begin{equation*} \ve{a}' + \ve{a} = (a'_1 + a_1, \, a'_2 + a_2) \text{.} \end{equation*}

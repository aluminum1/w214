We shall use the basis \(\basis B = \bopen x^3, x^3 + x^2, x,1 \bclose\text{.}\) Since \(\ve p(x) \) has no degree 2 term, we know immediately that

\begin{equation*} \ve p(x) = ax^3 + 0(x^3 + x^2) + cx + d. \end{equation*}

Reading off the rest of the coefficients, we see that \(a =3, c = -7, d = 1 \text{.}\) Hence

\begin{equation*} [p]_\basis{B} = \begin{bmatrix} 3 \\ 0 \\-7 \\1\end{bmatrix}. \end{equation*}

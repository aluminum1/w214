For instance, consider the specific example from Example 1.6.13, namely the plane \(W \subset \mathbb{R}^3\) of vectors orthogonal to \(\ve{v} = (1,2,3)\text{.}\) In other words,

\begin{equation} W = \{ (w_1, w_2, w_3) \in \mathbb{R}^3 : w_1 + 2w_2 + 3w_3 = 0 \}.\tag{2.3.25} \end{equation}

There is no 'standard' basis for \(W\text{.}\) But, here is one basis (as good as any other):

\begin{equation} \ve{a} = (1, 0, -\frac{1}{3}), \quad \ve{b} = (0, 1, -\frac{1}{2}).\tag{2.3.26} \end{equation}

You will show this is indeed a basis for \(W\) in Checkpoint 2.3.14 below. I computed these vectors as follows. To obtain \(\ve{a}\text{,}\) I simply set \(w_1 = 1, w_2 = 0\) and then solved for \(w_3\) using Equation (2.3.25). Similarly, for \(\ve{b}\text{,}\) I simply set \(w_1 = 0, w_2 = 1\) and then solved for \(w_3\) using (2.3.25).

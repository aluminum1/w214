Example 2.1.3 Checking if a vector is a linear combination of other vectors

In \(\mathbb{R}^4\text{,}\) is \(\ve{v}=(2, -1, 3, 0)\) a linear combination of

\begin{equation*} \ve{v}_1 = (1, 3, 2, 0), \ve{v}_2=(5, 1, 2, 4), \text{ and } \ve{v}_3 = (-1, 0, 2, 1) \text{?} \end{equation*}

To check this, we need to check if the equation

\begin{equation} \ve{v} = a_1 \ve{v}_1 + a_2 \ve{v}_2 + a_3 \ve{v}_3,\tag{2.1.2} \end{equation}

which is an equation in the unknowns \(a_1, a_2, a_3\text{,}\) has any solutions. Let us write out (2.1.2) explicitly:

\begin{align} (2, -1, 3, 0) \amp = a_1 (1, 3, 2, 0) + a_2 (5, 1, 2, 4) + a_3 (-1, 0, 2, 1) \tag{2.1.3}\\ \therefore (2, -1, 3, 0) \amp = (a_1 + 5a_2 -a_3, 3a_1 + a_2, 2a_1 + 2a_2 + 2a_3, 4a_2 + a_3) \tag{2.1.4} \end{align}

(2.1.4) is an equation between two vectors in \(\mathbb{R}^4\text{.}\) Two vectors in \(\mathbb{R}^4\) are equal if and only if their corresponding coefficients are equal. So, (2.1.2) is equivalent to the following system of simultaneous linear equations:

\begin{align} a_1 + 5a_2 -a_3 \amp = -2 \tag{2.1.5}\\ 3a_1 + a_2 \amp = -1 \tag{2.1.6}\\ 2a_1 + 2a_2 + 2a_3 \amp = 3\tag{2.1.7}\\ 4a_2 + a_3 \amp = 0 \tag{2.1.8} \end{align}

In other words, our question becomes: do equations (2.1.5)–(2.1.8) have a solution?

This is the kind of problem you already know how to solve by hand, from first year. We can also use SageMath to do it for us. We simply tell it what our unknown variables are, and then ask it to solve the equation. Press Evaluate (Sage) to see the result.

SageMath returns an empty list []. In other words, there are no solutions to equations (2.1.5)–(2.1.8). Therefore \(\ve{v}\) cannot be expressed as a linear combination of \(\ve{v}_1, \ve{v}_2, \ve{v}_3\text{.}\)

Let us mention a few features that the sets do not have, or at least do not have in common.

  • Set \(A = \mathbb{R}^2\) has a multiplication operation. Because we can think of \(\mathbb{R}^2\) as the complex plane \(\mathbb{C}\text{,}\) and we know how to multiply complex numbers. There is no clear choice of a multiplication operation on \(B\text{.}\) The same for \(C\text{:}\) if you try to multiply two degree \(4\) polynomials in \(C\) together, you will get out a polynomial of degree \(8\text{,}\) which does not live in \(C\text{!}\)

  • There is a ‘take the derivative’ operation on \(C\text{,}\)

    \begin{equation*} \ve{c} \mapsto \frac{d}{dx} \ve{c} \end{equation*}

    which we will meet again later. Note that taking the derivative decreases the degree of a polynomial by 1, so the result remains in \(C\text{,}\) and so this is a well defined map from \(C\) to \(C\text{.}\) There is no analogue of this operation in \(A\) and \(B\text{.}\)

    Note that there is no integration map from \(C\) to \(C\text{,}\) because integrating a polynomial increases the degree by \(1\text{,}\) so the result might be a polynomial of degree \(5\text{,}\) which does not live in \(C\text{!}\)

Let \(V\) be a vector space of dimension \(n\text{.}\) State whether each of the following statements is true or false. If it is true, prove it. If it is false, give a counterexample.

  1. Any linearly independent list of vectors in \(V\) contains at most \(n\) vectors.

  2. Any list of vectors which spans \(V\) contains at least \(n\) vectors.

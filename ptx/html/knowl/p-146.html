R4 :Let

\begin{align*} \amp \ve{v} = (v_1,v_2,v_3,\ldots) \\ \amp \ve{w} = (w_1,w_2,w_3,\ldots) \end{align*}

We must check whether \(k.(\ve{v} + \ve{w}) = k.\ve{v} + k. \ve{w}\text{.}\)

\begin{align*} \text{LHS} \amp = k.(\ve{v} + \ve{w})\\ \amp = k.\left[(v_1,v_2,v_3,\ldots) +(w_1,w_2,w_3,\ldots)\right]\\ \amp = k.\left(v_1 + w_1, v_2 + w_2, v_3 + w_3, \ldots \right)\\ \amp = (k(v_1+w_1), k(v_2 + w_2), k(v_3 + w_3), \ldots)\\ \amp = (kv_1+kw_1, kv_2 + kw_2, kv_3 + kw_3, \ldots)\\ \amp =(kv_1,kv_2,kv_3,\ldots) + (kw_1,kw_2,kw_3,\ldots) \\ \amp = k.(v_1,v_2,v_3,\ldots) + k.(w_1,w_2,w_3,\ldots) \\ \amp = k.\ve{v} + k.\ve={w} \\ \amp = \text{LHS} \end{align*}

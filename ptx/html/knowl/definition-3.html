Definition 2.1.1

A linear combination of a finite collection \(\ve{v}_1, \ldots, \ve{v}_n\) of vectors in a vector space \(V\) is a vector of the form

\begin{equation} a_1 \ve{v}_1 + a_2 \ve{v}_2 + \cdots + a_b \ve{v}_n\tag{2.1.1} \end{equation}

where \(a_1, a_2, \ldots, a_n\) are scalars. If all the scalars \(a_i\) are \(0\text{,}\) we say that it is the trivial linear combination.

Proof

\(\Rightarrow\text{.}\) Suppose that the list of vectors\(\basis{B} = \left\{ \ve{e}_1, \ve{e}_2,\ldots, \ve{e}_n\right\}\) is a basis for \(V\text{.}\) Suppose \(\ve{v} \in V\text{.}\) Since the list of vectors \(\basis{B}\) spans \(V\text{,}\) we know that we can write \(\ve{v}\) as a linear combination of the vectors in the list in at least one way,

\begin{equation} \ve{v} = a_1 \ve{e}_1 + a_2 \ve{e}_2 + \cdots + a_n \ve{e}_n\text{.}\tag{2.4.2} \end{equation}

We need to show that this is the only way to express \(\ve{v}\) as a linear combination of the vectors \(\ve{e}_i\text{.}\) Indeed, suppose that we also have

\begin{equation} \ve{v} = b_1 \ve{e}_1 + b_2 \ve{e}_2 + \cdots + b_n \ve{e}_n\text{.}\tag{2.4.3} \end{equation}

Subtracting these two equations gives

\begin{equation*} \ve{0} = (a_1 - b_1) \ve{e}_1 + (a_2 - b_2) \ve{e}_2 + \cdots + (a_n - b_n) \ve{e}_n\text{.} \end{equation*}

Since the list of vectors \(\ve{e}_1\text{,}\) \(\ve{e}_2\text{,}\) \(\ldots\text{,}\) \(\ve{e}_n\) is linearly independent, we conclude that

\begin{equation*} a_1 - b_1 = 0, \,\, a_2 - b_2 = 0, \,\, \cdots, \,\, a_n - b_n = 0\text{.} \end{equation*}

That is, \(a_1 = b_1\text{,}\) \(a_2 = b_2\text{,}\) and so on up to \(a_n = b_n\text{,}\) and hence the expansion (2.4.2) is unique.

\(\Leftarrow\text{.}\) Conversely, suppose that every vector \(\ve{v}\) can be written as a unique linear combination

\begin{equation*} \ve{v} = a_1 \ve{e}_1 + a_2 \ve{e}_2 + \cdots + a_n \ve{e}_n\text{.} \end{equation*}

The fact that each \(\ve{v}\) can be written as a linear combination of the vectors \(\ve{e}_1\text{,}\) \(\ve{e}_2\text{,}\) \(\ldots\text{,}\) \(\ve{e}_n\) means that \(\basis{B}\) spans \(V\text{.}\) We still need to show that this list \(\basis{B}\) is linearly independent. So, suppose that there exist scalars \(b_1\text{,}\) \(b_2\text{,}\) \(\ldots\text{,}\) \(b_n\) such that

\begin{equation} b_1 \ve{e}_1 + b_2 \ve{e}_2 + \cdots + b_n \ve{e}_n = \ve{0}\text{.}\tag{2.4.4} \end{equation}

We need to show that all the \(b_i\) must equal zero. We already know one possible solution of (2.4.4) : simply set each \(b_i = 0\text{.}\) But we are told that each vector (in particular, the vector \(\ve{0}\)) can be expressed as a linear combination of the \(\ve{e}_i\) in exactly one way. Hence this must be the only solution, i.e. we must have \(b_1 = b_2 = \cdots = b_n = 0\text{,}\) and so the list \(\basis{B}\) is linearly independent.
